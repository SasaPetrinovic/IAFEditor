IsLineModifiedExclude
   

This method determines whether the grid current line has been modified. Columns in the list are excluded from checking.


Format

            IsLineModifiedExclude (list_of_columns_to_be_excluded)


Parameters

·     list_of_columns - comma separated list of column names to be excluded from checking. Column name is the name of the field in particular column.


Return Value Type

            Numeric.

Return Value

            If a new line is entered, result is 2.
            For existing lines, if any field is modified, result is 1, otherwise 0.

Applied To

            Data-entry grid.

Note

            This function is usually used in 'Line ending actions' of the grid.

            The function takes into account all the destination, combobox and checkbox fields except
these in 'list_of_columns'.

Examples

  
      $status = Table_1.IsLineModifiedExclude(col1, col2)
      if $status = 2
            Pause 'New line'
      else_if $status = 1
            Pause 'Modified !'
      end_if
