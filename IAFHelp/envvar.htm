ENVVAR
   

This function returns value of specified environment variable.

Format

            ENVVAR(env_variable_name)

Return Value Type

            Alpha.

Parameters


·     env_variable_name - name of environment variable, alpha literal or alpha field. If the variable does not exist, result is blank field.

Examples

  
      $S = ENVVAR('SystemRoot')
      $S = ENVVAR('USERPROFILE')

      $ENAME = 'USERPROFILE'
      $S = ENVVAR($ENAME)
