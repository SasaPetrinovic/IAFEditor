Freezecolumns
   

This method enables columns on the left side to be visible when scrolling through the grid.

Format

            Freezecolumns (ncolumns)

Parameters


·    ncolumns - number of columns to be frozen, integer.


Applied To

            Grid.

Notes

            To unlock frozen columns enter table_x.Freezecolumns(0).

Example

  
      Table_1.Freezecolumns(1)      !! Locks the first column
