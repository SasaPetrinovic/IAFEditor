COMPARE_RECORDS
   

This function compares two record buffers, field by field. CLOB and BLOB fields are ignored.


Format 1          Compare all fields

            COMPARE_RECORDS(rec1, rec2)

Format 2          Compare all fields except Field1, Field2, ...

            COMPARE_RECORDS(rec1, rec2,'E',field1,field2, ...)

Format 3          Compare only fields Field1, Field2, ...

            COMPARE_RECORDS(rec1, rec2,'I',field1,field2, ...)


Return Value Type

            Numeric.

            Result value:

            = 0 if rec1 = rec2
            != 0 if rec1 != rec2

Parameters


·     rec1 - the first record, from Datasource dialog.
·     rec2 - the second record, from Datasource dialog.
·     field1, field2, ...- list of fields.

Note

            Records REC1 and REC2 have the same structure.

Examples

  
      $res = COMPARE_RECORDS(rec1, rec2)
      $res = COMPARE_RECORDS(rec1, rec2, 'E', fld1)
      $res = COMPARE_RECORDS(rec1, rec2, 'I', fld1,fld2)
