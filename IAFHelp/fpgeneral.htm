Form painter - General information
   

Form painter enables you to design (paint) forms or reports. To create a form (or report) you place objects to the form. Objects can be modified, moved around the form, resized, removed, etc. Active objects (destination field, combo box, check box, radio button, button, source field) have associated code buffer.

Inserting new object to the form


·     Click appropriate button on the Toolbox.
·     Move the cursor to the desired position, then click.

-or-


·     Choose menu command Objects/some_object.
·     Move the cursor to the desired position, then click.

-or-


·     Choose menu command Objects/Field explorer.
·     Click Sourcefield or Destinationfield on the Toolbox.
·     Drag a field to the form. Using this method we can create source and destination fields only.


Selecting object(s)

To select one object,  just click it.

-or-

Click and drag the mouse around the object.

Selecting multiple objects:


·     Select one object,
·     Ctrl/Click to select next object, and so on.

-or-

Click and drag the mouse around the objects.

To select an object that is behind another object (denoted by blue rectangle), hold down Alt then click inside blue rectangle.

If you want to remove one object from multiple selection, Ctrl/Click that object.

One of many selected objects is a dominant object. Such object has hatched tracker. The other objects have broken-line tracker.


Deleting objects

Select object(s) and

Press Delete key.

-or-

Right click to display context menu and choose Delete object(s) command.


Copying objects

Select object(s) and

Press Ctrl/C.

-or-

choose menu command Edit/Copy.

Copied objects can be pasted by pressing Ctrl/V or choosing menu command Edit/Paste. To define position(s) of newly created objects, you must previously click at desired position.

Moving objects

Select object(s) and drag it to the new position

-or-

Select object(s) and, using arrow keys, move it around the form.


Resizing objects

Select object, place the mouse cursor to the appropriate resize handle, and drag that edge.

-or-


·     Select the object.
·     Increase/decrease a dimension by 1 pixel using Ctrl/Right_arrow or Ctrl/Left_arrow.
·     Increase/decrease a dimension by 10 pixels using Shift/Right_arrow or Shift/Left_arrow.


Changing properties of object

Double-click the object.

-or-

Select the object and choose Properties command from the context menu.
