CHR
   

This function returns the character that has the ASCII code value specified by numeric_exp.
Numeric_exp must be in the range 0 to 255.

Format

            CHR(numeric_exp)

Return Value Type

            Alpha.

Parameters


·     numeric_exp - numeric literal or numeric field in the range 0 to 255.


Examples

  
      $A = CHR(9) !! result is a TAB character
      $A = CHR(13) !! result is a carriage return char
      $A = CHR(27) !! result is an ESC character
      $A = CHR(65) !! result is "A"
