Expand
   


This method expands a tree node.


Format

            Expand (level, key)

Parameters


·     level - tree level, integer variable or literal. Allowed values are 0 to 10. To expand all tree nodes, set level to 0 (zero). To expand one node only, the level must be in the range 1 to 10.
·     key - item key, alpha variable. This variable represents the concatenation of the all keys stored in the tree, from level 1 up to the current level. This parameter is ignored if level = 0.

Applied To

            Tree.

Note

            This method is usually used in Post tree-filling actions to expand desired node.

Example

  
Lets consider the following tree:

001 Item_1
002 Item_2
  01 Item_2_1
  02 Item_2_2
  03 Item_2_3
    1 Item_2_3_1
    2 Item_2_3_2
003 Item_3

001, 002, 003, 01, 02, ... are the keys stored in the tree nodes.
To expand Item_2 we have to do
      Tree_1.Expand(1,"002")

To expand Item_2_3, the call is
      Tree_1.Expand(2,"00203")

To expand all items, the call is
      Tree_1.Expand(0,"")
