EDIT
   

This function returns formatted representation of alpha, numeric, date, datetime or time variable.

Format

            EDIT(val_exp, edit_mask)

Return Value Type

            Alpha.

Parameters


·    val_exp - input value expression
·    edit_mask - edit mask

Note

            To convert an integer number in the range 1 to 3999 to Roman numerals, use edit mask "R". If the number is less than 1 or greater than 3999, one question mark will be moved to the destination buffer.

Examples

  
      $X = EDIT(acc_1,"ZZZ.ZZ9,99-")
      $X = EDIT(counter_1,"R") !! conversion to Roman numerals
      $X = EDIT(today,"DD/MM/YY")
      $X = EDIT(today,"YYYY-MM-DD")
      $x = EDIT(today,'L')    !! long date format
      $X = EDIT(systime,"HH:MM:SS")
      $X = EDIT($ALPHA,"XXX") !! move first 3 characters to $X
      $X = EDIT($ALPHA,"XXX-XXX")
! If $ALPHA = "ABCDEF", result will be "ABC-DEF" (7 characters)

      $X = EDIT($ALPHA,"XXX")
! gives the same result as
      $X = $ALPHA[1:3]
