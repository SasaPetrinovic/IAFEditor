Open
   

This method creates an empty XML file and opens it.

Format

            Open (xml_filename)

Parameters


·     xml_filename - an alpha variable or alpha literal. Specifies name of the XML file. Logical names in filename are allowed.

Return Value

            Status, numeric variable.

            Possible values:
  
0
Ok
-1
Logical name not found
-2
File already opened
> 0
File input/output error

Applied To

            Xml object.

Note

            To release current XML file use Close method.

Examples

  
      $stat = Xml.Open("c:\xml\test.xml")
      $stat = Xml.Open("xmldir:test.xml")
      $stat = Xml.Open($file_name)
      if $stat = 0
            ...
      end_if
