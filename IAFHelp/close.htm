CLOSE
   

This statement closes a file or database.

Format 1

            CLOSE[/DELETE] file_name

Format 2

            CLOSE /DATABASE


Where


·     file_name - a name of the file from datasource dialog.


Qualifiers


            /DELETE deletes file after closing.

            /DATABASE closes the connection to a database.


Example

  
      CLOSE file1
      CLOSE/DELETE file1

      CLOSE/DATABASE
