FUNCTION
   

This statement specifies a beginning of a function. The function is a procedure which has arguments and returns a value.


Format

            FUNCTION function_name (arg_1 desc_1 [ARRAY], . . ., arg_n desc_n [ARRAY]) RETURN_VALUE ret_desc
                        ...
                        statements
                        …
            END_FUNCTION


Where

·     arg_1,..., arg_n - names of formal arguments (maximum number of arguments is 30)
·     desc_1,..., desc_n - argument description represented in a form:

  
An
 n = length of ANSI alpha field
Nnn[,dd]
 nn = total number of digits, dd = number of decimals
Dn
 n = length of date field (6 or 8)
Tn
 n = length of time field (4 or 6)
DT
 datetime field
Wn
 n = length of unicode UTF-16 field


            If the input argument is an array, use keyword ARRAY. To determine real number of array elements, use NUMBER_OF_ELEMENTS function. If an array argument is omitted, number of elements is 0 (zero).

·     ret_desc - description of return value, format is the same as argument description.

            All the statements between FUNCTION and END_FUNCTION represent the function body.
            At least one statement in the body must be an assignment with function name on the left-hand side. In this way the return value will be established.

Examples

  
      FUNCTION myfnc(CODE A5, FAMMOUNT N15,2, FID N6) RETURN_VALUE A8
            myfnc = code
            if fammount > 100,50 OR fid > 10
                  myfnc = '12345678'
            end_if
      END_FUNCTION

      FUNCTION fnc(DATET DT) RETURN_VALUE DT
            fnc = datet
            if today > '1-jan-2012'
                  fnc = datetime(today,systime)
            end_if
      END_FUNCTION

! Input argument INARR is an alpha array, array element length is 5.
! Use NUMBER_OF_ELEMENTS function to determine real number of elements.
      FUNCTION fnc(INARR A5 ARRAY) RETURN_VALUE A5
            define_local $$cnt n 9 L
            $$cnt = number_of_elements(INARR)
            if $$cnt <> 0
                  fnc = INARR($$cnt - 1)
            else
                  fnc = SPACE
            end_if
      END_FUNCTION
