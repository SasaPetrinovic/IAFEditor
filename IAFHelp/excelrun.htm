Run
   

This method saves the document, breaks connection between program and MS Excel, and gives control to the user. After this, another document can be opened, etc.

Format

            Run([command,filename])

Parameters


·     command - an alpha variable or alpha literal.  Possible values are:
            "print" - prints current sheet on default printer,
            "print Xn:Ym" prints range of cells, from cell Xn to the cell Ym,
            "print X:Y" prints range of columns, from column X to column Y,
            "save" - invokes 'Save as',
            "save_exit" - invokes 'Save as' and stops Excel,
            "save_overwrite" - invokes 'Save',
            "save_overwrite_exit" - invokes 'Save' and stops Excel.

            For "save_overwrite" or "save_overwrite_exit", an existing file must be opened by Open method.


·     filename - an alpha variable or alpha literal. Specifies name of the output file which will be used in 'Save as'. Logical names in filename allowed. If command is "save_overwrite" or "save_overwrite_exit", filename is an empty string.

Return Value

            Status, numeric variable.

            Possible values:

            0          Ok
            -1         Document not opened
            -2         Invalid command
            > 0       File errors

Applied To

            Excel object.

Note

            This method is usually used to save results.
            If we want that Excel stays active after Run, the Close method need not be called.

Examples

  
            Variants

            Run()
            Shows Excel without 'Save as'
            Run("save",filename)
            Invokes 'Save as', shows Excel
            Run("save_exit",filename)
            Invokes 'Save as' and exits Excel

      $stat = Excel.Run()
      $stat = Excel.Run("save","c:\doc\out.xls")
      $stat = Excel.Run("save_exit","docdir:test.xls")

! Open an existing document
! change it
! save updated document
      $stat = Excel.Open("somefile.xls")
      $stat = Excel.SetCell(1,'A',acc_1)
      $stat = Excel.Run("save_overwrite", "")
