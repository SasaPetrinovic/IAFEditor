Modifyitem
   

This methods modifies text label of a tree item.

Format

            Modifyitem(level, text_label, key)

Parameters


·     level - tree level, integer variable or literal. Allowed values are 1 to 10.
·     text_label - text to be displayed.
·     key - concatenation of the all keys stored in the tree, from level 1 up to this level.


Applied To

            Tree.

Note

            To refill whole tree, we can use:

            Tree_1.Init()
            Tree_1.Setfocus()

            But, if we want to change text label of one item, we can use this method.


Example

  
      Tree_1.ModifyItem(1,"New text label",$key)

Lets consider the following tree:

001 Item_1
002 Item_2
  01 Item_2_1
  02 Item_2_2
  03 Item_2_3
003 Item_3

If we want to change text of Item_2_3, we should write
      Tree_1.ModifyItem(2,"Item_2_3 changed","00203")
