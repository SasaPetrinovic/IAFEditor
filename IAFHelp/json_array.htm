JSON_ARRAY
   

This function reads one element from a JSON array.


Format

            JSON_ARRAY(json_variable, array_name, index)

Result Value Type

            Alpha field or alpha string.

Parameters


·     json_variable - alpha variable containing JSON structure. Variable type is CLOB, longchar (type = S).
·     array_name - array name, alpha literal or alpha variable.
·     index - index, integer. The first element has index=1.

Note

            If index < 1 or index > number_of_elements, result is an empty string.

Examples

  

$fld = json_array($json,'array_1',1)
$fld = json_array($json,'array_1',counter_1)

! Read all elements ($s is string, type=S)
$index = 0
do_while $index < 99999 !! 99999 is an arbitrary large number
      add 1 to $index
      $s = json_array($json,'array_1',$index)
      if str_length($s) = 0
            break
      end_if
      ...
      do something with string $s
      ...
end_do
