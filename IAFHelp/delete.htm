Delete
   

This method deletes a file from the connected FTP server.

Format

            Delete(filename)

Parameters


·     filename - an alpha variable or alpha literal. Specifies name of the file, with or without path specification. Wildcards not allowed.


Return Value

            Alpha variable. If connecting was successful, value is blank. Otherwise, the variable holds an error message.


Applied To

            Ftp object.

Note


Examples

  
      $msg = Ftp.Delete('file1.txt')      !!current directory
      if $msg <> SPACE
            Pause 'Error: ' | $msg
      end_if
      $msg = Ftp.Delete('/file1.txt')     !!root directory
      $msg = Ftp.Delete('/subdir/file1.txt') !!absolute path
      $msg = Ftp.Delete('sub/file1.txt') !!relative path
