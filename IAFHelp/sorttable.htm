Sorttable
   

This method sorts a display-only table (grid) by a column.

Format

            Sorttable([col1[, col2, col3, col4]])

Parameters


·     col1, col2, col3, col4 - numeric literals or numeric variables representing sort columns. First column is 1, second one is 2, etc. Sort in descending order is represented by negative numbers: -1 for first column, etc.

Applied To

            Display-only grid.

Note

            Col1, col2, … are optional.
            
            Default is column 1. Maximum number of sort columns is 4.

            To disable column sorting, enter 0 as the first parameter, i.e. Sorttable(0). The context menu commands 'Sort column' and 'Sort complex' will be grayed.

            The best place to use this method is 'Table ending actions'. At this moment the grid is filled and prepared to display.
            If we do not use this method, line appearance is determined by SELECT_START or START instruction that we used in 'Table starting actions'.

Examples

  
      table_1.sorttable() !! sort by col 1 (default sort)
      table_1.sorttable(1) !! same as above
      table_1.sorttable(1,2) !! sort by col 1, then by col 2
      table_1.sorttable(-1)  !! sort by col 1, descending
      table_1.sorttable(0)    !! disable sorting
