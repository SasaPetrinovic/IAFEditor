ADD_YEARS
   

This function adds number of years to a date.

Format

            ADD_YEARS(date, numyears)

Return Value Type

            Date.

Parameters


·    date - date expression
·    numyears - number of years, numeric

Note

            Out-of-range result date will be corrected automatically.

Examples

  
      $Years = ADD_YEARS (today,5)
      $Years = ADD_YEARS (BEGINCY,-3)
      $Years = ADD_YEARS (today,counter_1)

      $Years = ADD_YEARS("29-feb-2000",1) !! result is 28-feb-2001
 

See Also    ADD_DAYS, ADD_WEEKS, ADD_MONTHS, Date Functions
