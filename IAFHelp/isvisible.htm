Isvisible
   

This method returns the state of visibility of an object.


Format

            Isvisible()

Return Value Type

            Numeric.

Return Value

            If the object is visible, result is 1, otherwise result is 0.

Parameters

            None.

Applied To

            All objects.

Examples

  
      $status = button_1.Isvisible()
      if button_3.Isvisible() = 1
            Pause "Visible !"
      end_if

      $objname = 'image_1'
      $status = $objname.Isvisible()
