STR_LENGTH
   

This function returns the position of the last non-blank character in an alphanumeric value.

Format

            STR_LENGTH(alpha_exp)

Return Value Type

            Numeric.

Parameters


·     alpha_exp - alphanumeric field or literal.

Note

            If alpha_exp contains all blanks result is 0, otherwise, result is between 1 and
            length of alpha_exp.

            The leftmost character has position 1.


            If alpha_exp contains an UTF-8 string, result is position of the last non-blank byte.

Examples

  
      $len = STR_LENGTH("ABCD   ") !! result is 4
      $len = STR_LENGTH("ABCD E  ") !! result is 6

      $win1250str = "ŠЎ"     !! Central European letters
      $len = STR_LENGTH($win1250str)      !! result is 3
      $res = convert($win1250str,1250,65001) !! convert to UTF-8
      $len = STR_LENGTH($res)       !! result is 6
