SELECT_START
   

This statement retrieves many records (rows) from a RDB table(s) in a cycle. The statement is used to fill a display-only grid. The cycle starts with this statement in 'Table_starting' actions and ends implicitly when the input stream is exshausted. After that, program transfers control to the 'Table_ending' actions.

Format

            SELECT_START select_statement

Note

            The block diagram is the following:

            'Table_starting' actions: SELECT_START prepares input stream.
cycle:
            Program fetches next record from the stream.
            If not end_of_stream
                        if not termination_condition
                                    if not suppressline_condition
                                                Display next line in the grid.
                                                Perform 'line_ending' actions.
                                    end_if
                                    Goto cycle
                        end_if
            end_if
            Perform 'table_ending' actions.
            ...

            Termination_condition and suppressline_conditions are grid attributes.

Example

  
            SELECT_START * FROM tbl1 Where PKID < 100 order by PKID
