DIFF_DT
   

This function returns the difference between two datetime expressions in the form: number_of_days/time.

Formats


·           DIFF_DT(date1, time1, date2, time2)
·           DIFF_DT(date_time1, date2, time2)
·           DIFF_DT(date1, time1, date_time2)
·           DIFF_DT(date_time1, date_time2)


Return Value Type

            12-digits alphanumeric, left filled with zeroes.

Parameters


·     date1, date2 - standard date fields
·     time1, time2 - standard time fields
·     date_time1, date_time2 - datetime fields


Note

            Result is a 12-digits number: the first 6 digits represent number of days, the second 6 digits represent time.

            Parameters date1 and time1 must be greater than date2, time2.

Examples

  
!
! $a12 is defined as alpha, 12 characters
!
      $A12 = DIFF_DT ($D1, $T1, $D2, $T2)
      $daysdiff = num ($a12[1:6])
      $timediff = time($a12[7:6])
!
! A numeric field can be used as a result.
! In this case, 12-digit unpacked numeric must be used.
!
      $N12 = DIFF_DT ($D1, $T1, $D2, $T2)
      $daysdiff = $n12[1:6]
      $timediff = time($n12[7:6])
