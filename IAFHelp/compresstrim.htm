COMPRESS_TRIM
   

This function creates a concatenation of many trimmed (left and right) input variables. Only leading and trailing SPACE characters are trimmed (removed from input variables).
If input variables are numeric or date/time ones, default editing will be performed.


Format

            COMPRESS_TRIM(val_exp1,  val_exp2, …, val_expN)

Return Value Type

            Alpha.

Parameters


·    val_exp1,..., val_expN - input variables. Maximum number of input values is 512. Special constant NBSP (non-breaking space) can be used as input variable. This constant indicate that one space will be inserted into the output buffer.


Note

            Trim operation, used in this function, is a little bit different from the other trim functions (LTRIM, RTRIM, ALLTRIM). COMPRESS_TRIM removes SPACE characters only, while LTRIM, RTRIM and ALLTRIM remove SPACE, TAB, newline and carriage return.


Example

  
$A = COMPRESS_TRIM("AB C   ", "   AB", "X") !! result is "AB CABX"
$A = COMPRESS_TRIM("ABC",NBSP,"AB") !! result is "ABC AB"
