Synchronize table definitions
   

This dialog enables you to compare table structures in your Data Dictionary to table structures in the database.

Compare all

Compare all the tables from the list. For each datasource in the list, the dialog 'Enter username/password' appears on the screen. If you click 'Cancel', current table will not be compared. If you click 'Skip this datasource', all the tables belonging the datasource will be ignored.

Compare selected

Compare selected tables only.

Show

Show table structure from the Data Dictionary and appropriate structure from the database.

Remove from list

Remove selected tables from the list. All the table definitions stay unchanged in the Data Dictionary. If you want to remove all the tables from a datasource, just sort column 'Datasource', select the tables and click 'Remove from list'.

Database schema (optional)

If database supports schemas, you will see all the tables from your default schema after connection. But, if you use a table from another schema in your application, you must specify appropriate schema to successfully read table structure.
