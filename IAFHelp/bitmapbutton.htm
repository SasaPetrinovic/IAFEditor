Bitmap Button
   

The bitmap button has the same function as push button. Visually,  this object displays text and/or an image.

To insert new bitmap button on the form, click button BitmapButton in toolbox or choose menu command Objects/BitmapButton. Move the cursor to the desired position,  then click.
Definition dialog appears on the screen. Choose one of images, enter text and click OK.
New bitmap button is placed on the form.

To change button properties double click on the button

-or-

select the button, right click to display context menu and choose Properties.
