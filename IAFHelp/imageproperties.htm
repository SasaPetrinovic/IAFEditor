Image properties
   

Object name

When the new image is created, generator automatically assigns the name Image_nnn, where nnn = 1, 2, …
Object name can be maximum 30 characters long. Allowed characters are A-Z, 0-9, _ and $.
First character must be a letter. Object name is case-insensitive. Object name can be used in Methods.

Member of group (information only, cannot be changed)

If the object is a member of a group, the name of this group is displayed. An object can be the member of one group only.

Picture

Specifies the path of the picture file. Double-click brings up file dialog. The default folder is IAFRES. You should copy all the pictures, which will be used in application, to the IAFRES folder.
Only ICO, BMP, JPEG and GIF images can be displayed. GIF animations are allowed, too. If you use a GIF animation, the form painter will display just the first image (frame), but at run-time real animation will be displayed.

The # sign in the first position of the path (for example, #myimage.gif) denotes IAFRES folder.

Visible

Determines whether the object is visible when the subform is first displayed.

Picture type

  
Embeded
Whole picture is stored into the EXE as a resource. Using this option, your EXE will be larger.
Linked
Picture path is stored in the program. At run-time, picture file must exist on the computer. In this case, your EXE will be smaller, but you need additional files at run-time.


Size mode

Values:

Clip                  Displays picture at actual size.
Stretch              The picture fills entire frame, proportions can be distorted.
Zoom                The picture fills entire frame, proportions preserved.

Background style

Normal
Transparent

Background color

This attribute is applied only if the background style is normal. Double click brings up a color definition dialog.

Special effects

Values:

Flat
Raised
Sunken
Etched
Bump
Shadowed

Border style

Values:

Transparent      border is not seen,
Solid                 the border is standard line,
Dots                 the border is dotted line,
Dashes             the border is dashed line.

Border color

Is applied only if border style is not transparent.  Double click brings up a color definition dialog.

Border width

Is applied only if border style is not transparent. Possibly values are 0, 1, 2,... pixels.

Picture alignment

Valid if size mode is Clip. Specifies picture position within defined frame.

Top Right
Center
Bottom Left

Auto play at run-time

If you want to automatically start GIF animation, set this option. If this value is FALSE, the Start method must be used somewhere in the program to start animation.

Z-Order

Possible values: 0, -1, -2, -3, ... See Z-Order.
