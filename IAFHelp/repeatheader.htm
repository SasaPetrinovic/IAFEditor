Repeatheader
   

This method repeats table heading on all pages. This is valid only for tables that are divided into several pages.

Format

            Repeatheader (rep_flag)

Parameters

·     rep_flag - numeric. Possible values: 0 or 1. Value 1 forces table heading printing on the next pages. Value 0 removes table heading from the next pages.

Return Value

            Status, numeric variable.

            Possible values:

            0          Ok
            -1         Document not opened
            
Applied To

            Word object.

Note

            Table heading is the first row of the table.


Examples
  
      ...
      $stat = Word.Opentable("#1",0,0)
!     ... Fill opened table
      $stat = Word.Repeatheader(1)
      $stat = Word.Closetable()
      ...
