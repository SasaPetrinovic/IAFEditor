New program name dialog
   

This dialog appears on the screen if we create new program.


Enter program name and click OK.

Program name is maximum 9 characters long. Allowed characters: letters A-Z, digits 0-9, underline (_) and dollar sign ($). The first character cannot be dollar sign.


Error messages

Invalid program name
Program already exists
Non-unique program name


To return to the painter, press ESC or click Cancel.
