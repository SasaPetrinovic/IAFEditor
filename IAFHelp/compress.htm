COMPRESS
   

This function creates a concatenation of many values adding one blank at the end of each input value. Input val_exp are left and right trimmed. If input variables are numeric or date/time ones, default editing will be performed.

Format

            COMPRESS(val_exp1, val_exp2, …, val_expN)

Return Value Type

            Alpha.

Parameters


·     val_exp1,..., val_expN - input variables. Maximal number of input values is 512.


Note

            If the length of an input value is 1, run-time engine supposes that the input value is
            an user-supplied separator and does not insert one blank space before it.

            Default edit mask for numeric variables includes leading minus, see edit mask.

Examples

  
      $C = COMPRESS("ABC", "   AB", "XX") !! result is "ABC AB XX"
      $C = COMPRESS("ABC", ",", "XX")     !! result is "ABC,XX"
      $C = COMPRESS(counter_1, acc_1, counter_2)
