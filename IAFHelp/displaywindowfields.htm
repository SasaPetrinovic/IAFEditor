DISPLAY_WINDOW_FIELDS
   

This statement displays fields in the subform (fields have to be filled, usually with SELECT or READ statement).

Format


            DISPLAY_WINDOW_FIELDS

Note

What the DISPLAY_WINDOW_FIELDS does is:

- Sets the subform to display mode (stores 1 to IAF_DISPLAYMODE).
- Moves from one active object to another, by sequence, executing all the statements. When comes to DESTINATION statement, value from the field buffer displays on the screen and continues.
- After the last object in sequence is processed, stores 0 to IAF_DISPLAYMODE and sets the focus to the first active object following the object containing the DISPLAY_WINDOW_FIELDS statement.


Example

  
      Destination = $seg1
      Select * from tbl1 Where kseg = :$seg1
      If record tbl1 found
            display_window_fields
      End_if
