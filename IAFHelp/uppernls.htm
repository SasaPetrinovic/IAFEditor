UPPER_NLS
   

This function converts all characters in alpha_exp to upper case.

Format

            UPPER_NLS(alpha_exp, code_page)

Return Value Type

            Alpha.

Parameters


·     alpha_exp - alpha field to be converted.
·     code_page - a valid code page


Examples

  
      $res = Upper_nls('abcd',1252) !! Result = ABCD
      $res = Upper_nls('abcdšč',1250)     !! Result = ABCDŠČ
      $res = Upper_nls('abcdüö',1252)     !! Result = ABCDÜÖ
