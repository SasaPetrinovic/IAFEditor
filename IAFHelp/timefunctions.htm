Time Functions
   


ADD_HOURS adds number of hours to a time.
ADD_MINUTES adds number of minutes to a time.
ADD_SECONDS adds number of seconds to a time.
DIFF_TIME returns time difference between two time values.
HOUR returns hours of a time.
MAKETIME returns time value calculated from the hour, minute and second.
MINUTE extracts minutes of a time.
SECOND returns seconds of a time.
SUB_HOURS subtracts a number of hours from a time.
SUB_MINUTES subtracts a number of minutes from a time.
SUB_SECONDS subtracts a number of seconds from a time.
TIME converts a numeric or an alphanumeric field to a time.
