MID
   

This function extracts specified number of characters from an alpha field.

Format

            MID(alpha_field, start[, count])

Return Value Type

            Alpha.

Parameters


·     alpha_field - source alpha field, ANSI or UTF-16.
·     start - starting position of the substring, integer literal or integer variable. The leftmost character has position 1.
·     count - number of characters to extract, integer literal or variable. Optional argument. If this argument is omitted, than the rest of the string is extracted.

Note

            If start <= 0 or start > field_length or count <= 0, output string will be initialized.


Examples

  
      $res = Mid($inp, 5, 3)
      $res = Mid($inp, 5)
      $res = Mid($inp, 0, 5)  !! error, start = 0
      $res = Mid($inp, 1, 0)  !! is the same as init $res
