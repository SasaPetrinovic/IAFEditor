MAKEDATE
   

This function returns date value calculated from the year, month and day.

Format

            MAKEDATE(year,month,day)

Return Value Type

            Date.

Parameters

·     year - numeric_exp
·     month - numeric_exp (from 1 to 12)
·     day - numeric_exp (from 1 to number of days in the month)
Note

            If month or day arguments are invalid, result is zero_date.

Examples
  
      $Date = makedate(2014,1,1) !! result is '1-jan-2014'
      $Date = makedate(2014,0,1) !! result is zero_date
      $Date = makedate(4,1,1) !! result is '1-jan-0004'
