SPLIT
   

This function splits an alpha field by separator into alpha array.

Format

            SPLIT(separator,alpha_field)

Return Value Type

            Alpha array.

Parameters

·     separator - alpha field, length = 1

Note

            The function can be used in an assignment only. Syntax is:

                        array() = SPLIT(separator, alpha_field)

Examples
  

If $arr is an alpha array, 10 elements, the following statement

$arr() = SPLIT(";","string1;string2;string3")

gives the result

$arr(1) = "string1"
$arr(2) = "string2"
$arr(3) = "string3"
$arr(4) = $arr(5) = ... = space
