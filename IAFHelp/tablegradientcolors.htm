Table gradient colors
   

In this dialog we define gradient style and gradient colors for grid headings.


Shading style

Specifies that Color_from fades gradually to Color_to.

Values:

None                No shading
Horizontal
Vertical

Shading variant

Valid if shading style is Horizontal or Vertical.
Possible values are 1 to 4:

1          Ascending
2          Descending
3          Ascending/Descending
4          Descending/Ascending

Corner button Color to

Corner button ending color. Valid if shading style is Horizontal or Vertical.

Side heading Color to

Side heading ending color. Valid if shading style is Horizontal or Vertical.

Top heading Color to

Top heading ending color. Valid if shading style is Horizontal or Vertical.


Note: in the all cases, Color_from is heading background color.
