To use database tables in our programs, we have to load their definitions into the DD.

Synchronize table definitions

Compare table structures in the Data Dictionary to table structures in the database. Tables (synonyms, views, indexes, ...) in database are created by a Database administration utility. To use these tables, we load definitions into the DD. If someone later changes table structure, our DD definition will be out of date. This is the reason why we need synchronization from the time to time.
