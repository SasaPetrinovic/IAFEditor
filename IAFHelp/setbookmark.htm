Setbookmark
   

 This method transfers a value from program to an empty bookmark in Word document.

Format

            Setbookmark(bookmark, variable [,code_page])

Parameters

·    bookmark - an alpha variable or alpha literal. Specifies name of the bookmark in Word document.

·    variable - value which will be written into the bookmark. If the variable is not an alphanumeric one, default editing will be applied.

·    codepage - this parameter specifies variable's code page. The parameter is very important because MS Office tools (Excel, Word, ...) use             UNICODE to represent text data, while text data in programs use, in most cases, single-byte representation. Default is code page set by previous Setcodepage method. If Setcodepage was not used, default is ANSI code page.

Return Value

            Status, numeric variable.

            Possible values:

            0          Ok
            -1         Document not opened
            -2         Bookmark not found
            
Applied To

            Word object.

Examples
  
      $stat = Word.Setbookmark("date_from",today)
      $stat = Word.Setbookmark("date_to",tomorrow)
      $stat = Word.Setbookmark("amount",edit(acc_1,"zzz.zz9,99"))
      $stat = Word.Setbookmark("name",$name,1250)
 

Using Word
