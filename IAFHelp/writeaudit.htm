WRITE_AUDIT
   

This statement writes record in journal file.

Format 1

            WRITE_AUDIT conc_expr

Format 2

            WRITE_AUDIT /NAME: "file_name" conc_expr

Format 3

            WRITE_AUDIT /NAME: $local_variable  conc_expr

Where


·     file_name - a physical name of a journal file. If this qualifier is omitted, the name of audit trail file from Change setup parameters dialog will be taken.

·     conc_exp - concatenation, which has to be written in journal file. Size of a concatenation must not be greater than 250 characters.

·     local_variable - the standard local alpha variable. This variable must be filled by appropriate file name before invoking this statement.

Note

            The journal file is a standard ANSI text file. Such files can be reviewed by any text editor.


Examples

  
      Write_audit "The audit text"
      Write_audit "The audit text: " | counter_1
      Write_audit/name: "aa.atf" "The audit text"
      $locatf = "c:\temp\aa.atf"
      Write_audit/name: $locatf "The audit text"
