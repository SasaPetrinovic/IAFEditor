Example 1 - Copy TBL1 (DSN1) to TBL2 (DSN2)
   

Data Dictionary actions:

Load TBL1 definitions into your Data Dictionary.
Load TBL2 definitions into the same DD.


Program generator actions:

In 'Define datasource' dialog  select TBL1 (for read), TBL2 (for write):

            TBL1, access=read
            TBL2 /DSN: DSN2;UID=user;PWD=pwd, acess=write

Select "Connect to multiple datasources" check box.

Program code could be

      reset tbl2
      select_all * from tbl1
            tbl2.fld21 = ...
            ...
            tbl2.fld2N = ...
            write TBL2
      end_select
      commit

Distributed queries are not allowed (for example, SELECT command from both datasources).
