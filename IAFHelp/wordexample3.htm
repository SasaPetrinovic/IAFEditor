Example 3
   

Prepare data file for mailmerge, activate mailmerge method and save document under new name. We suppose that merge_template.doc has been previously created and supplied by necessary mailmerge information.


$wstat = Word.Mergeopen("docdir:mergefile.txt")
if $wstat = 0
! write field names
      $wstat = Word.Mergedata("fld1")
      $wstat = Word.Mergedata("fld2")
      $wstat = Word.Mergedata("fld3")
      $wstat = Word.Mergewrite()
! write fields itself
      $wstat = Word.Mergedata($name)
      $wstat = Word.Mergedata(today)
      $wstat = Word.Mergedata(edit(acc_1,"------9,99"))
      $wstat = Word.Mergewrite()
! close data file
      $wstat = Word.Mergeclose()

! open tempate document
      $wstat = Word.Open("docdir:merge_template.doc")
      if $wstat = 0
            $wstat = Word.Run("mailmerge_save","outdir:newname.doc")
      end_if
else
      Pause "Error creating data file"
end_if

After the Run instruction, file outdir:newname.doc is created and Word is active.


Note

Using methods Mergeopen, Mergedata, Mergewrite and Mergeclose is the simplest way to create sequential, field delimited, files. Field delimiter is TAB, data format is Unicode.
