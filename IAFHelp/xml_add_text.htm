XML_ADD_TEXT
   

This function adds text to a XML stream.


Format

            XML_ADD_FIELD(xml_variable, text)

Result Value Type

            Xml variable (type = S).

Parameters


·     xml_variable - alpha variable containing XML stream. Variable type is CLOB, longchar (type = S).

·     text - text to be inserted into XML stream, alpha literal or alpha variable.

Note

            Alpha ANSI text must be in UTF-8 format.
            Alpha UTF-16 text will be converted into UTF-8.

            Trailing blanks will be removed.


Examples

  
! Local field $xml has type S
      init $xml
      $xml = xml_add_text($xml,'Free text')
      $xml = xml_add_text($xml,'<field_1>xxx</field_1>')
