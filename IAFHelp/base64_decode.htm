BASE64_DECODE
   

This function transforms data back from base64 format to the original data.

Format

            BASE64_DECODE(base64_encoded_data, data_length)

Return Value Type

            Alpha.

Parameters

·     base64_encoded_data - input data in base64 format.
·     data_length - number of input bytes.


Example
  
            $out = BASE64_DECODE($base64,str_length($base64))
