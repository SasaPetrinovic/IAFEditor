Setalignment
   

This method sets cell alignment (Excel).

Format 1

            Setalignment (range, alignment)

Format 2


            Setalignment (line, column, alignment)


Parameters


·     range - range of cells in the form STARTING_CELL:ENDING_CELL, alpha variable. ENDING_CELL can be omitted.
·     alignment - alpha variable, alignment in the form
                                    HORIZONTAL = {CENTER | LEFT | RIGHT | JUSTIFY} or
                                    VERTICAL = {CENTER | TOP | BOTTOM | JUSTIFY} or
                                    ORIENTATION = number_of_degrees {-90 to 90}

                        HORIZONTAL, VERTICAL and ORIENTATION can be abbreviated.


·     line - row, numeric value, integer. The first line is 1, etc.

·     column - column, numeric value, integer. The first column is 1.



Return Value

            Status, numeric variable.

            Possible values:

            0          Ok
            -1         Document not opened
            -2         Invalid range
            -3         Invalid alignment

Applied To

            Excel object.


Examples

  
      $status = Excel.Setalignment('A1','H=center')
      $status = Excel.Setalignment('A1','HORIZ=right')
      $status = Excel.Setalignment('A1','V=center')
      $status = Excel.Setalignment('A1','VERT=center')

      $status = Excel.Setalignment(1,1,'VERT=center')

      $status = Excel.Setalignment(1,1,'ORIENT=90')
      $status = Excel.Setalignment(1,1,'ORIENT=45')
      $status = Excel.Setalignment(1,1,'ORIENT=-45')
      $status = Excel.Setalignment('A1:X1','ORIENT=-90')
