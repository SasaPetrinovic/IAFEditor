TIME
   

This function converts a value expression to a time field.

Format

            TIME(val_exp)

Return Value Type

            Time.

Parameters


·     val_exp - time literal, alpha literal, numeric literal, alpha field or datetime.

Note 1

            The input field must be in the proper format. Alpha and numeric literals must have 4 or 6 digits.

Note 2

            If 'val_exp' is a datetime field, the function simply returns the time part of the field.

Note 3

            The instruction time_field = TIME(datetime_field)
            has the same effect as time_field = datetime_field

            See Notes in Assignment operator.


Examples

  

      $time_field = time("10:59")   !! time literal
      $time_field = time("10:59:30") !! time literal
      $time_field = "10:59:30"      !! same as above. In this case
!we can use simple assignment because the left and right operands
!are compatible.
!In the next lines we MUST use TIME function.
      $time_field = time("1059")    !! alpha literal
      $time_field = time("105930")  !! alpha literal
      $time_field = time("171215")  !! alpha literal
      $time_field = time("10593")   !! invalid, 5 digits
      $time_field = time(1059)      !! numeric literal
      $time_field = time(0815)      !! numeric literal
      $time_field = time(142530)    !! numeric literal

      if $a6 time
            $time_field = time($a6)
      end_if
