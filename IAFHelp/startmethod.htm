Start
   

This method starts the timer or GIF animation.

Format

            Start()

Parameters


·     none

Applied To

            Timer, Image,  Pushbutton, Bitmapbutton.


Note

·           If the timer is already started, this method will be ignored. Timer interval is defined statically (Timer interval property) or dynamically by Settimerinterval method.


·           To use an implicit animation on button click (pushbutton, bitmapbutton), following conditions should be satisfied:

            - Folder c:\iafw\iafres must hold file ANI_PROGRESS.GIF.
            - Define symbol ANI_BUTTON in HLM\Software\Sing\Iafw\1.0. Symbol value is #ani_progress.gif.

Examples

  
      button_1.start() !! start implicit animation
      ...
      ... long lasting operations
      ...
      button_1.stop() !! stop implicit animation
