SUB_WEEKS
   

This function subtracts a number of weeks from a date.

Format

            SUB_WEEKS(date,numweeks)

Return Value Type

            Date.

Parameters

·     date - date_exp
·     numweeks - number of weeks, numeric_exp
Note

            1 week = 7 days.

            SUB_WEEKS(date,numweeks) = SUB_DAYS(date,7*numweeks)
