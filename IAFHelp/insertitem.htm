Insertitem
   

This method inserts an item into an object.

Format


            Insertitem(text_label, data)


Parameters


·     text_label - text to be displayed.
·     data - data to be stored into this node. When user selects an item, the data will be transferred to appropriate variable.

Applied To

            ComboBox, data selector, tree (initial filling).

Note

            Each item has two parts: text and associated data. All texts are displayed by an object. When user selects an item, associated data will be moved to program variable.


            The method is obsolete for ComboBox, use InsertItemEx.

Example

  

! Example 1: Fill-in a combo box
      combo_1.Insertitem("Item 1","01")
      combo_1.Insertitem("Item 2","02")
      combo_1.Insertitem("Item 3","03")
      combo_1.Insertitem("Item 4","04")
      combo_1.Insertitem("Item 5","05")

! At run-time, if user selects "Item 3", value "03" will be moved to
! the variable defined by "Destination = prog_var"
 
! Example 2: fill-in combo box from the table
      combo_1.Deleteallitems()
      select_all * from tbl1 where fld1=100
            combo_1.InsertItem(tbl1.text, tbl1.data)
      end_read
