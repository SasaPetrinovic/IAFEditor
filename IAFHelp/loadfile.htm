LOAD_FILE
   

This function loads a file into an alpha field or a long field (LONGVARCHAR or LONGVARBINARY) of an RDB table. Result is a numeric_exp containing status. If successful, status = 0.

Format

            LOAD_FILE(file_name,field)

Return Value Type

            Numeric.

            Possible values:
                        0          load successful,
                        2          error: no such file or directory
                        13        error: permission denied

Parameters

·    file_name - file name, alpha literal or alpha variable. Logical names allowed.
·    field - result field. If the field is alpha one and its length is less than file size, truncation occurs. If the field is LONG, its length will be adjusted automatically to include whole file.

Example

  
!
!This function loads file c:\test\a.txt into the field MEMO
!
! $status is defined as numeric, without decimals.
!
$status = LOAD_FILE("c:\test\a.txt",tbl1.memo)
if $status <> 0
      pause "Load error" | $status
end_if

!This piece of the code inserts one row into the RDB table tbl1:

tbl1.col1 = 1
tbl1.col2 = today
tbl1.col3 = program_name
$status = LOAD_FILE("c:\test\a.txt",tbl1.memo)
if $status <> 0
      pause "Load error" | $status
      tbl1.memo = NULL        !! null value
end_if
write tbl1
