PROCEDURE
   

This statement declares a beginning of a procedure.

Format

            PROCEDURE procedure
            ...
            statements
            …
            END PROCEDURE

Where


·     procedure - the name of the procedure
·     statements - IAF instructions which are used to define a procedure

Note

            The procedures can be local and global ones. The local procedures can be used only in the program in which they are defined. A programmer defines these procedures by choosing Define/Local procedure(s), function(s) in main menu. The global procedures are visible from all the programs in the Data Dictionary and can be defined by Define/Global procedure(s), function(s).

Example

  
      Procedure myproc
            init acc_2
            read file1 key=1
            if record file1 found
                  acc_2 = 100,00 * acc_1 / (215,00 - fld1)
            end_if
      End_procedure
