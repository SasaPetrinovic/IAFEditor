Setbackcolor
   

This method sets the background color of a range of Excel cells.

Format 1

            Setbackcolor (range, color)

Format 2


            Setbackcolor (line, column, color)


Parameters


·     range - range of cells in the form STARTING_CELL:ENDING_CELL, alpha variable. ENDING_CELL can be omitted.

·     color - a numeric variable with valid RGB color value or one of the predefined colors or RGB function.

·     line - row, numeric value, integer. The first line is 1, etc.

·     column - column, numeric value, integer. The first column is 1.

Return Value

            Status, numeric variable.

            Possible values:

            0          Ok
            -1         Document not opened
            -2         Invalid range
            -3         Error

Applied To

            Excel object.

Examples

  
      $status = Excel.Setbackcolor('a1','black')
      $status = Excel.Setbackcolor('a1:f1','red')
      $lin = 1
      $status = Excel.Setbackcolor(compress_trim('A',$lin,':','F',$lin),'red')
      $status = Excel.setbackcolor('a1:a1',RGB(255,0,0))
      $status = Excel.Setbackcolor('a1:a10',255)

      $status = Excel.Setbackcolor(1,5,255)
      $lin = 1
      $col = 5
      $status = Excel.Setbackcolor($lin,$col,'red')
