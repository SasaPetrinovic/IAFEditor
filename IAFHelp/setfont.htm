Setfont
   

This method changes font attributes of an object.

Format 1

            Setfont(font_name, font_size, bold, italic, underline)

Format 2

            Setfont(line, column, font_name, font_size, bold, italic, underline)

Parameters


·    font_name - alpha literal or variable, font name: “Arial”, “Times New Roman”, ….
·    font_size - an integer, font size (8 or 10 or 12 or ...).
·    bold - intensity, an integer from 0 to 1000 (normal intensity is 400, bold 700, extra bold 900).
·    italic - integer, value 1 = italic,  value 0 = normal.
·    underline - integer, value 1 = underline, value 0 = normal.

·     line - grid line, numeric value. The lines are counted from 1.To specify current line, we can use the internal variable CURRENT_LINE or star (*).
·     column - grid column, numeric value. The columns are counted from 1. If this parameter has the value -1, the entire line will be changed.

Applied To

            All the objects displaying text.

Notes


·          Unused parameters can be omitted.

·          The font of a grid cell cannot be changed.

·     Use Format 2 to change font of a grid cell.

Examples

  
      obj.SetFont("Arial") !! change to Arial
      obj.SetFont("Arial",,700) !! change to Arial, bold
      obj.SetFont(,14) !! set font size
      obj.SetFont(,,,1) !! set to italic
      obj.SetFont(,,,,1) !! set to underline

      table_1.SetFont(current_line,1,'Arial')
      table_1.SetFont(current_line,1,,16)
      table_1.SetFont($line,$col,,16,700)
