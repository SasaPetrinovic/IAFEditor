SHIFT_ARRAY
   

This function shifts array elements left or right, simultaneously placing the element to be removed into the result variable. On exit, array is shifted and removed element is placed to the result.


Format

            SHIFT_ARRAY(array, shift_direction[, starting_index])

Return Value Type

            Any type.

Parameters


·     array - array to be shifted, any type.
·     shift_direction - shift direction, integer. Value = 0 left shift, value <> 0 right shift.
·     starting_index - index from which shift starts, integer. The first element has index 1. If omitted, value is 1. If the index is < 1 or > array_length, result is initialized and array stays  unchanged.


Algorithm

            Let's consider a local array $ARR with 7 elements.

            SHIFT_ARRAY($ARR,0) works as
                        Result = $arr(1)
                        $arr(1) = $arr(2)
                        $arr(2) = $arr(3)
                        $arr(3) = $arr(4)
                        $arr(4) = $arr(5)
                        $arr(5) = $arr(6)
                        $arr(6) = $arr(7)
                        $arr(7) initialized

            SHIFT_ARRAY($ARR,0,4) works as
                        Result = $arr(4)
                        $arr(1) unchanged
                        $arr(2) unchanged
                        $arr(3) unchanged
                        $arr(4) = $arr(5)
                        $arr(5) = $arr(6)
                        $arr(6) = $arr(7)
                        $arr(7) initialized

            SHIFT_ARRAY($ARR,1) works as
                        Result = $arr(7)
                        $arr(7) = $arr(6)
                        $arr(6) = $arr(5)
                        $arr(5) = $arr(4)
                        $arr(4) = $arr(3)
                        $arr(3) = $arr(2)
                        $arr(2) = $arr(1)
                        $arr(1) initialized

            SHIFT_ARRAY($ARR,1,4) works as
                        Result = $arr(7)
                        $arr(7) = $arr(6)
                        $arr(6) = $arr(5)
                        $arr(5) = $arr(4)
                        $arr(4) initialized
                        $arr(3) unchanged
                        $arr(2) unchanged
                        $arr(1) unchanged

            SHIFT_ARRAY($ARR,1,10) works as
                        Result initialized

Examples

  
      $res = shift_array($arr,0) !! shift left whole array
      $res = shift_array($arr,0,4) !! partially shift left
      $res = shift_array($arr,1) !! shift right whole array
      $res = shift_array($arr,1,4) !! partially shift right
