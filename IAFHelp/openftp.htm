Open
   


This method establishes connection to a FTP server. Must be invoked before any other Ftp method.

Format

            Open(server, username, password [, mode [, port]])

Parameters


·     server - an alpha variable or alpha literal, required. Specifies IP address or name of the FTP server.

·     username - name of the user, alpha variable or alpha literal, required.

·     password - password, alpha variable or alpha literal, required. To specify blank password, enter empty string.

·     mode - mode, alpha variable or alpha literal, optional. Two values are allowed: "active" or "passive". If omitted, default value is "active". If you want to avoid client's firewall problems, use PASSIVE mode.

·     port - TCP/IP port to use on the server, integer, optional. Default value is 21.


Return Value

            Alpha variable. If connecting was successful, value is blank. Otherwise, the variable holds an error message.


Applied To

            Ftp object.

Note

            

Example

  
      $msg = Ftp.Open('myserver','user1','pwd1')
      if $msg <> SPACE
            Pause 'Error: ' | $msg
      else
            Ftp.Close()
      end_if
