WARMSG
   

This statement displays warning message and continues program execution (moves input focus to the next field according to the sequence number).

Format

            WARMSG = message

Where


·     message - concatenated expression or number of the message from message file or an user defined function.

Example

  
      Destination = fld1
      Check fld1 >= 0
      WARMSG = "Entered number is negative !!!"
 


See Also ERRMSG  ERRWARMSG
