CEIL
   

This function rounds a numeric value.

Format

            CEIL(numeric_exp, ceil_factor)

Return Value Type

            Numeric.

Parameters


·    numeric_exp - numeric value to be rounded.
·    ceil_factor - ceil factor, numeric.


Note

            Ceil_factor is round position. If ceil_factor >= 0, integer part of numeric_exp
            will be rounded. If ceil_factor < 0, decimal part of numeric_exp will be rounded.

            Ceil_factor is defined as follows:

  
Numeric value
1
1
8
2,
5
6
Ceil factor
3
2
1
0
-1
-2

            The first digit after decimal point has ceil_factor -1.
            The first digit left from decimal point has ceil_factor 0.

            Ceil factor -3 means: round the third decimal to 0.
            Ceil factor -2 means: round the second decimal to 0.
            Ceil factor -1 means: round the first decimal to 0.

            Algorithm:

            If the digit at round position has value less than 5, it will be rounded to 0. All digits
            right from this position will be set to 0. All digits left from this position stay
            unchanged.

            If the digit at round position has value greater or equal 5, it will be rounded to 0,
            but first digit left from this position will be incremented by 1. All digits right
            from this position will be set to 0.

Examples
  
      acc_1 = 100,56
      $C = CEIL(acc_1,-2) !! result is 100,60
      $C = CEIL(acc_1,-1) !! result is 101,00
      $C = CEIL(acc_1,0) !! result is 100,00
