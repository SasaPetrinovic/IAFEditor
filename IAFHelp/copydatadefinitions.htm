Copy data definitions
   


List


List on the left-side of the dialog shows all tables, synonyms and views belonging to the selected ODBC datasource.

To copy tables definitions from database to your DD, just select one or more tables and click 'Copy to DD'. A view will be considered as an ordinary table.

Select all

Click this button to select all the tables/views/synonyms in the list.

Copy to DD

Click this button to copy definitions to Data Dictionary.

Consider text data as Unicode

This check-box is visible if we load definitions from MS Access database only. If the control is checked, all text fields will be described in Data Dictionary as Unicode. This option is recommended because the text data in MS Access are stored in Unicode format. Otherwise, text fields in Data Dictionary will be described as ANSI fields. This solution can bring us some difficulties at run-time.
