Setcodepage
   

This method sets the code page for subsequent "set" methods.

Format

            Setcodepage(codepage)

Parameters

·    codepage - a numeric variable or numeric literal. Denotes valid code page.

Return Value

            Status, numeric variable.

            Possible values:

            0          Ok
            -1         Document not opened
            
Applied To

            Word object.

Examples

  
      $stat = Word.Setcodepage(1250)

      $stat = Word.Setbookmark("b1",fld1)
      $stat = Word.Setbookmark("b2",fld2)
      $stat = Word.Setbookmark("b3",fld3,1252) !! overrides 1250
