Button
   

The push button is an object enabling the user to execute some actions by clicking it.

To insert new push button on the form, click button Button in toolbox or choose menu command Objects/Button. Move the cursor to the desired position,  then click. This is the top left corner of the button. Drag to the right and bottom and release the button. New push button with default text "Button" is placed on the form. The code buffer is empty.

The code (actions to be performed when user clicks the button at run-time) can be changed by context menu command "Modify code".

To change button properties double click on the button

-or-

select the button, right click to display context menu and choose Properties.
