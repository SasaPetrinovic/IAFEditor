LETTERS
   

This function returns textual representation of a number.

Format

            LETTERS(numeric_exp, language)

Return Value Type

            Letters.

Parameters

·     language - alphanumeric literal denoting language. Allowed values are: "ENG" for english, "MAK"  for macedonian, "SLO"  for slovenian, "SRB" for serbian.
Examples
  
      $out80 = letters(acc_1, "eng")

      !! If acc_1 contains 1234,50 result is:
      !! "one thousand two hundred thirty four 50/100"
