PAUSE
   

This statement pauses a program displaying a message box.

Format

            PAUSE [conc_exp]

Where


·     conc_exp - a literal, variable or concatenated expression, which will be displayed in the message box.

Example

  
      Pause       !! displays text "Pause"
      Pause "Text to be displayed"
      Pause "Today is " | today
      Pause "Ammount = " | acc_1
