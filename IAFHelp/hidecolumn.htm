Hidecolumn
   

This method hides columns of a grid.

Format

            Hidecolumn(column1, column2, ..., columnN)

Parameters


·    column1, column2, columnN - a numeric variable or numeric literal, integer.

Applied To

            Grid.

Note


            Column = -1, hides side heading.
            Column = 1, 2, … hides a column. The first column is 1.

Display-only grid

            This method simply hides one or more columns.

Data-entry grid

            If a column denotes source field, this method hides the column only.

            If a column denotes one of active fields (destination field, combo box or check box), this method hides the column. Also, statements from the 'Destination' up to the end of the field buffer will be ignored. Variable in the 'Destination' statement will have unknown value.

Field code:

...
before_instructions
...
Destination = my_variable
...
after_instructions
...
last_instruction_in_the_buffer

All the statements from Destination to the last_instruction_in_the_buffer will be ignored. Variable my_variable has unknown value.


Example

  
            Table_1.Hidecolumn(5)
            Table_1.Hidecolumn(5,6,10)
            Table_1.Hidecolumn(5,counter_1,counter_5)
