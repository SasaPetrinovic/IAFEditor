Tab control
   


The tab control is an object consisting of a number of pages that occupy the same space.

To insert new tab control on the form, click button Tab control in toolbox or choose menu command Objects/Tab control. Move the cursor to the desired position,  then click. This is the top left corner of the tab control. Drag to the right and bottom and release the button. New tab control with three pages is placed on the form.

Note: Tab control cannot be created within another tab control.


To change tab control properties double click on the tab control

-or-


select the tab control, right click to display context menu and choose Properties.



Add new page

Right click tab control and select 'Add TabPage' in context menu.

Remove page

Right click tab control and select 'Remove TabPage' in context menu.

Change page position

Right click tab control and select 'Properties' in context menu.
