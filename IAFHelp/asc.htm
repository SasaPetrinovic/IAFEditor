ASC
   

This function returns the code value of the leftmost character of alpha_exp.

Format

            ASC (alpha_exp)

Return Value Type

            Numeric.

Parameters


·     alpha_exp - alpha field, ANSI or UTF-16.


Notes

            If alpha_exp is ANSI field, result is a number in the range 0 to 255.
            If alpha_exp is UTF-16 field, result is number in the range 0 to 65535.

Examples

  
      $N = ASC('1') !! result is 49
      $N = ASC('A') !! result is 65
      $N = ASC('ABC') !! result is 65
      $N = ASC('@') !! result is 64

! $UTF16 is defined as
! $UTF16 W 1
      $UTF16 = 'Š'
      $N = ASC($UTF16) !! result is 352
