String Functions
   


ALLTRIM removes white space from both ends of alpha_exp.
ALPHA converts any field to alphanumeric one.
ASC returns the ASCII code value of the leftmost character of alpha_exp.
CHAR_OF extracts one character from an alpha_exp.
CHR returns the character that has the ASCII code value specified by numeric_exp.
CHRW returns the wide character that has the code value specified by numeric_exp.
COMPRESS creates a concatenation of many alphanumeric values.
COMPRESS_CSV creates a concatenation of many trimmed (left and right) alphanumeric values.
COMPRESS_SEP creates a concatenation of many trimmed (left and right) alphanumeric values, adding a separator at the end of each input value.
COMPRESS_TRIM creates a concatenation of many trimmed (left and right) alphanumeric values.
CONVERT converts an alpha_exp from one character set to another.
EDIT returns formatted representation of numeric, date or time variable.
ENVVAR returns value of an environment variable.
INITCAP converts the first letter of each word in uppercase, and all other letters converts in lowercase.
INITCAP_NLS converts the first letter of each word in uppercase, and all other letters converts in lowercase.
LEFT extracts a substring beginning with the first character in alpha_exp.
LETTERS returns textual representation of a number.
LIST returns value from list of values.
LOWER converts all characters in alpha_exp to lower case.
LOWER_NLS converts all characters in alpha_exp to lower case.
LPAD returns alpha field, left-padded with another string.
LTRIM removes white space from the start of alpha_exp.
MID extracts a substring from an alpha field.
REPLACE replaces all occurrences of one string by another string.
RIGHT extracts a substring beginning with the rightmost character in alpha_exp.
RPAD returns the alpha_exp, right-padded with another alpha_exp.
RTRIM removes white space from the end of alpha_exp.
STR_FILL fills alpha variable by 'fill string'.
STR_INDEX returns the position of a character in an alphanumeric value.
STR_LENGTH returns the position of the last nonblank character in an alphanumeric value.
SUBSTR_INDEX returns the position of the first occurrence of a substring.
TRANSLITER converts a text from Latin script to Cyrillic script and vice versa
TRNLOG translates a logical name.
UPPER converts all characters in alpha_exp to upper case.
UPPER_NLS converts all characters in alpha_exp to upper case.
