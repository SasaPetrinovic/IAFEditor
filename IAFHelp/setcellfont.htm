Setcellfont
   

This method changes font attributes of an Excel cell.

Format 1

            Setcellfont (range, [font_name],  [font_size], [bold], [italic], [underline])

Format 2

            Setcellfont (line, column, [font_name],  [font_size], [bold], [italic], [underline])

Parameters


·     range - range of cells in the form STARTING_CELL:ENDING_CELL, alpha variable. ENDING_CELL can be omitted.

·     line - row, numeric value, integer. The first line is 1.

·     column - column, numeric value, integer. The first column is 1. Instead of column number, an alpha literal containing column header can be used: 'A', 'B', ..., 'AA', 'AB', ...

·     font_name - alpha literal or variable, font name: “Arial”, “Times New Roman”, …. Optional parameter.

·     font_size - an integer, font size (8 or 10 or 12 or ...). Optional parameter.

·     bold - intensity, > 400 bold, <= 400 normal. Optional parameter.

·     italic - integer, value 1 = italic,  value 0 = normal. Optional parameter.

·     underline - integer, value 1 = underline,  value 0 = normal. Optional parameter.


Return Value

            Status, numeric variable.

            Possible values:

            0          Ok
            -1         Document not opened
            -3         Error

Applied To

            Excel object.

Examples

  
      $stat = Excel.Setcellfont(1,1,'Arial')
      $stat = Excel.Setcellfont(1,"B",,14)
      $stat = Excel.Setcellfont(1,3,,,700) !! bold
      $stat = Excel.Setcellfont(100,'A','Arial',14,700,1,1)

      $stat = Excel.Setcellfont('A100','Arial',14,700,1,1)
      $stat = Excel.Setcellfont('A100:A120','Arial',14,700,1,1)
 

Using Excel
