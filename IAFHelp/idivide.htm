IDIVIDE
   

This function returns result from the integer division.

Format

            IDIVIDE(dividend, divisor)

Return Value Type

            Numeric.

Parameters


·     dividend -
·     divisor -

Examples

  
      $N = IDIVIDE(10,3) !! result is 3
      $N = IDIVIDE(20,5) !! result is 4
