Program properties
   

Program, Version, Date created / modified: information only.


Caption

This text appears in the program title, maximum 80 characters.

Icon

Small icon displayed in the upper-left corner of the program's frame.

Background color

Background color of the program title (reports, batch) or main form if an interactive program has no subforms.

Visibility

Initial visibility of the program: normal, minimized, maximized.

Run mode

Not implemented yet.

Border style

Resizing: frame can change its size at run-time.
Thin: frame cannot change its size at run-time.

Title bar

When not checked, the program has no program title. Default: checked.

Control box

When not checked, the program has no caption buttons: "X", minimize, maximize. Default: checked.

Maximize box

When checked, the maximize button appears on the frame.

Minimize box

When checked, the minimize button appears on the frame. By clicking this button, system reduces window to the size of icon.

Status bar

When checked, the program will create status bar at the bottom of the frame. Reports always have status bar. To display a text in the status bar, use Display method, object name is STATUS_BAR.

X position

X position of the left upper corner of the program frame at run-time.

Y position

Y position of the left upper corner of the program frame at run-time.

X size

The width of the program frame.

Y size

The height of the program frame.

Autocenter

When checked, this option centers program frame relative to the screen at run-time. If checked, X position and Y position are ignored.

Topmost

Defines program position on the screen.
  
No
Default position.
Always
Places the program above all programs.
If subprogram
Places the program above all programs only if the program is activated via CALL_PROGRAM or /PROGRAM.


Use default dialog color at run-time

Message place

Use IAF program sequence

One instance only allowed

Ignore menu protection

Disable message pump in program cycle

Close program before Preview

Description

Program description, maximum 80 characters. This text appears in Program name dialog when you click 'Descriptions' radio button.
