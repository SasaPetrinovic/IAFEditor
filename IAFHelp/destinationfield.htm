Destination field
   

A destination field accepts a value from the keyboard.

To insert new destination field on the form, click button Destination Field in toolbox or choose menu command Objects/Destination Field.

Move the cursor to the desired position, then click. A dialog "Destinationfield definition" appears on the screen.


Enter the variable name (field to be accepted) or click Browse to choose a variable from a list. Click OK. If the variable exists, object DESTINATIONFIELD will be created: default edit mask is displayed on the screen, and one source line is placed in code buffer. The line is

Destination = variable_name

See Destination statement.

The destination field object has the two components: visual appearance (edit mask on the screen) and code buffer (contains statements defining the output variable, check conditions, calculations, etc.).

The edit mask can be changed by context menu command "Change edit mask". To change the code, choose context menu command "Modify code".

Maximal number of lines in the code buffer (for one object) is 1000, and only one DESTINATION statement must be encountered in the buffer.

Destination = acc_1
check acc_1 > 1000
errmsg = "Entered value must be greater than 1000"

Multiline field

Enter the variable name, check "Multiline" check box and click OK. Resize the object.
At run-time, word wrapping is allowed and explicit new line can be inserted by pressing Ctrl/Enter.

Image

Sometimes we want to store a path to an image into an alpha field. It is very easy. Enter variable name and click OK. Change edit mask to all X's. At run-time we simply enter file name into this field, for example c:\myimg\image.bmp.

But, if we want to see an image itself at run-time, not a path, then we must use different definition procedure: enter variable name, check "Image" check box and click OK. Resize the object.

An alternative way to insert new destination field

Choose Objects/Field Explorer menu command. Expand the tree in the dialog.

Click button Destination Field in toolbox or choose menu command Objects/Destination Field.

Drag the variable name from the tree to the form.
