Getcell
   

This method reads a value from MS Excel cell.

Format 1

            Getcell(range[,codepage])

Format 2

            Getcell(line,column[,codepage])

Parameters


·     range - cell in the form COLUMNline, alpha variable. Examples:  A1, A2, B1, X1, AA1, AA2, AA100, ...

·     line - row, numeric value, integer. The first line is 1, etc.

·     column - column, numeric value, integer. The first column is 1. Instead of column number, an alpha literal containing column header can be used: 'A', 'B', ..., 'AA', 'AB', ...

·     codepage - if the cell contains an alpha value, this parameter specifies variable's code page. The parameter is very important because MS Office tools (Excel, Word, ...) use UNICODE to represent text data, while text data in programs use, in most cases, single-byte representation. Default is 1252 (Western).


Return Value

            Value of the cell.

Applied To

            Excel object.

Note

            This method can be used in assignment only, on the right-hand side of the equation.

            If the cell has NULL value, output variable will be initialized.
            If line or column are invalid, output variable will be initialized, too. To avoid this situation, use Getnumberrows() and Getnumbercols() methods to get proper intervals.

            If you do not know type of the cell, use an alpha variable on the left-hand side of assignment. In that case, the value visible on the screen will be stored into the output variable as alpha string.

            But, if you know cell type (let's say, cell 1,1 contains the date), then variable of correct type can be used as output.

Examples

  
      acc_1 = Excel.Getcell(1,1)    !!line=1, col=1
      $date = Excel.Getcell(1,2)    !!line=1, col=2
      $date = Excel.Getcell(1,"B")  !!same as above
      $textdata = Excel.Getcell(1,3,1250)

      $date = Excel.Getcell('B1')         !!range
      $text = Excel.Getcell('AA100')      !!range
