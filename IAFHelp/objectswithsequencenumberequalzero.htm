Objects with sequence number equal zero
   


Destination field, check box, radio button and combo box with sequence = 0

Processing of these objects is little bit specific regarding to those objects having sequence <> 0. The program code, supporting these objects, usually has the following form:

...
before_instructions
...
Destination = variable
...
after_instructions
...

- All the instructions before the 'Destination' statement will be ignored.
- 'Destination' stores value to the variable.
- The instructions after the 'Destination' denotes click actions. This piece of code will be activated each time when user clicks the object.
