DELETE_RECORD
   

This statement removes an existing record from an indexed file or RDB table.

Format

            DELETE_RECORD table_name

Where


·    table_name - name of the indexed file or the name of RDB table from datasource dialog with UPDATE access.

Notes

            The record to be removed must first be read and locked using a READ, SELECT or READ_ALL instruction.

            If RDB table does not have a primary key, this statement cannot be used.


Example

  
      Select * from tbl1 Where kseg = :$seg1
      If record tbl1 found
            Delete_record tbl1
      End_if

! or

      Read tbl1 key=$seg1
      If record tbl1 found
            Delete_record tbl1
      End_if
