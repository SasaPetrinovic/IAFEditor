Setxmldecpoint
   

Defines decimal point for numeric elements if the application decimal point does not correspond to the xml output.

Format

            Setxmldecpoint (xml_decpoint)

Parameters


·     xml_decpoint - an alpha variable or alpha literal. Specifies the decimal point (comma or point).

Return Value

            Status, numeric variable.

            Possible values:

            0          Ok
            <> 0     Error
            
Applied To

            Xml object.


Example

  
      $stat = Xml.Setxmldecpoint(".")
