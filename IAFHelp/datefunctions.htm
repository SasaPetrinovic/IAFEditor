Date Functions
   


ADD_DAYS adds number of days to a date.
ADD_MONTHS adds number of months to a date.
ADD_WEEKS adds number of weeks to a date.
ADD_YEARS adds number of years to a date.
BEGINM returns a date representing beginning of the month of date_exp.
BEGINQ returns a date representing beginning of the quarter of date_exp.
BEGINW returns a date representing beginning of the week of date_exp.
BEGINY returns a date representing beginning of the year of date_exp.
CONVERT_DMY_YMD converts a numeric or alphanumeric field to the date value.
CONVERT_MDY_YMD converts a numeric or alphanumeric field to the date value.
CONVERT_LIT_YMD converts a date literal to the date value.
CONVERT_YMD_DMY converts a date value to an alphanumeric one.
CONVERT_YMD_MDY converts a date value to an alphanumeric one.
DATE converts a numeric or an alphanumeric field to a date.
DATETIME returns a datetime field.
DAY returns day of a date.
DAY_OF_WEEK returns day of the week.
DAY_OF_YEAR returns day of the year.
DAYS_IN_MONTH returns number of days in a month.
DIFF_DATE returns the difference between two date values.
DIFF_DAYS returns number of days between two date values.
DIFF_DT returns the difference between two date/time expressions.
DIFF_MONTHS returns number of months between two date values.
DIFF_WEEKS returns number of weeks between two date values.
ENDM returns a date representing end of the month of date_exp.
ENDQ returns a date representing end of the quarter of date_exp.
ENDW returns a date representing end of the week of date_exp.
ENDY returns a date representing end of the year of date_exp.
MAKEDATE returns date value calculated from the year, month and day.
MONTH returns month of a date.
NEXT_WORKING_DAY returns next working day.
QUARTER returns a quarter of a year.
SUB_DAYS subtracts a number of days from a date.
SUB_MONTHS subtracts a number of months from a date.
SUB_WEEKS subtracts a number of weeks from a date.
SUB_YEARS subtracts a number of years from a date.
WEEK_OF_YEAR returns week of a year.
YEAR returns 2- or 4-digit year of a date.
YEAR2 returns 2-digit year of a date.
YEAR_MONTH returns 6-digit year and month of a date.
