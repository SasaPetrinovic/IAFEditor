Align/Size
   

Left

Aligns the left edges of the selected objects with the dominant object.

Right

Aligns the right edges of the selected objects with the dominant object.

Top

Aligns the top edges of the selected objects with the dominant object.

Bottom

Aligns the bottom edges of the selected objects with the dominant object.

Horizontal Center

Horizontally center selected objects with the dominant object.

Vertical Center

Vertically center selected objects with the dominant object.

Align To Grid

Aligns selected objects to the grid if grid is visible.

Bring To Front

Moves passive object to the top (sets Z-Order = 0).

Send To Back

Moves passive object to the bottom.

Make same width

Resize selected objects to have the width of the dominant object.

Make same height

Resize selected objects to have the height of the dominant object.

Make same size

Resize selected objects to have the size (width and height) of the dominant object.

Equal Vertical Distance

Evenly spaces the selected objects vertically. The distance between objects is calculated with possible rounding. Both top and bottom objects preserve unchanged positions.

Equal Vertical Distance (fixed)

Evenly spaces the selected objects vertically. Only one object (top or bottom) preserves unchanged position.

Equal Horizontal Distance

Evenly spaces the selected objects horizontally. The distance between objects is calculated with possible rounding. Both left and right objects preserve unchanged positions.

Equal Horizontal Distance (fixed)

Evenly spaces the selected objects horizontally. Only one object (left or right) preserves unchanged position.

Set distance between two objects

Sets fixed distance between two objects.

Center Vertically

Centers selected objects vertically within the nearest frame.

Center Horizontally

Centers selected objects horizontally within the nearest frame.

Center Vertically between two lines

Centers selected objects vertically between two horizontal lines.

Center Horizontally between two lines

Centers selected objects horizontally between two vertical lines.

Auto Size

Resizes selected objects to fit the text.
