UPPER
   

This function converts an alpha field to upper case.

Format

            UPPER (alpha_exp)

Return Value Type

            Alpha.

Parameters


·     alpha_exp - alpha field, ANSI or UTF-16.

Note

            If alpha field is ANSI one, the conversion is performed according to the current code page.


Examples

  
      $res = Upper('abcd')    !! Result = ABCD
 

See Also  LOWER, LOWER_NLS, UPPER_NLS, INITCAP, INITCAP_NLS
