ERRWARMSG
   

This statement is a combination of ERRMSG and WARMSG statements. If Registry symbol ERRWAR exists and has value 1 the statement becomes WARMSG, else ERRMSG is applied.


Format 1

            ERRWARMSG = concatenated_exp

Format 2

            ERRWARMSG = function(...)


Examples

  
      Destination = fld1
      Write_external /name: $errwar '1'
      Check fld1 >= 0
      ERRWARMSG = 'Entered number must not be negative' !!WARMSG

      Destination = fld1
      Write_external /name: $errwar '0'
      Check fld1 >= 0
      ERRWARMSG = 'Entered number must not be negative' !!ERRMSG
