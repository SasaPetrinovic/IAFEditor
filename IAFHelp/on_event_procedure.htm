ON_EVENT_PROCEDURE
   

This statement declares beginning of the event handler.

Format

            ON_EVENT_PROCEDURE event_name([identifier])

Where


·           event_name      - the name of the event: OnClickMenuItem and OnRClicked for context menu, or OnKeyPress for function keys.
·           identifier            - the name of the identifier: function key or menu item identifier.


Note

            An event handler must be ended by End_procedure statement.

            OnRClicked handler will be executed after pressing the right mouse button and before menu displaying. This handler is usually used to adjust menu items (setenable(0), etc).

Examples


  
!Event handler for F5 function key:

ON_EVENT_PROCEDURE OnKeyPress(F5)
      form.display("F5 key")
END_PROCEDURE

!Event handler for Ctrl+F5 function key:

ON_EVENT_PROCEDURE OnKeyPress(Ctrl+F5)
      form.display("Ctrl F5 key")
END_PROCEDURE

!Event handler for context menu right click:

ON_EVENT_PROCEDURE OnRClicked()
      form.display("Rclick")
END_PROCEDURE

!Event handler for menu item CMITEM_1:

ON_EVENT_PROCEDURE OnClickMenuItem(cmitem_1)
      form.display("cmitem_1 selected")
END_PROCEDURE
