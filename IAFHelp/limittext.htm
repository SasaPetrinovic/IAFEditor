Limittext
   

This method limits amount of the text, in characters, that can be entered into an alphanumeric destination field.

Format 1
  
 Limittext (numchars)
 Limits text of a field

Format 2
  
 Limittext  (line,column,numchars)
 Limits text of a grid cell


Parameters

·    numchars - numeric variable or numeric literal greater than zero.

·     line - grid line, numeric value. The lines are counted from 1.To specify current line, use the internal variable CURRENT_LINE or star (*).

·     column - grid column, numeric value. The columns are counted from 1.


Applied To

            Destination field, field type is alphanumeric.

Note

            - Number of input characters is defined by edit mask, see Edit mask.
            - If IgnoreEditMask attribute is set, edit mask is ignored and number of input characters is equal to field length.

            To change number of input characters dynamically, this method should be used.

            However, this method can't increase number of input characters defined by painter. For example, if edit mask is XXXXXXXX (allows eight characters), then any value less than 8 can be used as parameter.


            To limit the text in entire grid column, use ENTIRE_COLUMN constant in the form

                        Table_1.Limittext (ENTIRE_COLUMN,column,numchars)

Example

  
      DestField_1.LimitText(40)     !! maximum 40 characters can be entered
