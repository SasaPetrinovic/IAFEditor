RIGHT
   

This function extracts a substring beginning with the rightmost character in alpha_exp.

Format

            RIGHT(alpha_exp, count)

Return Value Type

            Alpha.

Parameters


·     alpha_exp - input field, alpha
·     count - number of characters to extract, numeric.


Note

            To extract the characters from the actual string, use RIGHT(RTRIM(alpha_exp), count).

Examples

  
      $outstr = right('ABCD ',2)          !! result is 'D '
      $outstr = right(rtrim('ABCD '),2)   !! result is 'CD'
