Setvisible
   

This method displays or hides an object.

Format

            Setvisible(value)

Parameters


·    value - a numeric variable or numeric literal.
                        value = 0, hides the object,
                        value <> 0, shows the object.

Applied To

            All objects.

Example

  
      Label_1.SetVisible(0) !! Set object Label_1 invisible
