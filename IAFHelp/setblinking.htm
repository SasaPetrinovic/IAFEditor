Setblinking
   

This method starts / stops blinking.

Format

            Setblinking (interval, bkColor,  txColor)

Parameters


·     interval - blink interval in milliseconds, integer. Allowed values are between 0 and 2000. Value 0 stops blinking. If value is out of these limits, default is 500.
·     bkColor - background color, a numeric variable with valid RGB color value or one of the predefined colors or RGB function. Optional.
·     txColor - text color, a numeric variable with valid RGB color value or one of the predefined colors or RGB function. Optional.


Applied To

            Text label.

Note


Examples

  
      obj.Setblinking(500,'black','white')
      obj.Setblinking(500,rgb(255,0,0),rgb(255,255,255))
      obj.Setblinking(500,rgb(255,0,0)) !! blink background only
      obj.Setblinking(500,,rgb(255,255,255)) !! blink text only
      obj.Setblinking(0) !! stops blinking
