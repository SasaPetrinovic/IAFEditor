Application description

Short description of the application.


Decimal point

Decimal point determines the way of defining edit mask for numeric fields.

If you select comma, decimal separator is COMMA and group separator is POINT, for example 1.000,00.

If you select point, decimal separator is POINT and group separator is COMMA, for example 1,000.00.

This setting also affects numeric literal notation (see Value expression):

Default date edit mask

When we insert sourcefield object or destinationfield object in form painter, and field type is date, this edit mask will appear on the screen.

Timeout period

This parameter applies to the all programs in the application. If there is no user's activities (keyboard, mouse movement) in that period, current transaction will be rolled back, and program will be stopped. Timeout period unit is minute. For instance, value 10 means ten minutes. Value -1 (minus 1) means no timeout.

File system

If the application use flat files only, select Classic.

Otherwise, if you use any of relational databases (MS Access, MS SQL Server, Oracle, DB2, Informix, MySQL, ...), select ODBC. This means generated programs will access to the database via ODBC interface. At the end, choose database.


Default data directory

Default data directory is default prefix of physical file name, used in file definition dialog (Data Dictionary utility). We can enter absolute path or logical name.

Examples:

            c:\data
            c:\mydata
            \\myserver\c\data
            data_dir

In above examples, c:\data, \\myserver\c\data are absolute paths, data_dir is a logical name.
At run-time, Logical names table establishes connection between logical name and physical path. See also Logical names table maintenance.

Default print directory

Default print directory is a folder where the print files will be created. We can enter absolute path or logical name. When we create report in report painter, this string is used as a default prefix in print file name.

Audit trail file

This file records activities all the programs generated by this generator: when program starts, one record will be written in this file. Also, program termination will be recorded along with some fatal errors: size overflow, invalid index, etc.

Default language

If you do not plan to create a multilingual application, leave this field empty.

In multilingual applications, this is your default language, ie. the language which you use to create text labels, messages, etc. in your programs.
This field (language name) appears in the title of various dialogs, like Strings definition, etc.
