File Manipulation
   

The list shows already defined files with their characteristics.

File name is maximum 30 characters long, the first character must be a letter. Allowed characters: letters A - Z, digits 0 - 9, underline (_) and a dollar sign ($). File name in one Data Dictionary must be unique.

Modify

To modify an existing file definition, select the file in the list and click Modify

-or-

Double click the file name in the list.

The File definition dialog appears on the screen.


New

To create new file definition, click button New, enter file name and go through the file definition dialog.

Copy


To create new file definition based on existing file definition, select the file in the list and click Copy. This option is useful if we have similar file definitions.

Delete


To delete a file  definition from the Data Dictionary, select the file in the list and click Delete.

Delete unused files

To delete all unused file definitions (the files not used by any program), click Delete unused files. The list of unused files appears on the screen. We can delete all these files or just some of them.

Unlock

If during file definition (modification) phase Data Dictionary Maintenance program crashes, the file remains locked. To unlock the file, select it and click Unlock.
