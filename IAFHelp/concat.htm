CONCAT
   

This function concatenates many alpha fields into one string.


Format

            CONCAT (val_exp1,  val_exp2, …, val_expN)

Return Value Type

            Alpha or memo.

Parameters


·    val_exp1,..., val_expN - input variables. Maximal number of input variables is 512.


Notes

·           To append text to a string use the format

            $s = CONCAT ($s,'any text') or
            $s = CONCAT (alltrim($s),'any text')



·           Default editing (as in COMPRESS functions) is not applied to input variables. Therefore, integer, packed or double floating numeric cannot be used as input variable. Unpacked numeric, date or time field will be stored in its internal form.

Examples

  
$S = CONCAT("AB C", "AB", "X")
$S = CONCAT(alltrim($s), 'some text')
$S = CONCAT($s, alltrim($s1), alltrim($s2))
$S = CONCAT($s, edit(acc_1,'ZZZ.ZZ9,99'), alltrim($s2))
