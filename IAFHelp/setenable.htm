Setenable
   

This method enables or disables mouse and keyboard input.

Format

            Setenable(value)

Parameters


·    value - a numeric variable or numeric literal.
                        value = 0, object will be disabled,
                        value <> 0, object will be enabled.

Applied To

            Combobox, Checkbox, Radiobutton, Pushbutton, Bitmapbutton, Grid, Form, Destinationfield, Sourcefield.

Notes

            FORM.Setenable(0) disables Close button.

            SOURCEFIELD.Setenable(0) just changes text color to dark gray.

            Using SETENABLE we can disable a button (standard or hyperlink) in the grid. Because this method does not use line and column arguments, we can disable a button in the current line only. The best place for this action is 'Line ending actions':

            if condition
                        button_grid.setenable(0)
            end_if

            If we use this method to disable a hyperlink in the grid, then text gets dark gray color and hand cursor does not appear when the cursor hovers over the cell.

Example

  
      Button_1.Setenable(0)   !! Disables input
      Button_1.Setenable(1)   !! Enables input
