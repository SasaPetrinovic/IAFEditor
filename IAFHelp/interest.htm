INTEREST
   

This function calculates the interest payment for a given period.

Format

            INTEREST(principal,rate,date_from,date_to,flag1,type)

Return Value Type

            Numeric.

Parameters

·     principal - amount
·     rate - interest rate
·     date_from - beginning of the period
·     date_to - end of the period
·     flag1 - 0 or 1, see formula below
·     type - 0 for monthly interest, 1 for annual interest
Formula

            Interest = principal * (((1+(rate/100)) ^ (diff_days(date_to,date_from)/period))-1)

            Period calculates as:

            if flag1 = 1
                        if type = 1
                                    period = day_of_year(endyr(datum_from))
                        else
                                    period = days_in_month(datum_from)
                        end_if
            else
                        if type = 1
                                    period = day_of_year(endyr(datum_to))
                        else
                                    period = days_in_month(datum_to)
                        end_if
            end_if
