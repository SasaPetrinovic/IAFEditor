Settextcolor
   

This method sets the text (foreground) color of an object or grid cell.

Format 1
  
 Settextcolor (color)
 Sets text color of an object
Format 2
  
 Settextcolor (line,column,color)
 Sets text color of a grid cell

Parameters


·    color - a numeric variable with valid RGB color value, one of the predefined colors or RGB function.
·    line - grid line, numeric value. The lines are counted from 1.To specify current line, we can use the internal variable CURRENT_LINE or star (*).
·    column - grid column, numeric value. The columns are counted from 1. If this parameter has the value -1, the entire line will be colored.

Examples

  
      label_1.settextcolor("black")
      label_1.settextcolor("red")
      label_1.settextcolor(RGB(255,0,0)) !! same as above
      label_1.settextcolor(255)
      label_1.settextcolor(256*128 + 255)

      table_1.settextcolor(counter_1,2,"white")
      table_1.settextcolor(current_line,2,"white")
      table_1.settextcolor(*,2,"white") !! same as above
      table_1.settextcolor(*,-1,"white")
