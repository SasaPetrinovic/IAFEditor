Downloadfile
   

This method downloads file from the connected FTP server and saves it on the local machine.

Format

            Downloadfile (remotefile, localfile [, override_flag [,progressbar_object]])

Parameters


·     remotefile - an alpha variable or alpha literal. Specifies name of the file on the server, with or without path specification. Wildcards not allowed.

·     localfile - an alpha variable or alpha literal. Local file name.

·     override_flag - an alpha variable or alpha literal, optional. Override flag: value "O" denotes that local file will be overridden if exists. If the parameter has the value different from "O" and local file exists, error will be reported. Default value = "O".

·     progressbar_object - an alpha variable or alpha literal, optional. Name of a progress_bar used to display progress of file transfer.

Return Value

            Alpha variable. If connecting was successful, value is blank. Otherwise, the variable holds an error message.


Applied To

            Ftp object.

Note

            Transfer method is BINARY.

Examples

  
      $msg = Ftp.Downloadfile('file1.exe','c:\temp\a.exe','O','progress_1')
      $msg = Ftp.Downloadfile('file1.exe','c:\temp\a.exe') !!override, no progress
