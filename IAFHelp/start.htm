START
   

This statement positions on a record which is determined by a key.

Format


            START file [KEY[n] = key]  [WHERE cond_exp]


Where


·     file - the name of the file from datasource dialog.
·     key - concatenation
·     cond_exp - conditional expression. If FALSE, record wiil not be displayed in the grid.


Note

            The statement is used with index-sequential or sequential files to fill a display-only grid. If an application uses RDB tables, see SELECT_START.


Example

  
      START file1
      START file2 KEY='ABC'
      START file2 KEY='ABC' Where fld1 <> 0
