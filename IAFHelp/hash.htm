HASH
   

This function calculates hash value of a data.

Format

            HASH(algid, field [, field_length])

Return Value Type

            Alpha.

            For MD5, result length is 32 characters.
            For SHA1, result length is 40 characters.
            For SHA_256, result length is 64 characters.


Parameters


·    algid - algorithm identifier, numeric literal or numeric field. Possible values:
                              1 for MD5,
                              2 for SHA1,
                              3 for SHA_256.

·     field - input field, alpha or memo.
·     field_length - length of input field, optional. If omitted, whole input field will be processed.


Note

            Hash value is fixed-length binary data. For example, MD5 is an 128-bit binary field (16 bytes). Generally, hash value is displayed in readable form as a hexadecimal number. Therefore, MD5 hash value is 32 characters (hexadecimal digits) long.

Examples

  
      $a64 = HASH(3,$inp,str_length($inp)) !!calculate sha_256
      $a64 = HASH(3,$inp_str)              !!calculate sha_256

! Calculate hash value sha_256 for a file
! $s - memo field
! $a64 - alpha ANSI field, field length is 64 characters
      $status = LOAD_FILE("c:\test\a.txt", $s)
      if $status = 0
            $a64 = HASH(3, $s)
            init $s     !!Release allocated memory
      end_if
