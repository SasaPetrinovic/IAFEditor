SAVE_FILE
   

This function stores a long field, string,  (LONGVARCHAR or LONGVARBINARY, CLOB, BLOB) into a file. Result is an integer containing status.


Format

            SAVE_FILE(field,file_name)

Return Value Type

            Numeric.

            Possible values:
                        0          successful
                        -1         The file name is too long
                        3          No such directory
                        > 0       other file errors

Parameters


·    field - long field (ANSI or UTF-16).
·    file_name - output file name, logical names allowed.


Note


Example

  
! $status is defined as numeric, without decimals.
! This function stores PCT field into c:\test\a.bmp
      if tbl1.pct is not NULL
            $status = SAVE_FILE(tbl1.pct,"c:\test\a.bmp")
            if $status <> 0
                  pause "Save_file error=" | $status
            end_if
      end_if
