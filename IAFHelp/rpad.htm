RPAD
   

This function returns the alpha_exp, right-padded with another alpha_exp.

Format

            RPAD(alpha_exp, padstr)

Return Value Type

            Alpha.

Parameters


·     alpha_exp - field to be right-padded, alpha.
·     padstr - pad character or pad string, alpha.


Note

            All white space characters at the end of alpha_exp will be replaced by padstr.

Examples

  
      $Outstr = RPAD($INPSTR,'%')
      $Outstr = RPAD($INPSTR,'abc')
      $Outstr = RPAD($Outstr,'%')
      $Outstr = RPAD($Outstr,$PAD)

! Fill a field by a character
      init $Outstr
      $Outstr = rpad($Outstr,'0')
