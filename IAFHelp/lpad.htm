LPAD
   

This function returns an alpha field, left-padded with another string.

Format

            LPAD(alpha_exp, padstr)

Return Value Type

            Alpha.

Parameters


·     alpha_exp - field to be left-padded, alpha literal or alpha field.
·     padstr - pad character or pad string, alpha literal or alpha field.


Note

            Input field is aligned to the right and all left spaces are replaced by padstr.

Examples

  
      $Outstr = LPAD($INPSTR,'0')
      $Outstr = LPAD($INPSTR,'abc')
      $Outstr = LPAD($Outstr,' ')
      $Outstr = LPAD($Outstr,$PAD)

! Fill a field by a character
      init $Outstr
      $Outstr = lpad($Outstr,'0')

! Suppose that $a10 is defined as alpha, length = 10.
      $a10 = 'A'
      $a10 = LPAD($a10,'0') !! result is: '000000000A'
