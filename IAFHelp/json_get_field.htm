JSON_GET_FIELD
   

This function reads one field from a JSON structure.


Format

            JSON_GET_FIELD(json_variable, field_name)

Result Value Type

            Any field.

Parameters


·     json_variable - alpha variable containing JSON structure. Variable type is CLOB, longchar (type = S).

·     field_name - field name, alpha literal or alpha variable.

Note


Examples

  

$fld = json_get_field($json,'field_1')
