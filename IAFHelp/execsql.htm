EXECSQL
   

This statement executes a SQL statement.

Format 1

            EXECSQL SQL_statement

Format 2

            EXECSQL $local_variable


Where


·     SQL_statement - a SQL statement, text.
·     $local_variable - an alpha local variable containing SQL statement.
            

Notes

            A SELECT (SELECT_ALL) statement cannot be executed by EXECSQL.

            Data manipulation statements (CREATE TABLE, CREATE INDEX, ...) do not support parameters, i.e. we cannot write CREATE TABLE :$tblname (col1 int, col2 int).

Examples

  
      EXECSQL update tbl1 set fld1=100, fld2=’ABC’ where keyfld=1
      EXECSQL delete from tbl1 where keyfld=1
      EXECSQL delete from tbl1 where keyfld= :$key
      $aloc = "Create table tbl1 (col1 int, col2 char(10))"
      EXECSQL $aloc
