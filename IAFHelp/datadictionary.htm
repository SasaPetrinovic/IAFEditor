Data Dictionary
   


Show

This dialog shows list of the applications defined in the Data Dictionary.

Compress

Removes deleted entities (fields, records, procedures, programs, ...) from the DD. The previous versions of these entities also will be removed. Previous Data Dictionary files *.DDn will be renamed to *.BAn (name.DD1 -> name.BA1, name.DD2 -> name.BA2, ...), and new files *.DDn will be created.

Copy DD to another DD

Create new Data Dictionary and copy one application to the new dictionary.

DD Manager

Shows, copies and compares DD entites.

DD Analyzer

Analyzes contents of Data Dictionary files: DD2, DD3 and DD4.

DD Converter

Converts DD structure version_1 into version_2. The version_2 structure allows size of the DD4 file up to 1GB.
