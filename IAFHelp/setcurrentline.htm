Setcurrentline
   

This method sets current line to line.

Format

            Setcurrentline(line)

Parameters


·     line - numeric variable or numeric literal. Constant LAST_LINE denotes last grid line.

Applied To

            Display-only grid.

Note

            This method is used usually in 'Table ending actions', when all the grid lines are displayed.


Example

  
            Table_1.Setcurrentline(10)
            Table_1.Setcurrentline(LAST_LINE)
