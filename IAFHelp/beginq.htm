BEGINQ
   


This function returns a date representing beginning of the quarter of date_exp.

Format

            BEGINQ(date_exp)

Return Value Type

            Date.

Parameters


·     date_exp - input date or datetime.

Examples

  
      $Date = BEGINQ("12-mar-2001") !! result is 1-jan-2001
 


See Also   ENDM, ENDQ, ENDW, ENDY, BEGINM, BEGINW, BEGINY, Date Functions
