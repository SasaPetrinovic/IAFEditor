Getbackcolor
   

This method returns background color of a cell in current sheet.

Format

            Getbackcolor(line, column)

Parameters


·     line - row, numeric value, integer. The first line is 1, etc.

·     column - column, numeric value, integer. The first column is 1. Instead of column number, an alpha literal containing column header can be used: 'A', 'B', ..., 'AA', 'AB', ...


Return Value

            Background color of a cell, RGB value, integer.

Applied To

            Excel object.

Note

            To get background color of current tab, set line and column to -1.

Examples

  
      $color = Excel.Getbackcolor(1,1)    !!line=1, col=1
      $color = Excel.Getbackcolor(1,2)    !!line=1, col=2
      $color = Excel.Getbackcolor(1,'B')  !!same as above

      $color = Excel.Getbackcolor(-1,-1)  !! tab color
 

Using Excel,  Grid Getcell
