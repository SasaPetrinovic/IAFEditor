XML
   

This function opens and reads an XML file (obsolete).

Format

            XML(operation[,arg1,arg2,…])

Result Value Type

            Numeric.

Parameters

·     operation - file operation, an alpha literal. Possible values are: "OPEN", "CLOSE", "NODE", "RECORD", "FIELD", "ATTRIBUTE", "SKIP".
·     arg1, arg2,... argN - arguments

Usage

            a/         Open a XML file.

                        XML("open", file_name, root_element)
                        file_name          XML file name
                        root_element     name of the root element

                        This call should be invoked before any other XML call.

                        Example:

                        $status = XML("open","c:\dir\test.xml","root")
                        if $status <> 0
                                    Pause "Error opening file"
                        end_if

            b/         Close an opened XML file

                        XML("close")

            c/         Position at the next node

                        XML("node",node_name)

            d/         Read a field into a host variable

                        XML("field",field_name,host_variable)

            e/         Read an attribute into a host variable

                        XML("attribute",attribute_name,host_variable)

            f/          Read a record

                        XML("record",record_name)

                        Record record_name must be defined in the Iaf Data Dictionary and declared in Datasource dialog.

            g/         Skip fields up to the end of the current level

                        XML("skip")

Note

            If the function succeeds, the result is zero.
            If the function fails, result contains error status.

Examples
  

Copy Test.xml into standard seq file.

Test.xml:

<?xml version = "1.0" encoding = "utf-8"?>
<Books>
    <Book>
        <Title>Title 1</Title>
        <Author>Author 1</Author>
        <Publisher>New Company</Publisher>
        <Pub_year>1989</Pub_year>
    </Book>
    <Book>
        <Title>Title 2</Title>
        <Author>Author 2</Author>
        <Publisher>New Company</Publisher>
        <Pub_year>1990</Pub_year>
    </Book>
</Books>

The simplest way: define record BOOK (and corresponding file) in the DD. Fields are
            TITLE   Alpha 30
            AUTHOR Alpha 30
            PUBLISHER Alpha 60
            PUB_YEAR Num 4
Fields order in DD record must be the same as in XML file.
If we do not need all the fields from XML, we can define DD record containing desired fields only. For example:
            TITLE   Alpha 30
            PUBLISHER Alpha 60
Fields order must be preserved.


Procedure 1 ($status is a local numeric field, XML_DIR is logical name defined in Logntt00.iaf, file BOOK declared in Datasource list):

! First command must be "open"
$status = xml("open","xml_dir:test.xml","books")
if $status = 0
      do_while 1 = 1          !! endless loop
            $status = xml("record","book")
            if $status <> 0
                  break       !! no more records, exit loop
            end_if
            write book        !! write seq record
      end_do
      $status = xml("close")  !! each succ. open must have corr close
else
      Pause "Open error: " | $status
end_if

Procedure 2: read XML file without DD record

$status = xml("open","xml_dir:test.xml","books")
if $status = 0
      $status = xml("node","book")  !! position on node <book>
      do_while $status = 0
! read field TITLE, host variable is $TITLE
            $st1 = xml("field","title", $title)
            break /cond: $st1 <> 0  !! break if error
! read PUB_YEAR, skipping AUTHOR and PUBLISHER
            $st1 = xml("field","pub_year", $year)
            break /cond: $st1 <> 0  !! break if error
            $status = xml("node","book")  !! position on next node
      end_do
      $status = xml("close")  !! each succ. open must have corr close
else
      Pause "Open error: " | $status
end_if



Let us consider more complicated XML file

<?xml version = "1.0" encoding = "utf-8"?>
<Books>
      <hdrfld1>fld 1</hdrfld1>
      <hdrfld2>fld 2</hdrfld2>
      <hdrfld3>fld 3</hdrfld3>
      <Book>
        <Title>Title 1</Title>
        <Author>Author 1</Author>
        <Publisher>New Company</Publisher>
        <Pub_year>1989</Pub_year>
      </Book>
      <Book>
        <Title>Title 2</Title>
        <Author>Author 2</Author>
        <Publisher>New Company</Publisher>
        <Pub_year>1990</Pub_year>
      </Book>
</Books>

Procedure 3: if we do not need hdrfld1, hdrfld2 and hdrfld3, we can use Procedure 1.

Procedure 4: if we need those fields, then the procedure is

$status = xml("open","xml_dir:test.xml","books")
if $status = 0
      $st1 = xml("field","hdrfld1", $h1)
      $st1 = xml("field","hdrfld2", $h2)
      $st1 = xml("field","hdrfld3", $h3)
      do_while 1 = 1
            $status = xml("record","book")
            if $status <> 0
                  break       !! no more records, exit loop
            end_if
            write book        !! write seq record
      end_do
      $status = xml("close")  !! each succ. open must have corr close
else
      Pause "Open error: " | $status
end_if

File Test.xml can be written on the following way (using attributes):

<?xml version = "1.0" encoding = "utf-8"?>
<Books>
 <Book Title="Title1" Author="Auth1" Publisher="Cmp" Pub_year="1989" />
 <Book Title="Title2" Author="Auth2" Publisher="Cmp" Pub_year="1990" />
</Books>

In that case we can use Procedure 5:

$status = xml("open","xml_dir:test.xml","books")
if $status = 0
      do_while 1 = 1
            $status = xml("field","book", $A100)
            break /cond: $status <> 0 !! no more fields, break
            $status = xml("attribute","Title", $title)
            $status = xml("attribute","Author", $auth)
            $status = xml("attribute","Publisher", $pub)
            $status = xml("attribute","Pub_year", $year)
      end_do
      $status = xml("close")  !! each succ. open must have corr close
else
      Pause "Open error: " | $status
end_if
