DATE
   

This function converts a value expression to a date.

Format

            DATE(val_exp)

Return Value Type

            Date.    

Parameters


·    val_exp - date literal, alpha literal, numeric literal or alpha field.

Note

            Proper date format (yyyymmdd) must be stored in the value expression.

Examples

  
      $date_field = date("20051201") !! alpha literal

!In next two lines we can use simple assignment because the !left and right operands are compatible.
      $date_field = '1-dec-2015'
      $date_field = '2015-12-01'

!In the next lines we must use DATE function.
      $date_field = date(20051101)   !! numeric literal
      $date_field = date('20151301') !! invalid month

      if $a8 is date
            $date_field = date($a8)
      end_if
