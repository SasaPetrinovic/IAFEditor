Edit
   

Undo

Undo last operation.

Cut

Copies selected objects to the Clipboard and remove them from the screen.

Copy

Copies selected objects to the Clipboard.

Paste

Creates new objects if they are previously copied to the Clipboard by Copy or Cut.

Place the cursor at the position where we want to create new object and click. This position represents left upper corner of the new object. Choose Edit/Paste menu command or press Ctrl+V.

See Insert at last point clicked option in View section.

Find

Finds a string in code, labels, procedures, etc.

Replace

Replace one string by another string.

Adjust properties to current profile

Changes video attributes of all selected objects according to the current profile.

Make uppercase

Converts all characters to upper case according to object's font.

Make lowercase

Converts all characters to lower case according to object's font.

Make sentence case

Converts the first character to the upper case, all other characters to lower case according to object's font.

Insert character

Inserts a special character from ANSI character set into text label.

Select all

Selects all objects.

Reset screen layout

Removes all the objects from the screen.
