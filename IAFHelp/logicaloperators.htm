Logical operators
   

Logical operators let us join two or more conditional expressions.

Logical operators are

            AND     combines two conditions and returns TRUE when both conditions are TRUE.
            OR       combines two conditions and returns TRUE when either of conditions are TRUE.
            NOT     reverses value of a Boolean expression, unary operator.


            Examples

      if a = b AND c = d
      if a = b OR c = d
      if a = b AND (c = d OR e = f)
      if NOT a = b
      if a = b AND NOT c = d


Logical operators precedence (order of evaluation)

List of operators with decreasing precedence is

            NOT
            AND
            OR

The precedence of NOT is greater than that of AND, etc.
To avoid confusion, use parentheses !


Example 1

      if a = b AND c = d OR e = f

is the same as

      if (a = b AND c = d) OR e = f


Example 2

      if a = b OR c = d AND e = f

is the same as

      if a = b OR (c = d AND e = f)

Example 3

      if a = b OR c = d AND e = f OR g = h

is the same as

      if a = b OR (c = d AND e = f) OR g = h
