RGB
   

This function returns an integer representing RGB color value.

Format

            RGB(red,green,blue)

Parameters


·     red - red component of the color, an integer in the range 0 to 255
·     green - green component of the color, an integer in the range 0 to 255
·     blue - blue component of the color, an integer in the range 0 to 255



Return Value Type

            Integer.


Note

            This function can be used in SETBACKCOLOR and SETTEXTCOLOR methods.

Examples

  
      obj_1.settextcolor( rgb(0,0,255) )
      table_1.setbackcolor( *, 1, rgb(0,0,0) )

      obj1.settextcolor("RED") !!is the same as
      obj1.settextcolor(RGB(255,0,0))
