ADD_MONTHS
   

This function adds number of months to a date.

Format

            ADD_MONTHS(date, nummonths)

Return Value Type

            Date.

Parameters


·     date - date expression
·     nummonths - number of months, numeric_exp

Note

            Out-of-range result date will be corrected automatically.

Examples

  
      $Months = ADD_MONTHS (today,5)
      $Months = ADD_MONTHS (BEGINCY,-3)
      $Months = ADD_MONTHS (today,counter_1)

      $Months = ADD_MONTHS ("31-mar-2001",1) !! result is 30-apr-2001
      $Months = ADD_MONTHS ("31-jan-2001",1) !! result is 28-feb-2001
