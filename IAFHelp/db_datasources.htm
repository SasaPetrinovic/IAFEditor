DB_DATASOURCES
   


This function stores list of ODBC datasource names into text file.

Format

            DB_DATASOURCES(output_file)

Result Value Type

            Numeric. If the function fails result is -1, otherwise, result is 0.

Parameters


·     output_file - output file name, alpha literal or alpha variable. The name can be: absolute path, like c:\temp\tbl.txt, or path with logical name, like work_dir:tbl.txt. The record consists of the two fields: datasource name and datasource description. Field delimiter is TAB (ASCII code 9). In DD, File definition dialog, select 'Variable with field delimiters' option for record format.


Note

            To use this function, your program doesn't need to be connected to a datasource.


Examples

  
      $status = db_datasources("c:\temp\ds.txt")
      $status = db_datasources(workdir:ds.txt")
