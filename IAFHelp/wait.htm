WAIT
   

This statement suspends the program execution for a specified interval (seconds or milliseconds).

Format

            WAIT time_interval

Where


·     time_interval - time interval. If time_interval > 0, this parameter denotes seconds. If time_interval < 0, the parameter denotes milliseconds.

Example

  
      Wait 3      !! wait 3 seconds
      Wait -100   !! wait 100 milliseconds
