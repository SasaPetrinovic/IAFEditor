DB_EXPORT
   

This function copies data from an RDB table to output file.

Format

            DB_EXPORT(table_name, output_file,additional_SQL_criteria,progressbar)

Result Value Type

            Numeric, number of records written to the output file. If schema/table does not exist, result is -1. Result value of -2 indicates error on output file. Result value of -3 indicates other errors.

Parameters


·     table_name -name of table to be copied, alpha literal or alpha variable. If you want to copy a table from your default schema, supply table name only. If you want to use table from another schema, enter the table name in the form SCHEMA.TABLE_NAME.

·     output_file - output file name, alpha literal or alpha variable. The name can be: absolute path, like c:\temp\t1.dbe, or path with logical name, like export_dir:t1.dbe.

·     additional_SQL_criteria - additional WHERE, ORDER BY or JOIN clause, optional parameter. Normally, run-time engine generates SELECT * FROM table_name and copies all the records (rows) to the output file. Using this parameter, you can copy just a few records. SQL parameters, like Where fld1 = :$locvar, are not allowed.

·     progressbar - identifier, name of the progress bar, optional parameter. If specified, this object displays progress of the table export operation.


Note

            To use this function, your program has to be connected to a datasource. To enable datasource connection, specify at least one RDB table in Datasource dialog. Only tables from the current connection can be exported. If the program uses multiple connections, the first connection will be used for export. Username, used in connection string, must have privileges to read catalog information from the database.


Examples

  
      $status = db_export("mytbl","c:\temp\mytbl.dbe")
      $status = db_export("mytbl","expdir:mytbl.dbe")
      $status = db_export("myschema.mytbl","expdir:mytbl.dbe")

      $asql = "Where fld1 = 'AK'"
      $status = db_export("mytbl","expdir:mytbl.dbe",$asql)

      $asql = "Where fld1 = 'AK' Order by 1,2,3"
      $status = db_export("mytbl","expdir:mytbl.dbe",$asql)

      $asql = ", tbl2 Where fld1 = 'AK' and f1=tbl2.f1"
      $status = db_export("mytbl","expdir:mytbl.dbe",$asql)
      $status = db_export("mytbl","expdir:mytbl.dbe",$asql,progress_1)

      $status = db_export("mytbl","expdir:mytbl.dbe",,progress_1)
