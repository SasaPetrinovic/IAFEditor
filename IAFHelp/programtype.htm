Program type
   

In this dialog we define program type: report or batch.

A report has program title (optional) and program cycle which generates report.

A batch program has program title (optional) and program cycle which performs some database updates.

If we select Report, report type need to be defined.

Standard
The report of this type has standard elements: report header, page header, subheaders, detail line and subtotal lines. The main characteristic is: one record from the input record stream generates one detail line.
Label
The main characteristic of labels: one record from the input record stream generates one label.
Form
The form looks like a pre-printed form. Maximum number of pages in the form is 20. The report painter has good set of commands to draw rectangles, lines, frames; to connect lines to other objects, etc.
The main characteristic of forms: one record from the input record stream generates one form.
