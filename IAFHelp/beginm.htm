BEGINM
   

This function returns a date representing beginning of the month of date_exp.

Format

            BEGINM(date_exp)

Return Value Type

            Date.

Parameters


·     date_exp - input date or datetime.

Examples

  
      $Date = BEGINM("12-mar-2001") !! result is 1-mar-2001
 


See Also   ENDM, BEGINW, BEGINY, Date Functions
