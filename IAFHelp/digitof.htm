DIGIT_OF
   

This function returns one digit from a numeric value.

Format

            DIGIT_OF(numeric_exp, digit_index)

Result Value Type

            Numeric.

Parameters


·     numeric_exp - numeric field
·     digit_index - digit index


Note

            Digit_index is defined as follows:
  
Numeric value =
1
2
3
4,
5
6
Digit_index
3
2
1
0
-1
-2

            The first digit after decimal point has digit_index -1.
            The first digit left from decimal point has digit_index 0.


Examples

  
      $D = DIGIT_OF(1024,0) !! result is 4
      $D = DIGIT_OF(1024,1) !! result is 2
