Open
   

This method starts MS Excel and opens (creates) document. Must be invoked before any other Excel method.

Format

            Open([filename [,hide][,password]])

Parameters

·     filename - an alpha variable or alpha literal. Specifies name of the existing Excel document. If we create new, empty document, filename is empty string. Logical names in filename are allowed.

·     hide - second parameter, optional, is alpha literal containing text "hide". In that case, Excel is active in background and cannot be seen on the screen. If a lot of data need to be transfered between program and Excel, this parameter is required because it accelerates the transfer.

·     password - alpha literal or alpha field, optional. If Excel document is password protected, the password should be supplied to automatically open the document. If the password is not supplied and the document is password protected, then the dialog appears at run-time.


Return Value

            Status, numeric variable.

            Possible values:

            0          Ok
            -1         Cannot activate Excel
            -2         Document already opened
            -3         File not found
            -4         Logical name not found
            -5         Invalid password

Applied To

            Excel object.

Note

            To release current Excel document use Close or Run method.

Examples

  
            Variants

            Open()
            Opens new, empty document, Excel is visible
            Open("","hide")
            Opens new, empty document, Excel is hidden
            Open(filename)
            Opens existing document, Excel is visible
            Open(filename,"hide")
            Opens existing document, Excel is hidden
            Open(filename,,'mypwd')
            Opens existing document, password = 'mypwd'

      $stat = Excel.Open()
      $stat = Excel.Open('','hide')
      $stat = Excel.Open('c:\exc\test.xls')
      $stat = Excel.Open('c:\exc\exdoc.xls',,'pwd1')
      $stat = Excel.Open('docdir:test.xls')
      $stat = Excel.Open('c:\exc\test.xls','hide')
      if $stat = 0
            ...
      end_if
