COUNT
   

This aggregate function returns number of records between two break fields.

Format

            COUNT [(cond_expr)]

Return Value Type

            Numeric.

Parameters


·     cond_expr - optional conditional expression. If cond_expr does not exist, all records from the input record stream will be counted. If cond_expr is specified, only records satisfying cond_expr will be counted.

Notes


·     Cond_expr is calculated in detail lineset.


·     This function can be used in reports and batch programs only.

Examples

  
      Source = COUNT
      Source = COUNT(fld1 > 0)
