Getnextselected
   

This method returns index of the next selected line in a grid (table).

Format

            Getnextselected (line)

Parameters


·     line - a numeric variable or numeric literal, integer. Index of the line to begin the searching with, or -1 to find the first line. The specified line is excluded from search. The first line has index 1.


Return Value

            Index of the next selected line (1, 2, ...), or -1 if there is no next selected line.

            
Applied To

            Grid.

Examples

  

      if table_1.getselectedcount() > 0
            $sellin = table_1.Getnextselected(-1)     !! Index of the first selected line
            do_while $sellin <> -1
                  ...
                  $sellin = table_1.Getnextselected($sellin)      !! Next selected
            end_do
      end_if
