DIFF_DATE
   

This function returns the difference between two date values.

Format

            DIFF_DATE(date1, date2, flag)

Result Value Type

            Numeric, 6 digits.
            
Parameters


·           date1, date2 - date or datetime fields.
·           flag - alpha, difference flag. Value ‘0’ denotes that the difference between two equal dates is 000000.Value ‘1’ denotes that the difference between two equal dates is 000001.

Note

            Result is a 6-digit positive number. The first two digits denotes number of years,
            the second two digits denotes number of months and the two last digits denotes
            number of days.

Examples

  
      $n6 = DIFF_DATE("10-feb-2009","1-jan-2009","0") !! result is 000109
      $n6 = DIFF_DATE("10-feb-2009","1-jan-2009","1") !! result is 000110
      $n6 = DIFF_DATE("31-jan-2009","1-jan-2009","0") !! result is 000030
      $n6 = DIFF_DATE("31-jan-2009","1-jan-2009","1") !! result is 000100
