NEXT_WORKING_DAY
   

This function returns next working day.

Format

            NEXT_WORKING_DAY(date_exp)

Return Value Type

            Date.

Parameters


·     date_exp - date or datetime field.

Note

            Working days are all days except sunday, and dates defined with IAF_HOLIDAYS symbol.

Examples

  
      $d8 = next_working_day($d8)
