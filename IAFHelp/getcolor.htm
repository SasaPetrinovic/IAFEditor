Getcolor
   

This method returns color of the object.

Format

            Getcolor (flag)


Parameters


·     flag - numeric value, integer. This parameter defines which color to be returned.

      flag       = 0,      initial back color
      flag       = 1,      current back color
      flag       = 2,      current text color

Return Value

            Numeric, RGB color.

            If object does not exist, function returns -1.
            If 'flag' is invalid, function returns -1.

Example

  
! Let's $$color defined as
      define_local $$color n 9 L          !! numeric, integer

      $$color = button_1.Getcolor(1)      !! get current back color
