DAY_OF_WEEK
   

This function returns day of the week.

Format

            DAY_OF_WEEK(date_exp)

Result Value Type

            Numeric.
            
Parameters

·     date_exp -
Note

            Result is a positive number in the range 1 to 7.
·     1 represents Monday
·     2 represents Tuesday
·     3 represents Wednesday
·     4 represents Thursday
·     5 represents Friday
·     6 represents Saturday
·     7 represents Sunday
