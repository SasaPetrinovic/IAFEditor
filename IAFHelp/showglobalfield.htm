Show global fields
   

This dialog shows all the global fields defined in the DD.

General tab

The list shows the following field attributes:

·     field name,
·     field type,
·     field size,
·     sign existence,
·     data-type and
·     edit mask


Not used tab

Shows all the fields not used by any record definition.


Cross reference tab

For particular field, shows all the records using that field definition.
