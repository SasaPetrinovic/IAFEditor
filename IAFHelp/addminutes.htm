ADD_MINUTES
   

This function adds number of minutes to a time or datetime field.

Format

            ADD_MINUTES(time, numminutes)

Return Value Type

            If 'time' parameter is a time field, return value is time.
            If 'time' parameter is a datetime field, return value is datetime.

Parameters


·     time - time_exp, or datetime field
·     numminutes - number of minutes, numeric_exp

Note 1

            If 'time' parameter is a time, out-of-range result time will be corrected automatically.

Note 2

            If 'time' parameter is a datetime, result is datetime field and all the field's components will be adjusted.


Examples

  
      $Min = ADD_MINUTES (systime,5)
      $Min = ADD_MINUTES (systime,counter_1)

      $Min = ADD_MINUTES ("23:55:00",10) !! result is 00:05:00

      $dt = ADD_MINUTES("2009-12-31 23:59:00",2) !!result is 2010-01-01 00:01:00
