ADD
   

This statement adds a numeric value to numeric field.

Format

            ADD numeric_value TO numeric_field [TRUNCATED]

Where


·     numeric_value - any numeric expression, like numeric literal, numeric field, formula, numeric function, etc.
·     numeric_field - a numeric database field, numeric local field, counter_nn or acc_nn

            TRUNCATED - this keyword can be used if decimal arithmetic enabled, see Options dialog. If numeric_field is unpacked numeric, BCD packed or integer, output result will be truncated. Otherwise, the result will be rounded according to selected rounding method.

If decimal arithmetic is disabled, or numeric_field is a floating-point number, the TRUNCATED has no effect.


Examples

  
      ADD 1 TO acc_1
      ADD acc_1 * acc_2 TO acc_3
      ADD day(date_field) TO counter_1
      ADD acc_1 TO $numarr($ndx)

      ADD ($a * $b) to $res TRUNCATED
! if $a = 0.25 and $b = 0.35 then $a * $b = 0.0875
! if $res has two decimals, its value will be $res = 0.08

      ADD ($a * $b) to $res
! if $a = 0.25 and $b = 0.35 -> $res = 0.09
