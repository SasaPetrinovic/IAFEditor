Writeelement
   

This method writes starting tag, ending tag and data in the XML file.

Format

            Writeelement (element_name, field_value [,edit_mask])

Parameters


·     element_name - an alpha variable or alpha literal, element name.
·     field_value - data field or NULL.
·     edit_mask - if the data field is numeric, date or time,  this parameter defines presentation form of a field. If the data field is alphanumeric or NULL, this parameter have to be omitted.


Return Value

            Status, numeric variable.

            Possible values:

            0          Ok
            -1         File not opened
            > 0       File input/output error

Applied To

            Xml object.

Note

            The first line in XML file should be XML declaration in the form:

            <?xml [attributes]?>

            One of attributes is ENCODING. This attribute specifies character set that is used in the document. Attribute values can be "WINDOWS-1250", "WINDOWS-1252", "ISO-8859-1", "UTF-8", "UTF-16", etc. All alpha fields have to contain characters from the character set defined by ENCODING attribute. Default value is "UTF-8".

            
Examples

  
      $stat = Xml.Writeelement("fld1", acc_1, "ZZZZ9,99")

Output line in XML file will be

      <fld1>100.00</fld1>

      $stat = Xml.Writeelement("fld1", NULL)

Output line in XML file will be

      <fld1/>
