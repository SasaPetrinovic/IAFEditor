Create SQL script from record definition
   


Available records


List on the left-side of the dialog shows all the records defined in Data Dictionary.
Select one or more records and click button ">" to move them to the right-side list.

Rdb tables

SQL script will be created for all records in this list.
To remove record from the list, select it and click button "<".

Database

Select one the the databases. Depending on this choice, various keywords will be used in the script file. Example: the data type containing data and time in MS SQL Server is DATETIME, in Oracle is DATE, in DB2 is TIMESTAMP.

Create SQL script

Click this button to create SQL script for selected tables. Result is a text file with extension SQL on default directory. This file can be modified by any text editor. Later on, this file can be used by a Database administrator program to create tables, indexes, etc.
