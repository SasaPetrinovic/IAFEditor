BREAK_FIELD
   

This statement defines break fields in batch programs. When break field changes its value, some group (total) actions are executed.


Format 1


            BREAK_FIELD = field_1|field_2|…|field_n

Format 2

            BREAK_FIELD = LAST_RECORD


Where

·     field_1,..., field_n - fields from input record. Maximum number of segments is 100.
·     LAST_RECORD - a special value which represents actions after the last record of the main file (table).

Note

            This statement can be used in batch calculations after TOTAL_CALC only.

Example

  
DETAIL_CALC
      ...
      some_code_det
      ...
TOTAL_CALC
      BREAK_FIELD = fld1
            ...
            some_code_t1
            ...
      BREAK_FIELD = LAST_RECORD
            ...
            some_code_t2
            ...

See also Detail calculations, Total calculations
