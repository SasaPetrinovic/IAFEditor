Methods
   

Methods are functions that operate on objects.

An object represents an element of a form, or report, such as label, source field, destination field, combobox, checkbox, button, form itself, subform, status bar, etc.

Object must be identified by its name. The object name is a standard identifier, like Label_1, Table_1, Subform_1. If an object belongs to a tabcontrol, identifier is in the form tab.tabpage.object_name, like Tab_1.tabPage_1.Label_1.

Object name can be seen in form(report) painter selecting 'Show/Object names' in main menu, or activating 'Property dialog' of the object. In this dialog, object name can be changed, too.

Syntax:

If the method does not return a value, the statement format is

            Object_name.method(arg1, arg2, ..argN)

Example

            Button_1.Setvisible(0)

If the method returns a value, the statement format is

            output_value = Object_name.method(arg1, arg2, ..argN)

Example

            $result = Table_1.Getcell($line, $column)
            $result = Label_1.Objvalue()


Object_name is an identifier or local field name.


Examples

            Sourcefield_1.Display(acc_1)
            Sourcefield_1.Display(Compress(fld1,fld2))
            Sourcefield_1.Display(alltrim(fld1) | ',' | alltrim(fld2))
            Table_1.Display(current_line,1,"some text")
            Table_1.Display(*,1,"some text")
            Table_1.Display(counter_1,1,"some text")
            Table_1.Init()
            Table_1.Setfocus()
            Subform_temp.Display("Working ...")
            Me.SetFocus()
            Tab_1.tabPage_1.Setcurrentpage()
            Tab_1.tabPage_1.Label_1.Setvisible(0)

            $objnam = "Table_1"
            $objnam.Display($line,1,"some text")

            $result = $objnam.Getcell(5,1)
            $result = $objnam.Gettopline()
            $result = $objnam.Getnumberrows()
            $result = Table_1.Getnumberrows()

Objects

            LABEL, SOURCE FIELD, DESTINATION FIELD, LINE, RECTANGLE,
            ELLIPSE, 3D PANEL, PUSH BUTTON, BITMAP BUTTON, IMAGE,
            COMBOBOX, CHECKBOX, RADIO BUTTON, GROUPBOX, TABLE (GRID),
            PROGRESS BAR, TREE, GRAPH, DATA SELECTOR, TIMER, SUBFORM_nn,
            GROUP, FORM, SUBFORM_TEMP, STATUS_BAR, CURRENT_MOVE_OBJECT, ME.

            CURRENT_MOVE_OBJECT is an object over which the cursor moves,

            ME is special object denoting current object, i.e. the object containing current line of the code.
