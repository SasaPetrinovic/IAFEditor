Insertrow
   

This method inserts an empty row into opened MS Word table.

Format

            Insertrow (below_row)

Parameters


·     below_row - row, numeric value. New empty row will be inserted below the one defined by the parameter. If 'below_row' is greater than number of the rows in the table, new line will be added at the end of the table.


Return Value

            Status, numeric variable.

            Possible values:

            0          Ok
            -1         Document not opened
            -2         Table not opened

Applied To

            Word object.

Examples

  
      $stat = Word.Opentable("#1",0,0)
      if $stat = 0
            $stat = Word.Insertrow(1) !! insert row 2 (below row 1)
            $stat = Word.Closetable()
      end_if
