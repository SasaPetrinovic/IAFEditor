DIFF_MONTHS
   

This function returns number of months between two date values.

Format

            DIFF_MONTHS(date1, date2)

Result Value Type

            Numeric.

Parameters


·    date1, date2 - date or datetime fields.

Note

            Result is a positive number.
            
            Algorithm:

            Diff_months = abs (12*year(date1) + month(date1) - 12*year(date2) - month(date2))

Examples

  
      $n = DIFF_MONTHS('2009-02-01', '2009-01-01') !! result = 1
