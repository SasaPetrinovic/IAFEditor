Using Word - Example 1
   

Open existing Word file enabling review, editing and all other options.

$wstat = Word.Open("docdir:text1.doc")
if $wstat = 0
      $wstat = Word.Run()
end_if


Open existing Word file just enabling review.

$wstat = Word.Open("docdir:text1.doc","read_only")
if $wstat = 0
      $wstat = Word.Run()
end_if


Print Word file on default printer without user action.

$wstat = Word.Open("docdir:text1.doc","read_only")
if $wstat = 0
      $wstat = Word.Run("Print")
      $wstat = Word.Close()
end_if
