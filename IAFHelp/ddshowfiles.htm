Show Files
   

File name

This is file name filter. Wildcard characters are: * and ?.

Examples:
            *
            A*
            *A
            *A*
            ????
            AB??

List

The list shows already defined files according to the filter. The line contains the following information: file name, file organization, record format, record size (in bytes) and number of keys.

Show file

Show detailed information about the selected record.

Print

Print file information or store it into text file.


Cross ref Tab

List on the left-side of the dialog shows all file definitions in the DD.
Click the file name to see cross reference in the right-side list: the list of all the programs which use that file definition.

Rebuild programs

Rebuild all the programs from the right-side list. This option generates source files for each program in the list and one command file: IAFRBLD.BAT. At the end of this process a message appears on the screen: Launch IAFRBLD.BAT? If our answer is "Yes", the command file will be activated and as a result, executable programs will be generated.
