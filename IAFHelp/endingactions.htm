Ending actions
   

This piece of code executes when user clicks Close button, presses ESC key or when program executes STOP_RUN statement. After ending actions, program frees all allocated memory blocks, updates Symntt00.iaf file,  rollbacks pending transaction (if exists), closes all opened files and disconnects from database.

Maximum number of lines is 10. If this number is too little, PERFORM statement can be used to execute a procedure.
