Text properties
   

Object name

When the new static text is created, generator automatically assigns the name Label_nnn, where nnn = 1, 2, …
Object name can be maximum 30 characters long. Allowed characters are A-Z, 0-9, _ and $.
First character must be a letter. Object name is case-insensitive. Object name can be used in Methods.

Member of group (information only, cannot be changed)

If the object is a member of a group, the name of this group is displayed. An object can be the member of one group only.

Caption

The text which appears on the screen or report.
If you want to display single ampersand in the caption, enter two consecutive ampersands.

Visible

Determines whether the object is visible when the subform is first displayed.

Font name

Choose one from offered font names.

Font size

Specifies font size of the text.

Bold

Specify that font weight is 700. Normal weight is 400.

Italic

Specify whether text is italic.

Underline

Specify whether text is underlined.

Orientation

Orientation of text, in degrees. Horizontal text has orientation 0.
Possible values are between -90 and 90 degrees.

Background style

  
Normal
Background is filled with the current background color
Transparent
Background remains unchanged (color of the subform or report underneath the object is visible)

If you plan to use the Display method with this object, background style should be set to normal, otherwise you can choose between normal and transparent.

Background color

This attribute is applied only if the background style is normal.

Special effects

Values:

Flat
Raised
Sunken
Etched
Bump
Shadow

Border style

  
Transparent
Border is not seen
Solid
Border is standard line
Dots
Border is dotted line
Dashes
Border is dashed line

Border color

Is applied only if border style is not transparent.

Border width

Is applied only if border style is not transparent. Possibly values are 0, 1, 2,... pixels.

Text color

Color of text.

Horizontal text alignment

Values:

Left
Center
Right

Vertical text alignment

Values:

Top
Bottom
Center

Shading style

Valid if background style is Normal. Specifies that Color_from fades gradually to Color_to.

Values:

None                No shading
Horizontal
Vertical

Shading variant

Valid if shading style is Horizontal or Vertical.
Possible values are 1 to 4.

Color from

Starting color. Valid if shading style is Horizontal or Vertical.

Color to

Ending color. Valid if shading style is Horizontal or Vertical.

Z-Order

Possible values: 0, -1, -2, -3, ... See Z-Order.

Tooltip text

Tooltip text is a text that will be displayed when the cursor hovers above the label. See Tooltip text definition.
