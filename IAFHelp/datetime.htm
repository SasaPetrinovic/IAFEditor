DATETIME
   

This function returns a datetime field.

Some databases use datetime data type to store date and time data.
In Oracle, this is date data type, is MS SQL Server datetime, in DB2 timestamp, etc.

Format 1

            DATETIME(date_part, time_part)         Stores the both parts of the field.

Format 2

            DATETIME(, time_part)                        Stores time part only. Date part remains unchanged.

Format 3

            DATETIME(date_part)                          Stores date part only. Time part remains unchanged.


Return Value Type
            
            Datetime.

Parameters


·    date_part - date field.
·    time_part - time field.


Note

            In an assigment
            ------------------------------------------------------------
                        datetime_field = date_field
            ------------------------------------------------------------
            program zeroes time part of output datetime_field.

Examples

  
            $datetime = DATETIME(today,systime)
            Database_Field = DATETIME(,systime)
            Database_Field = DATETIME(today)
