Search global procedures / functions
   

In this dialog you can enter search criteria.

Procedure/function name


Enter procedure/function name, wildcards characters (* and ?) allowed.

Examples:
  
*
 All procedures
A*
 Procedure names starting with letter A
*A
 Procedure names ending with letter A
*PRC*
 Procedure names containing PRC
PRC?
 Procedure names exactly four characters long, starting with PRC


User

Enter * for all users or a user name.

Search string

Enter the text which will be searched for.

Look in

Select one of options: procedures and functions, procedures only, functions only.

Match whole word only

A word is a text preceded and followed by space, tab or a punctuation character.

Match case

When selected, searching is case sensitive.
