CHECK_DIGIT
   

This function calculates check digit.

Format

            CHECK_DIGIT (field, field_length, modulo, qualifier)

Return Value Type

            Alpha.

Parameters

·    field - alphanumeric, unpacked numeric field
·    field_length - number of characters
·    modulo, qualifier
  
Modulo
Qualifier
Description
10
0
modulo 10, weight digits 1,2
10
1
modulo 10, weight digits 1,2 reduced to single digit (IBM check for credit cards)
10
2
modulo 10, weight digits 1,3,7 (routing numbers for EFT)
11
0
modulo 11, weight digits 7,6,5,4,3,2; all numbers included
11
1
modulo 11, weight digits 7,6,5,4,3,2; ignore remainders 0 and 1
11
2
ISBN mod 11 check
1000
0
barcode check digit, EAN13, EAN8

Examples
  
      $Dgt = CHECK_DIGIT("0001",4,11,0) !! result is 9
      $Dgt = CHECK_DIGIT("860016960002",12,1000,0) !! result is 2
      $Dgt = CHECK_DIGIT("2050000000011742",16,97,0) !! result is 7
            $fld = alltrim($fld) | check_digit($fld,12,1000,0)
