Tree
   

We use a tree to display hierarchical structure of data.

To insert new tree on the form, click button Tree in toolbox or choose menu command Objects/Tree. Move the cursor to the desired position,  then click. This is the top left corner of the tree. Drag to the right and bottom and release the button. Tree definition dialog appears on the screen.

Num levels        number of the levels in the tree. If, for example, you want to display companies and their departments, number of levels should be 2: in the first level the company names will be displayed, and in the second level, departments belonging to each company will be seen.

Initial filling        If TRUE, the level will be filled when program control reaches this object by sequence. If FALSE, this level will be empty initially. If the user, at run-time, wants to see child items, double click will activate appropriate 'Insert code' and expand the node.

Sort elements   determines order of appearance of the tree items. If TRUE, all the items within one level will be sorted internally by the tree. If FALSE, order of appearance is defined by insertion order.

Normal image   the two images are assigned to each tree item. Type of image is BMP, 16x16 pixels. When the item is not selected, this image will be seen on the screen.

Selected image when you select an item at the run-time, the tree will display this image on the screen.

Fill the tree by data

Now, after the tree has been defined, we have to write the code for tree filling. Select the tree, right click to display context menu and choose Edit tree code. In edit frame, select 'Level 1: Insert code', enter the code for level 1 filling and click Save. After that, select 'Level 2: Insert code' and so on.
