Group of objects

            All objects within a GROUPBOX object make an implicit group. Group name is name of the group box.
            An explicit group can be defined via 'Objects/Groups/Create group' command.

            To initiate a group operation, invoke GROUPBOX_NAME.method(), or EXPLICIT_GROUP_NAME.method().
