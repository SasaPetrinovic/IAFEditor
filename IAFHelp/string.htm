STRING
   


This function returns string text.

Format

            STRING(identifier)

Return Value Type

            Alpha.

Parameters


·     identifier - string identifier, case-insensitive.


Examples

  
      errmsg = STRING(IDS_1)
      errmsg = string(ids_1)
      obj.Display(string(ids_1))
      $loc = STRING(IDS_1)
