Options
   

Save current work

Defines autosave time interval: how often we want the painter to save the program. The form painter writes all the information about the program in recovery file named program_name.JOU. If the painter crashes, this file will be used for recovery when we try to modify this program later. If the painter successfully exits, this file is deleted.

Store text labels as

If we select ANSI, labels will be stored as ANSI text: one byte represents one character. Only characters from selected code page can be stored to the label.

If we select Unicode, labels will be stored in UTF-8 character set: one character is represented by multiple bytes. In this case, we can use any character in a label.

Code-page for literals, program title, etc

Program generator is not an Unicode application itself. All the literals in program code are stored as ANSI text, according to the current code-page for the system.

Number of tooltext lines

If 'View/Show tooltip' option is checked, number of tooltip text lines is 10, by default. To see larger piece of the code, increase this value.

Show field name / code

By default, tooltip text shows field name only in objects like 'Destinationfield', 'Sourcefield', etc. If you want too see the code behind the object, choose 'Show code' option.

Use decimal arithmetic

If you want to use decimal arithmetic in formulas and aggregate functions,  set this option. This kind of arithmetic will be used always when result field is an unpacked or BCD packed numeric. We recommend to use unpacked numbers.

Rounding method

If a value has more decimals than result field, rounding is necessary.

Rounding algorithm:
- retain the desired number of decimals,
- take the digit to the right of the place we wish to round to,
- if the digit is 0, 1, 2, 3 or 4, drop remain decimals,
- if the digit is 6, 7, 8 or 9 the last significant digit will be rounded up,
- if the digit is 5, round previous digit according to the chosen rounding method:

            Half up (round half away from zero): last significant digit will be rounded up.
            Half even (round half to even): last significant digit will be rounded to the nearest even digit (Banker's rounding).

Generate interoperable SQL code

During program generation phase, each READ command, including default read, will be transformed into appropriate SQL SELECT statement. This is a standard SELECT statement, but according to the chosen database, see Change setup parameters dialog, some extensions can be added to the statement. Those extensions prevent the program to access different databases without re-generation.
If this check box is selected, programs will be generated without particular SQL extensions, enabling maximum interoperability: by simply changing value of DSN_APPLNAME symbol at run-time, the same program can access another database management system.

This parameter has no impact to SELECT, SELECT_ALL, SELECT_START, EXECSQL.

Example

Default read for a table, access mode=UPDATE, database=MS SQL Server, generates

SELECT TOP 1 * FROM mytable WITH(UPDLOCK) WHERE ...

Interoperable command will be

SELECT * FROM mytable WHERE ...


Write the list of unused local fields/local procedures to GER file

If this check box is selected, names of unused local fields and local procedures will be written to GER file during program generation phase.

Write the list of USED local fields to GER file

If this check box is selected, names of all used local fields will be written to GER file during program generation phase. Using simply copy / paste method we can remove all unused local fields from a program.

Store language info into Data Dictionary

If you plan to create a multilingual application, this option should be checked. In this case, language info will be stored into the Data Dictionary on exit from the painter. To enable this check box, two conditions must be satisfied: 'Store text labels as Unicode' on this dialog, and 'International support enabled' on International support dialog. This information will be used, later on, by Translation Software to translate text from your default language to any other language.
