Writeendnode
   

This method writes ending tag of the node.

Format

            Writeendnode (node_name)

Parameters


·     node_name - an alpha variable or alpha literal, node name.


Return Value

            Status, numeric variable.

            Possible values:

            0          Ok
            -1         File not opened
            > 0       File input/output error
            
Applied To

            Xml object.

Examples

  
      $stat = Xml.Writeendnode("node1")

Result line in XML file will be

      </node1>
