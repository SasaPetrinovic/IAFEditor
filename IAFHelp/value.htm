VALUE
   

This function converts numeric literal, date, date-time or time field to the numeric value.

Format


            VALUE(value_exp)

Return Value Type

            Numeric.


Parameters


·    value_exp - a value expression: alpha, date, time, datetime or numeric.

Notes

            The function is usually used to converts alpha fields containing numeric literals to numeric fields, because alpha fields cannot be used in calculations. The function skips leading spaces and ends with the first non-digit character. Thousands group separator is ignored. If input literal has more decimals than output field, excess decimals will be truncated.


            If 'value_exp' is a numeric field, this is an ordinary numeric move.

            If 'value_exp' is date, time or datetime, the result is an integer with appropriate number of digits.

Examples

  
! decimal point = comma (,)
      $numfld = value('-1234,56')         !!is the same as
      $numfld = value('-1.234,56')        !!is the same as
      $numfld = value('   -1234,56')      !!is the same as
      $numfld = value('-1234,56abc')      !!is the same as
      $numfld = value('  -1234,56abc')    !!is the same as
      $numfld = -1234,56
!
! decimal point = point (.)
      $numfld = value('-1234.56')         !!is the same as
      $numfld = value('-1,234.56')        !!is the same as
      $numfld = value('   -1234.56')      !!is the same as
      $numfld = value('-1234.56abc')      !!is the same as
      $numfld = value('  -1234.56abc')    !!is the same as
      $numfld = -1234.56
!
      $numfld = 0
      if $alphafld is numeric_literal
            $numfld = value($alphafld)
      end_if
      $result = $numfld * (acc_1 - acc_2)
!
      $numfld = value(today)        !!8 digits, like 20100601
      $numfld = value(timestamp)    !!14 digits
      $numfld = value(systime)      !!6 digits
!
      $numfld = value(acc_1)        !!is the same as
      $numfld = acc_1

! hexadecimal input value. Hexadecimal literal starts with 0x.
      $numfld = value('0xFF')       !! result = 255

! Trailing minus
      $numfld = value('1234,56-')   !! result = -1234,56

! Leading and trailing signs: trailing sign will be ignored
      $numfld = value('+1234,56-')  !! result = 1234,56
