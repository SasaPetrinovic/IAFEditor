Progressbar properties
   

Object name

When the new progress bar is created, generator automatically assigns the name Progress_nnn, where nnn = 1, 2, …
Object name can be maximum 30 characters long. Allowed characters are A-Z, 0-9, _ and $.
First character must be a letter. Object name is case-insensitive. Object name can be used in Methods.

Visible

Determines whether the object is visible when the subform is first displayed.

Background style

Background style should be set to normal.

Background color

This attribute is applied only if the background style is normal. Double click brings up a color definition dialog.

Special effects

  
Flat
Border is standard line
Raised
Border with raised edge
Sunken
Border with sunken edge
Etched
Border is etched line
Bump
Border is bumped line
Shadow
Shadow below and to the right of the rectangle


Border style

  
Transparent
Border is not seen
Solid
Border is standard line
Dots
Border is dotted line
Dashes
Border is dashed line

Border color

Is applied only if border style is not transparent.  Double click brings up a color definition dialog.

Border width

Is applied only if border style is not transparent. Possibly values are 0, 1, 2,... pixels.

Text color

Color of text. Double click brings up a color definition dialog.
