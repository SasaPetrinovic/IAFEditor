ALLOCSTRING
   

This function creates new memo field and allocates continuous memory block to it.

Format

            ALLOCSTRING (number_of_chars)

Return Value Type

            String.

Parameters


·     number_of_chars - number of characters, size of memory block, numeric.


Note

            When we use many consecutive concatenations, heap memory can be fragmented (because every new concatenation can request enlarged memory block) :

            $s = 'Any text'
            $s = concat($s,'string1')
            $s = concat($s,'string2')
            ...
            $s = concat($s,'stringN')

            To avoid such situation, we can initially allocate a memory block of sufficient size.

            $s = allocstring(1000)    !! Allocate block of 1000 characters
            $s = 'Any text'
            $s = concat($s,'string1')
            $s = concat($s,'string2')
            ...
            $s = concat($s,'stringN')


            ALLOCSTRING can be used in assignment statement only.


Examples

  
      $memo = ALLOCSTRING(10000)
! If $memo is defined as ANSI memo, above function allocates 10000 bytes.
! If $memo if defined as UTF-16 memo, 20000 bytes will be allocated.
