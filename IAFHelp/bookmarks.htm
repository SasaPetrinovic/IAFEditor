Bookmarks
   

The bookmarks will be written into the print file. The print file will be created if we click Preview button in Print dialog or if we set IAF_PRINTMODE internal variable to 9 or 12 (See Internal fields).

Bookmark is document navigation tool allowing users to instantly skip to any location in the print file.

If you use this feature, bookmarks tree will be displayed when you review print file. Usually, this tree resides on the left side of the screen.


Create bookmarks

Enables bookmark creation.

Generate default bookmarks

If bookmark creation is enabled, this option creates default bookmarks for the report. If this check-box is cleared, you have to create custom bookmarks using context menu command "Create bookmark: level N".

Root node text options


·     Root node does not exist.
·     Document title - root node text is document title defined in Report parameters dialog.
·     Program caption - root node text is program title.
·     Value expression - root node text is an alpha literal or alpha variable.

How to create default bookmarks?


·     Select 'Create bookmarks'.
·     Select 'Generate default bookmarks'.
·     Optionally set root node text.
·     Click OK.


How to create custom bookmarks?


·     Activate menu command Define bookmarks. The dialog appears on the screen.
·     Select 'Create bookmarks' check-box.
·     Optionally set root node text.
·     Click OK, dialog disappears from the screen.

·     Select object(s) representing your bookmark (click, Ctrl+click).
·     Right click to display context menu.
·     Select bookmark level by "Create bookmark: level N".
 

Modify existing bookmark (level N)


·     Select object(s) representing new bookmark (click, Ctrl+click).
·     Right click to display context menu.
·     In submenu, select "Level N". Previous objects will be replaced by new objects.


Show custom bookmarks

Select menu command Show bookmarks. All the objects belonging to the bookmarks will be displayed in blue color.

Remove all the bookmarks


·     Select menu command Define bookmarks.
·     Clear 'Create bookmarks' check-box.
·     Click OK.
