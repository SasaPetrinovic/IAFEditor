Setfocus
   

This method sets the focus to the specified object.

Format

            Setfocus()

Parameters

·     none


Applied To

            Destination field, push button, bitmap button, combo box, check box, radio button, grid, tree, data selector.


Notes

            - If the object is invisible or disabled, Setfocus will not be executed.

            - Compared to traditional programming, Setfocus corresponds to GOTO statement. Statements after Setfocus will not be executed.


Example

  
      table_1.SetFocus()
