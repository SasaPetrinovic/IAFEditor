Setsheet
   

This method changes current sheet.

Format 1

            Setsheet (numsheet)

Format 2

            Setsheet (sheet_name)


Parameters


·    numsheet - sheet number, numeric value.
·    sheet_name - sheet name, alpha literal or variable.


Return Value

            Status, numeric variable.

            Possible values:

            0          Ok
            -1         Document not opened
            -2         Invalid sheet number
            
Applied To

            Excel object.

Examples

  
      $stat = Excel.Setsheet(2)
      $stat = Excel.Setsheet(counter_1)
      $stat = Excel.Setsheet("Sheet3")
 

Using Excel
