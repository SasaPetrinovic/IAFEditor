MAIL
   

This statement sends a mail.

Format

            MAIL address,subject,body,attachment[,"NODIALOG"]

Where

·     address - recipient's address
·     subject - subject text (can be empty string)
·     body - message text (can be empty string)
·     attachment - files to be sent as attachment, separated by semicolon (can be empty string)
·     NODIALOG - enables to send email without the dialog box prompting the user for sending options. Can be abbreviated to "N". If this parameter is omitted, the dialog box appears on the screen.


Note

            This statement will launch default e-mail client on local PC. A program simply uses set of functions exported by MAPI32.DLL to prepare and send a mail.
            MAPI32.DLL must be installed.

Examples
  
      $address = "COMPANY@XXX.NET"
      $body = "message text, …"
      $attach = "c:\a1.txt; c:\a2.txt"

      MAIL $address, "subject text", $body, $attach
      MAIL "COMPANY@XXX.NET", "subject text", $body, $attach
      MAIL $address, "subject text", $body, $attach,"NODIALOG"
      MAIL $address, "subject text", $body, $attach,"N"
      MAIL $address, "subject text", $body, "","N"
