CHAIN
   

This statement calls another executable program.

Format

            CHAIN program_name

Where


·     program_name - name of the executable program (must be an alphanumeric literal or an alpha field). Command line parameters can be used.

Note

            The current (active) program stops the execution after activating next program.


Examples

  
      CHAIN "nextpgm"
      CHAIN "nextpgm PARAM=1"
      CHAIN "c:\test\pgm1"
      CHAIN "apli_exe:pgm1"
      $pgm = "P1.EXE"
      CHAIN $pgm
