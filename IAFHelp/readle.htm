READ_LE
   

This statement reads a record from an indexed file which key is less than or equal to given key.

Format

            READ_LE file KEY[n] = key DECREMENT field

Where

·    file - the name of the file from datasource. File has to be indexed one.
·    key - concatenation
·    field - one of the fields listed in key. It has to be numeric, date or time type.


Because we currently deal with RDB tables and sequential files, this statement will not be further explained.
