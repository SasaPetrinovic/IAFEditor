ExportPDF
   

This method exports a Winword document to a PDF file.

Format

            ExportPDF (pdf_file_name)

Parameters

·     pdf_file_name - output PDF file name, absolute path, alphanumeric.


Return Value

            Status, numeric variable.

            Possible values:

            0          Ok
            -1         Document not opened
            
Applied To

            Word object.

Note

            The document must be opened.

Examples
  
      ...
      $stat = Word.Open('c:\temp\File.doc')
      $stat = Word.ExportPDF('c:\temp\File.pdf')
      $stat = Word.Close()
      ...
