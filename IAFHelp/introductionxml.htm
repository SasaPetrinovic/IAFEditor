Introduction
   

XML file is a text file combining data and tags like HTML. Such files can be edited by any text editor. In Data Dictionary, we can define these files as "sequential, variable record format". Therefore, instructions like WRITE, READ_ALL can be used to process XML files.


To make processing these files easier, we use object named Xml and its methods.
There is only one Xml object in a program. This means that two files cannot be opened at the same time. In this case, XML file  doesn't need to be defined in the Data Dictionary.


All methods return a value.


Object name:

            Xml

Methods:

            Open
            Close

            SetPrefix
            SetXmlDecPoint

            WriteText
            WriteNode
            WriteEndNode
