TOTAL_CALC
   

Part of batch calculations which are executed at break, i.e. when break field change its value.

Format


            TOTAL_CALC


Example

  
TOTAL_CALC
      BREAK_FIELD = fld1 | fld2
            ...
            some code
            ...
      BREAK_FIELD = fld1
            ...
            some code
            ...
      BREAK_FIELD = LAST_RECORD
            ...
            some code
            ...

See also Detail calculations, Break_field
