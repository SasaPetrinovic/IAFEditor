Search Data Dictionary
   

We use this dialog to search for string(s) in programs, global procedures/functions and menus.

Search for string(s)

Enter the text string(s) you want to search for. String separator is COMMA, for example

      string1, string2, string3

If a particular string contains spaces, quotation marks have to be used:

      "string1 with spaces", string2, "string3 xxx"

If the string contains leading or trailing spaces, use quotation marks:

            "   leading spaces", "trailing spaces   "

If the string contains a TAB character (tabulator), use escape character in the form \t (backslash lower-t):

            "\tleading_tab"


Boolean expression

  
OR
Search is successful if any of entered strings exists.
AND
Search is successful if all the strings exist.
NOT
Search is successful if entered string does not exist. In this case, only one string can be entered.
AND NOT
Search is successful if the first string exists and the second string does not exist. In this case, exactly two strings have to be entered.


Match case

If this check box is not selected, all strings are converted to uppercase before comparison.

Search in

Select appropriate program type, procedures/functions or menus.



Note

To activate appropriate generator, double click the list.
