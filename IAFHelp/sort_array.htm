SORT_ARRAY
   

This statement sorts an array.

Format

            SORT_ARRAY array , sort_order , starting_index, ending_index ,collation [, dep_array1, dep_array2, ..., dep_arrayN]

Where

·     array     - main array to be sorted, required parameter.
·     sort_order - "A" for ascending, "D" for descending. Default is "A". If we use an alpha variable and the variable contains invalid value (a character different from "A" or "D"), ascending order will be applied. If the length of the variable is greater than 1, the first character will be used.
·     starting_index - index of the first element to be sorted, integer. Default value is 1 (the first element).
·     ending_index - index of the last element to be sorted, integer. Default is the last element.
·     collation - collation name. Default is binary sort. This parameter is ignored if main array is not an alpha one.
·     dep_array1, dep_array2, ..., dep_arrayN - dependent arrays. These arrays will be rearranged according to main array.



Note

Consider three arrays, each of them have 3 elements:
  
$main_array
$dep_arr1
$dep_arr2
'11'
101
1001
'12'
102
1002
'13'
103
1003


After command

            sort_array $main_array,'D',1,3,,$dep_arr1,$dep_arr2

array elements will be
  
$main_array
$dep_arr1
$dep_arr2
'13'
103
1003
'12'
102
1002
'11'
101
1001


Examples

  
      sort_array $arr               !!ascending sort, all elements
      sort_array $arr,"d"           !!descending sort
      sort_array $arr,"a",1,10      !!sort first 10 elements

      $sortdir = "A"
      sort_array $arr, $sortdir, 1, counter_1

      $sortdir = "X"
      sort_array $arr, $sortdir     !!"A" will be applied at runtime

      $sortdir = "DA"
      sort_array $arr, $sortdir     !!sort order is "D"

      sort_array $arr,'a',1,10,'English_CI'     !!English collation

      sort_array $arr,'a',1,10,,$arr1,$arr2     !!dependent arrays
      sort_array $arr,,,,,$arr1,$arr2
