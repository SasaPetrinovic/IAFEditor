READ_ALL
   

This statement reads group of records from a file or RDB table in a cycle.

Format 1


            READ_ALL [/load_key][/nohold][/trim]  file [[KEY[n] = key]] [WHERE cond_exp]
                        ...
            END_READ

Format 2 (for ISAM files only)


            READ_ALL [/load_key][/nohold][/trim] file
                        [[STARTING_KEY[n] = key [ENDING_KEY[n] = key]] [WHERE cond_exp]
                        ...
            END_READ

Format 3 (for XML dataset)


            READ_ALL XML_DATASET(xml_variable[, node_name])
                        ...
            END_READ

Where

·    file - the name of the file (table) from datasource dialog
·    key - concatenation of key segments
·    cond_exp - conditional expression. Expression after WHERE serves for record selection
·    xml_variable - alpha variable containing XML dataset. Variable type is CLOB, longchar.
·    node_name - alpha variable or alpha literal containing initial node name. If omitted, default value is 'Newdataset.Table'.


Allowed qualifiers

            /LOAD_KEY      prepare data-entry grid
            /NOHOLD         read an ISAM record without locking
            /TRIM               trim last key segment (ISAM files only)


Notes

            Cycle ends with END_READ statement.

            In format 3, the first call moves internal pointer to the first occurrence of node defined by NODE_NAME. If the node does not exist, program leaves READ_ALL ... END_READ cycle immediately. Next call moves internal pointer to the next node, etc. Look example below.

Examples

  
!! Read a seq. file
      READ_ALL seqfile
            ...
      END_READ

!! Read a group records from an indexed (Btrieve) file by key2
      READ_ALL idxfile KEY2 = seg1 | seg2
            ...
      END_READ

!!    
      READ_ALL idxfile starting_key='0' ending_key='9'
            ...
      END_READ

!! Read RDB table
      READ_ALL tbl Where ... ORDER BY ...
            ...
      END_READ

Above instruction will be translated to
      SELECT_ALL * FROM tbl Where ... ORDER BY ...
            ...
      END_SELECT
