Download
   

This function downloads a file (page) from the Internet using HTTP protocol (GET method) and saves it on the local machine.

Format

            Download (URL, localfile [, headers, username, password, override_flag [,progressbar_object]])

Parameters


·     URL - universal resource locator, like http://google.com/, http://mysite/files/xx.zip, etc.

·     localfile - an alpha variable or alpha literal. Local file name.

·     headers - additional headers to be sent to the HTTP server, optional. Two or more header fields should be separated by CRLF (carriage-return/line feed pair).

·     username - an alpha variable or alpha literal, optional.

                  If username is not an empty string, the function generates HTTP header field

                              Authorization: Basic username:password

·     password - an alpha variable or alpha literal, optional.

·     override_flag - an alpha variable or alpha literal, optional. Override flag: value "O" denotes that local file will be overridden if exists. If the parameter has the value different from "O" and local file exists, error will be reported. Default value = "O".

·     progressbar_object - an alpha variable or alpha literal, optional. Name of a progress_bar used to display progress of file transfer.


Return Value

            Numeric.

            Possible values:

                        0          OK.
                        -1         Internet connection cannot be opened.
                        -2         Invalid local file path.
                        -3         Incomplete download.
                        -4         Local file exists, override_flag = 'N'.
                        > 0       HTTP errors, for example 404 means 'File not found'.

Note

            Transfer method is BINARY.

Examples

  
      $intstat = Download('http://mysite.com/f1.zip','c:\temp\f1.zip',,,,'O','Prog_1')
      $intstat = Download('http://mysite.com/','c:\temp\ini.html')
