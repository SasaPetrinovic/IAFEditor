RESET
   

This statement closes, deletes, re-creates and opens empty file. Also, it deletes all records from an RDB table.

Format

            RESET file


Where

·    file - the name of the file (table) from datasource dialog. The access mode must be one of WRITE, WRITE_NEW, UPDATE.

Note

            RESET rdb_table_name                is the same as

            EXECSQL DELETE FROM rdb_table_name


Example
  
            RESET file1
 

See also OPEN   File.Reset
