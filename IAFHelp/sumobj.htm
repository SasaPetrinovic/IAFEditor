SUMOBJ
   

This function summarizes values of numeric objects (source field, destination field).

Format

            SUMOBJ(obj1_name, obj2_name, …, objN_name)

Return Value Type

            Numeric.

Parameters

·     obj1_name,..., objN_name -
Note

            If an object is not visible, its value is 0 (zero).

Examples
  
      acc_1 = sumobj(sf_1, sf_2, sf_3)
      acc_1 = sumobj(sf_1, formula_1, formula_2)
