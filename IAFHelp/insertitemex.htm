InsertItemEx
   

This method inserts an item into a combobox, listbox or dataselector, enabling multi-column listbox portion of the combobox.

Format


            InsertItemEx(data, field1, field2, ..., fieldN)


Parameters


1.   data - data to be stored in this node. When user selects an item, the data will be transferred to appropriate variable. Some special data values enable us to perform commands such as: set listbox headings. This value is: "%heading%"".
2.   field1, field2, ..., fieldN - these fields are displayed in the listbox portion of the combobox, when user clicks combobox down arrow or presses any key on the keyboard. Numeric and date fields will be edited using default edit mask. Column widths are defined automatically at run-time. Default column alignment is: LEFT for alpha and date fields, RIGHT for numeric ones.

Applied To

            Combo box, Data selector.

Note

            If you use special value %HEADING% to set listbox headings, the first text denotes heading for data, next one for column 1, etc. If data is not visible (look Combo box properties, attribute Display data and text), heading text for data will be ignored.

            It is possible to display checkbox in a column of listbox. To setup this option, use %CHECKBOX% special value and one string for each column: if column displays ordinary text, enter empty string; if column has checkbox form, enter checked value:

            combo.InsertItemEx("%checkbox%","","","Y","")

            In the above example, column 3 displays checkbox. "Y" denotes 'checked' value.

Handling disabled items in dataselector and combobox

            If you want to display disabled items, use the following format

            DataSelector.InsertItemEx(data,field1,'%DISABLED%')
            ComboBox.InsertItemEx(data,field1,field2,'%DISABLED%')

            Disabled items cannot be selected.

Inserting subheader in listbox

            Listbox.InsertItemEx('%SUBHEADING%','Subheader text')


Examples

  

! Example: Fill-in a combo box
! ============================
! Set listbox headings
      combo_1.InsertItemEx('%heading%','Data','Col 1','Col 2')
! Insert values + two columns
      combo_1.InsertItemEx('01','Text_1','Text_1_1')
      combo_1.InsertItemEx('02','Text_2','Text_2_1')
      combo_1.InsertItemEx('03','Text_3','Text_3_1')
      combo_1.InsertItemEx('04','Text_4','Text_4_1')
      combo_1.InsertItemEx('05','Text_5','Text_5_1')
! Insert disabled item
      combo_1.InsertItemEx('05','Text_5','Text_5_2','%disabled%')

! At run-time, if user selects "Item 3", value "03" will be moved to
! the variable defined by "Destination = prog_var"

! Fill-in a data selector
!========================
      dataselector_1.InsertItemEx('01','Text_1')
      dataselector_1.InsertItemEx('02','Text_2')
      dataselector_1.InsertItemEx('03','Text_3','%disabled%')
      dataselector_1.InsertItemEx('04','Text_4')
      dataselector_1.InsertItemEx('05','Text_5')

! Fill-in listbox
! Listbox is a combobox with 'Show combo as Listbox' attribute set
      listbox_1.InsertItemEx('%subheading%','Group 1')
      listbox_1.InsertItemEx('01','Item 1_1')
      listbox_1.InsertItemEx('02','Item 1_2')
      listbox_1.InsertItemEx('%subheading%','Group 2')
      listbox_1.InsertItemEx('03','Item 2_1')
      listbox_1.InsertItemEx('04','Item 2_2')
