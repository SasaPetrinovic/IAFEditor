Function key handlers
   

A function key event handler is a procedure activated as the result of pressing a function key.
The function keys are F1, F2, F3, ..., F12 with modifiers Ctrl+, Shift+ and Alt+.

Each event handler has the form

ON_EVENT_PROCEDURE OnKeyPress(functionkey_identifier)
            ...
            event handler code
            ...
END_PROCEDURE

On_event_procedure and OnKeyPress are keywords denoting the start of the handler.
End_procedure denotes end of the particular handler.

Functionkey_identifier defines function key, case-insensitive, and has the form

[Ctrl+][Shift+][Alt+]function_key

Examples:

F5
Ctrl+F5
Shift+F5
Ctrl+Shift+F5
Alt+F5
Ctrl+Shift+Alt+F5

Code examples:

ON_EVENT_PROCEDURE OnKeyPress(F5)
            ...
            event handler code
            ...
END_PROCEDURE

ON_EVENT_PROCEDURE OnKeyPress(Ctrl+F5)
            ...
            event handler code
            ...
END_PROCEDURE

ON_EVENT_PROCEDURE OnKeyPress(Ctrl+Shift+F5)
            ...
            event handler code
            ...
END_PROCEDURE

ON_EVENT_PROCEDURE OnKeyPress(Alt+F5)
            ...
            event handler code
            ...
END_PROCEDURE
