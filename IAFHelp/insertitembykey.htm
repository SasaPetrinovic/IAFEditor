Insertitembykey
   

This method randomly inserts an item into a tree.

Format


            Insertitembykey(level, text_label, data, key)


Parameters


·     level - tree level, integer variable or literal. Allowed values are 1 to 10.
·     text_label - text to be displayed.
·     data - data to be stored into this node. When user selects the node, this value will be stored into TREEDATA internal variable.
·     key - concatenation of the all keys stored in the tree, from level 1 up to the parent level. If we try to insert an item into the first level, this parameter can be omitted, because there in no parent item.

Applied To

            Tree.

Note

            When we fill a tree initially, we use Insertitem method (Level NN, Insert actions).
            But, if we want to add an item later-on, we should use this method.

Example

  
Lets consider the following tree:

001 Item_1
002 Item_2
  01 Item_2_1
  02 Item_2_2
  03 Item_2_3
    01 Item_2_3_1
003 Item_3

To insert item [004 Item_4], we write
      tree_1.Insertitembykey(1,"Item_4","004","")

To insert a child item [04 Item_2_4], we write
      tree_1.Insertitembykey(2,"Item_2_4","04","002")

To insert a child item [02 Item_2_3_2], we write
      tree_1.Insertitembykey(3,"Item_2_3_2","02","00203")
