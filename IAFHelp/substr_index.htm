SUBSTR_INDEX
   

This function returns the position of the first occurrence of a substring.

Format

            SUBSTR_INDEX(alpha_var, sub_string, case_flag, direction_flag)

Return Value Type

            Numeric.

Parameters


·     alpha_var - input alpha variable.
·     sub_string - a substring to search for.
·     case_flag - case sensitivity flag, integer. If this parameter has value 0 (zero), the strings are case-sensitive. If you want to perform case-insensitive searching, enter code_page (1250, 1251, 1252, ...) to be used in internal uppercase conversion.
·     direction_flag - search direction flag, integer. Value 0 (zero) denotes searching from left to the right. Non-zero value specifies right to the left searching.


Notes

            If substring is empty, result is 1.

            If substring was not found, result is –1. Otherwise, result is between 1 and length of alpha_var.

            The leftmost character has position 1.

            At the start of search process, the function trims trailing spaces from alpha_var. Also, trailing spaces from substring will be trimmed, if the substring is not a literal. Therefore, searching will be performed just in non-blank part of alpha_var.
            
            Let's consider the following example: local alpha variable $A20, twenty characters long, has the value 'ABCD' (four letters and 16 spaces). Result of SUBSTR_INDEX($A20,'B',0,0) is 2. But, SUBSTR_INDEX($A20,' ',0,0) returns -1, because the first four characters do not contain a space. If the variable was 'ABC D' (three letters, space, letter), then the result of SUBSTR_INDEX($A20,' ',0,0) will be 4.

Examples

  
      $res = SUBSTR_INDEX('ABCD','B',0,0)       !! result is 2
      $res = SUBSTR_INDEX('ABCD','b',0,0)       !! result is -1
      $res = SUBSTR_INDEX('ABCD','B   ',0,0)    !! result is -1
      $res = SUBSTR_INDEX('ABCD','  B',0,0)     !! result is -1
      $res = SUBSTR_INDEX('ABCD','X',0,0)       !! result is -1
      $res = SUBSTR_INDEX('ABCD','b',1252,0)    !! result is 2
      $res = SUBSTR_INDEX('ABCD','x',1252,0)    !! result is -1
      $res = SUBSTR_INDEX('ABCDB','b',1252,1)   !! result is 5
      $res = SUBSTR_INDEX('ABCDŠ','š',1250,0)   !! result is 5
      $res = SUBSTR_INDEX('?????','?',1251,0)   !! result is 4

! The statement
      $res = SUBSTR_INDEX('ABCD','ab',1252,0)
! has the same effect as
      $a4 = 'ABCD'
      $a2 = 'ab'
      $res = SUBSTR_INDEX(upper_nls($a4,1252),upper_nls($a2,1252),0,0)
