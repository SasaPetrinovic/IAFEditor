Getline
   


This method reads one line from a grid. All parameters except the first one are output ones.

Format

            Getline(line, field1, field2, ..., fieldN )


Parameters

·     line - grid line, numeric value, integer, required parameter. The first line is 1, etc.  To specify current row, we can use the internal variable CURRENT_LINE or star (*). Also, the constant ENTIRE_GRID can be used to denote all rows in the grid.
·     field1 - content of the first column, can be omitted. If ENTIRE_GRID is specified as the first argument, array qualifier is expected, i.e. field1().
·     field2 or field2()- content of the second column, can be omitted.
·     fieldN or fieldN() - content of the the last column, can be omitted.


Applied To

            Grid.

Notes

            If we use ENTIRE_GRID constant, one column will be stored in an array. If number of rows is bigger than number of array elements, the first N rows will be read. Remain rows (from N+1 up to end of the grid) will be ignored.

Examples

  
      Table_1.Getline(current_line, counter_1, acc_1)
      Table_1.Getline(*, counter_1, acc_1)
      Table_1.Getline(current_line, counter_1,, counter_3)
      Table_1.Getline(5, counter_1,, counter_3)
      Table_1.Getline($line, counter_1,, counter_3)

!! Read entire grid
      define_local $$aa1 a 10 100   !! array, 100 elements
      define_local $$aa2 a 5  100   !! array, 100 elements
      define_local $$nn3 n 11,2 100 !! array, 100 elements
      if Table_1.Getnumberrows() <= 100
            Table_1.Getline(ENTIRE_GRID, $$aa1(), $$aa2(), $$nn3())
      end_if
