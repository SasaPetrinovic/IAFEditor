VALUE_DATE
   

This function converts a date literal to date or datetime value.

Format


            VALUE_DATE(input_date, date_format)

Return Value Type

            Date or datetime.


Parameters


·     input_date - alpha field or alpha literal containing date literal.
·     date_format -date format in input field. Possible values: 'ymd' for year-month-day, 'dmy' for day-month-year, 'mdy' for month-day-year. If this parameter is empty, expected format is ymd.

Notes

            The function converts alpha field containing date or date-time literal to date field, because alpha fields cannot be used in calculations. The function skips leading spaces. Formatting characters, like slash, minus or dot are ignored.

            If input date is invalid, result is ZERO_DATE.

Examples

  
      $datefld = value_date('   2011-12-20','ymd')
      $datefld = value_date('   2011-12-20','')
      $datefld = value_date('20/12/2011','dmy')
      $datefld = value_date('20.12.2011','dmy')
      $datefld = value_date('20-12-2011','dmy')
      $datefld = value_date('2/1/2012','dmy')   !! 2-jan-2012
      $datefld = value_date('2/1/2012','mdy')   !! 1-feb-2012
      $datefld = value_date('31/04/2011','dmy') !! Invalid day
      $dt = value_date('2011-12-22 10:11:12','ymd')
      $dt = value_date('2011-12-22T10:11:12','ymd')

      $a10 = '2014-12-11'
      $datefld = value_date($a10,'ymd')
