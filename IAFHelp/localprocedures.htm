Local procedures / functions definition
   

Local procedure(s)/function(s) are subroutines defined in a program and can be used in this program only, see Global procedure(s)/ function(s).

A procedure starts with PROCEDURE keyword and ends with END_PROCEDURE.

A function starts with FUNCTION keyword and ends with END_FUNCTION.

All the local procedures/functions are defined in the same place.
In procedure/function definition we can use DEFINE_LOCAL statement to define local fields which can be seen just from this procedure/function.

Note: if we want to call one local procedure from another local procedure, called procedure must be defined at the beginning of the buffer.

Examples:

Procedure proc1
 ...
 perform proc2    !!invalid, proc2 not defined yet
end_procedure

Procedure proc2
 ...
 perform proc1    !!ok, proc1 defined before proc2
end_procedure

Function fnc1() return_value A5
...
end_function

Function fnc2(arg1 n3) return_value N1
...
end_function
