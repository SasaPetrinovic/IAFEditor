INPUTBOX
   

This function displays prompt in a dialog box, waits for the user to input text or click a button, and returns string containing the contents of the text box.

Format

            INPUTBOX(prompt[, title] [, default] [, xpos] [, ypos] [,passlength])

Return Value Type

            Text.

Parameters

·     prompt - Text to be displayed in the dialog box.
·     title - Text to be displayed in the title bar of the dialog box.
·     default - Text to be displayed in the edit box of the dialog as a default value.
·     xpos - x position (pixels) of the dialog box.
·     ypos - y position (pixels) of the dialog box.
·     passlength - if a password to be entered, this parameter specifies maximum numbers of the characters. If this parameter is omitted, normal text will be entered (with echo).
Note

            To omit some arguments, you must include the corresponding comma delimiter.

Examples
  
      $text = inputbox("Enter your ID")
      $text = inputbox("Enter your ID",,"1")
      $passw= inputbox("Enter password",,,,,8)
