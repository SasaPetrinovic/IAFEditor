Timer properties
   

Object name

When the new timer is created, generator automatically assigns the name Timer_nnn, where nnn = 1, 2, …
Object name can be maximum 30 characters long. Allowed characters are A-Z, 0-9, _ and $.
First character must be a letter. Object name is case-insensitive. Object name can be used in Methods.


Timer interval

When this time interval elapses, the code (procedure) defined by 'Timer actions' will be executed. Time interval unit is millisecond. For instance, if you want to specify 30 seconds, 30000 should be entered. This parameter can be changed by Settimerinterval method.

Timer activity

  
Program
The timer will be created at the start of the program and remains active disregarding which subform is currently active. Such timers should be created on the first subform.
Active subform
The timer of this type will be active only if its subform has focus. When the subform loses the focus, the timer stops.


Automatic start at runtime

Determines if the timer will be started after its creation. If the value is FALSE, then the Start method must be used somewhere in the program to start this timer.
