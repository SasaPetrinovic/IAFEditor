Getposition
   

This method returns position and size of the object.

Format

            Getposition()

Return Value

            Alpha field containing values in the form: Xpos;Ypos;Width;Height (the delimiter is semicolon).

            If object does not exist, function returns blank value.

Example

  
      $alphares = table_1.Getposition()
      if $alphares <> space
            $arr() = split(';',$alphares)
            $xpos  = value($arr(1))
            $ypos  = value($arr(2))
            $width = value($arr(3))
            $height= value($arr(4))
      end_if
