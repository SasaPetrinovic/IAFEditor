Run
   

This method saves the document, breaks connection between program and MS Word, and gives control to the user. After this, another document can be opened, etc.

Format

            Run([command [,output_filename]])

Parameters


·     command - this parameter, if used, is alpha literal or alpha variable containing one of the following values:
                        "print" - prints document on default printer,
                        "print N" - prints N copies of document on default printer, N < 10.
                        "save" - invokes 'Save as',
                        "save_exit" - invokes 'Save as' and stops Word,
                        "save_print" - invokes 'Save as', prints document on default printer and stops Word,
                        "save_print N" - the same as above + copies, N < 10,
                        "mailmerge" - invokes 'Mailmerge'
                        "mailmerge_save" - saves documents, Word is active
                        "mailmerge_exit" - invokes 'Mailmerge', saves document and stops Word,
                        "mailmerge_print" - invokes 'Mailmerge', saves and prints document and stops Word.

·     output_filename - an alpha variable or alpha literal. Specifies name of the output file which will be used in 'Save as'. Logical names in filename are allowed. If command has some of 'save' or 'mailmerge' options, this parameter is required.

Return Value

            Status, numeric variable.

            Possible values:

            0          Ok
            -1         Document not opened
            -2         Invalid command
            
Applied To

            Word object.

Note

            This method is usually used to save results.
            If we want that Word stays active after Run,  the Close method need not be called.

Examples
  
            Variants

            Run()               !! Shows Word without 'Save as'
            Run("print")     !! Prints document on default printer
            Run("print 3")  !! Prints 3 copies on default printer

            $pcmd = "Print "|edit($numcop,"9")
            Run($pcmd)     !! number of copies is variable

            Run("save",filename)  !! Invokes 'Save as', shows Word
            Run("save_exit",filename)       !! Invokes 'Save as' and exits Word
            Run("save_print",filename)     !! Invokes 'Save as', prints document and exits Word

      $stat = Word.Run()
      $stat = Word.Run("save","c:\doc\out.doc")
      $stat = Word.Run("save_exit","docdir:test.doc")
