NUM
   

This function converts alpha field to numeric field. Also, this function can be used to cast date, time and datetime fields to numeric.

Format


            NUM(value_exp)

Return Value Type

            Numeric.


Parameters


·     value_exp - a value expression: alpha, date, time, datetime or numeric.

Note

            If 'value_exp' is an alpha field, it must have all numeric characters (digits).

            The NUM function is a redefinition: if we know that an alpha variable contains all digits, then with NUM function we declare that variable as numeric. This is why we cannot use this function with partially numeric field.
            For instance, if $a6 is an alpha field whose length is 6 characters and contains '123', we cannot use NUM($a6). Instead, VALUE function should be called.

Examples

  
      if $a8 is numeric
            $num_field = num($a8)
      end_if
!
      if $a8 is numeric
            $res = num($a8) + 5
      else
            $res = 5
      end_if
!
      $num_field = num(today)
      $num_field = num(timestamp)
