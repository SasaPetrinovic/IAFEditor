Subform calculations

This command activates Subform calculations editor. Subform calculations execute after  last active field on the screen is processed, or when program executes FINISH_WINDOW statement.

Help screens

Sequence

Sequence definition for active objects.

Default sequence (resequence)


This command sets default sequence for active objects. All active objects (except those with sequence = 0), are sorted from left to the right and from up to down. The first object obtains the sequence number 1, next one number 2, etc.


Toolbar

A toolbar is control bar containing buttons and separators. The toolbar is usually located under program caption. Click on toolbar buttons calls often used procedures or programs.
This command activates Define toolbar dialog to define toolbar buttons, button names, tooltip text.

Toolbar code

For each toolbar button, we need to define appropriate action. This is a procedure which executes when user clicks a button at run-time.
