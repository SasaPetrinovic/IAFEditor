IF
   

Conditional execution of block of the statements.

Format

            IF cond_exp
                        …
            [ELSE_IF cond_exp
                        …]
            [ELSE_IF cond_exp
                        …]
            [ELSE
                        …]
            END_IF

Where


·     cond_exp - conditional expression.


Note

            ELSE_IF and ELSE parts are optional, END_IF keyword is required.

Examples

  
      IF acc_1 = 1
            ADD 1 TO counter_1
      ELSE_IF acc_1 = 2
            ADD 2 TO counter_1
      ELSE_IF acc_1 = 3
            ADD 3 TO counter_1
      ELSE
            ADD 100 to counter_1
      END_IF

      IF acc_1 = 1
            add 1 to acc_2
      END_IF
