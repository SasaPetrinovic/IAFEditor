Edit mask
   

Edit mask is a string of characters that determines visual representation of a field on an external device (screen, printer, print file).

For destination fields, edit string has control function as well: defines number of digits in a numeric field, allowed characters in an alpha field, etc.

Edit mask for source fields

Alpha field

Editing characters in edit mask for an alpha field are: X, 9, N, U, L. Each editing character is just placeholder for one character from input field. All editing characters have the same meaning: character placeholder. An edit mask can contain and other characters. These characters we consider as formatting characters and they will be moved directly to the output buffer.

Date field

Editing characters for a date field are: D, M, Y, W, R and Z. All other characters in the mask are formatting characters.

Display  number of months, two digits, suppress leading zero (1, 2, ..., 12)

Display month name. Number of Ms determines month name length (JAN, JANUARY, ...)

Display month in Roman numerals (I, II, III, ..., XII)

YY
Display two-digits year (00, 01, ..., 99)

Displays long date format, based on selected locale (look in Regional and language options, Regional options tab).

Edit mask for destination fields

Alpha field

Number of characters in the edit mask determines maximum number of input characters. To override this option, see Destination field properties (Ignore edit mask attribute).
Editing characters for the alpha field are: X, 9, N, U, L.

  
Editing character
Meaning
 
 
X
any character allowed at this position
9
only digits allowed at this position
N
digits or space
U
make uppercase conversion
L
make lowercase conversion

Editing characters for the date field are: D, M, Y. All other characters in the mask are formatting characters.
