Append
   

Appends a Word file or Excel worksheet (or a part of the worksheet) to the current Word document.


Format

            Append(filename)

Parameter


·     filename - an alpha variable or alpha literal. Specifies name of the existing Word or Excel document. Logical names in filename are allowed.

            If we append Excel worksheet, filename format is

                        file_name [, worksheet_number [, range]]

            where is:
                        file_name - physical or logical file name,
                        worksheet_number - number of worksheet,
                        range - in the form start_cell : end_cell

            Default values: worksheet_number = 1, range of cells = whole sheet.

            For example

                        "c:\temp\a.xls"
                        "c:\temp\a.xls, 3"
                        "c:\temp\a.xls, 3, A1:D10"
                        "excel_dir:a.xls, 3, A1:D10"

Return Value


            Status, numeric variable.

            Possible values:

            0          Ok
            -3         File not found

            
Applied To

            Word object.

Examples
  
      $stat = Word.Append("c:\temp\a.doc")
      $stat = Word.Append("docdir:a.doc")
      $stat = Word.Append("c:\temp\a.xls")
      $stat = Word.Append("c:\temp\a.xls, 3")
      $stat = Word.Append("c:\temp\a.xls, 3, A1: F20")
 

Using Word
