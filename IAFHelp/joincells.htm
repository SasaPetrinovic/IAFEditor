Joincells
   

This method combines many adjacent cells into one large cell.

Format

            Joincells(line1, line2, column1, column2)

Parameters


·     line1 - starting line, a numeric variable or numeric literal.
·     line2 - ending line, a numeric variable or numeric literal.
·     column1 - starting column, a numeric variable or numeric literal.
·     column2 - ending column, a numeric variable or numeric literal.


Applied To

            Display-only grid.

Note

            To display a text into large cell, use (line1, column1) as parameters.


Example

  
      Table_1.Joincells(1,1,5,10)   !! the first line, columns 5 thru 10
      Table_1.Display(1,5,'New large text')
