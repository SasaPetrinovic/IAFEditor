MAKETIME
   

This function returns time value calculated from the hour, minute and second.

Format

            MAKETIME(hour,minute,second)

Return Value Type

            Time.

Parameters

·    hour - numeric_exp (from 0 to 23)
·    minute - numeric_exp (from 0 to 59)
·    second - numeric_exp (from 0 to 59)
Note

            If arguments are invalid, result is zero_time.

Examples
  
      $time6 = maketime(10,1,50) !! result is "10:01:50"
      $time6 = maketime(25,0,1) !! result is zero_time
