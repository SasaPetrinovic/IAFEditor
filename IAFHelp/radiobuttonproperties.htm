Radiobutton properties
   

Object name

When the new radiobutton is created, generator automatically assigns the name Radiobutton_nnn, where nnn = 1, 2, …
Object name can be maximum 30 characters long. Allowed characters are A-Z, 0-9, _ and $.
First character must be a letter. Object name is case-insensitive. Object name can be used in Methods.

Member of group (information only, cannot be changed)

If the object is a member of a group, the name of this group is displayed. An object can be the member of one group only.

Caption

The text which appears on the button. If you want to define one letter as a prefix (mnemonic) character, precede it with ampersand (&). At run-time, that letter will be underlined and user can click that button by pressing Alt + letter.
If you want to display single ampersand in the caption, enter two consecutive ampersands.

Within one subform, all prefix characters must be different.

Visible

Determines whether the radiobutton is visible when the subform is first displayed.

Enabled

Determines whether the radiobutton is displayed as disabled when the subform is first displayed. When the radiobutton is disabled, mouse clicks are ignored.

Background style

Background style should be set to normal.

Background color

This attribute is applied only if the background style is normal. Double click brings up a color definition dialog.

Special effects

Values:

Flat
Raised
Sunken
Etched
Bump
Shadowed

Border style

Values:

Transparent      border is not seen,
Solid                 the border is standard line,
Dots                 the border is dotted line,
Dashes             the border is dashed line.

Border color

Is applied only if border style is not transparent.  Double click brings up a color definition dialog.

Border width

Is applied only if border style is not transparent. Possibly values are 0, 1, 2,... pixels.

Text color

Color of text. Double click brings up a color definition dialog.

Font name

Choose one from offered font names.

Font size

Specifies font size of the text.

Bold

Specify that font weight is 700. Normal weight is 400.

Italic

Specify whether text is italic.

Underline

Specify whether text is underlined.

Sequence number

Radiobutton cannot have sequence number 0.
All radio buttons in one radiobutton group must have consecutive sequence numbers.

The radiobutton group is a group of buttons surrounded by the Frame.

Tooltip text

Tooltip text is a text that will be displayed when the cursor hovers above the button. See Tooltip text definition.
