Applications
   


Rebuild programs

Generates one or more executable programs. This option gives us possibility to generate all the executable programs in an application.

Rebuild Record Descriptions

Generates records (tables) descriptions and stores them into text files. These files are used later by C compiler to generate an executable program. File extension is CLB.

Show

Prints program documentation and shows source code of global procedures.

Undelete Programs

When we delete a program in generator, the program will be just marked as removed. Physically, it stays in the Data Dictionary. This option gives us chance to restore such programs.

Note: function 'Compress DD', removes these programs from the DD. After that, we can not undelete any removed program.
