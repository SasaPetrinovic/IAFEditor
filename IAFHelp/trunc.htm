TRUNC
   

This function truncates a numeric value.

Format

            TRUNC(numeric_exp, numdec)

Return Value Type

            Numeric.

Parameters


·     numeric_exp - numeric value to be truncated.
·     numdec - number of resulting decimals, numeric.
·     If numdec is greater than zero, then it represents number of decimals.
·     If numdec is 0, input number will be truncated to integer.
·     if numdec is less than zero, then the input number is truncated to the left of the decimal point.


Note

            Numdec is defined as follows:

  
Numeric value
1
1
8
2,
5
6
Numdec
-3
-2
-1
0
1

Examples

  
      acc_31 = 157,567
      $C = TRUNC(acc_31,2) !! result is 157,56
      $C = TRUNC(acc_31,1) !! result is 157,5
      $C = TRUNC(acc_31,0) !! result is 157
      $C = TRUNC(acc_31,-2) !! result is 100

      acc_31 = -157,567
      $C = TRUNC(acc_31,2) !! result is -157,56
      $C = TRUNC(acc_31,1) !! result is -157,5
      $C = TRUNC(acc_31,0) !! result is -157
      $C = TRUNC(acc_31,-2) !! result is -100

      acc_31 = 9,999
      $C = TRUNC(acc_31,2) !! result is 9,99
      $C = TRUNC(acc_31,1) !! result is 9,9
      $C = TRUNC(acc_31,0) !! result is 9
