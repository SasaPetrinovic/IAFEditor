UPLOAD
   

This function sends HTTP POST request to an Internet site.

Format

            Upload (URL, message [,headers, username, password])

Parameters


·     URL - universal resource locator, like http://mysite/files/test.php, etc.

·     message - a CLOB variable (field type = S). Text to be sent via POST request.

·    headers - additional headers to be sent to the HTTP server, optional. Two or more header fields should be separated by CRLF (carriage-return/line feed pair).


·     username - an alpha variable or alpha literal, optional.

                  If username is not an empty string, the function generates HTTP header field

                              Authorization: Basic username:password

·     password - an alpha variable or alpha literal, optional.


Return Value

            CLOB variable.

            Possible values:

                        -1         Invalid (empty) message.
                        -2         Invalid URL.
                        -3         Cannot establish connection
                        -4         Cannot open http request
                        -5         Upload file not found

                        other     Server response

Notes


·           HTTP request header generated by this function is:

            POST target_object HTTP/1.1
            Host: hostname[:port]
            Content-Type: application/x-www-form-urlencoded
            Content-Length: nnn
            
            If username and password supplied, additional header line is added:

            Authorization: Basic ...

·           To send JSON text, use additional header:

                        message = '{ "fld1": "a_value1", "fld2": "a_value2", "fld3": 100.00}'

                        $sres = UPLOAD(url,message,'Content-Type: application/json')

                        HTTP request header generated by this function is:

                  POST target_object HTTP/1.1
                  Host: hostname[:port]
                  Content-Type: application/json
                  Content-Length: nnn


·           To send two additional headers:

                        $message = '{ "fld1": "a_value1", "fld2": "a_value2", "fld3": 100.00}'
                        $hdr = compress_trim(''Content-Type: application/json',crlf,'Authorization: Bearer xyz')
                        $sres = UPLOAD($url, $message, $hdr)

                        HTTP request header generated by this function is:

                  POST target_object HTTP/1.1
                  Host: hostname[:port]
                  Content-Type: application/json
                  Authorization: Bearer xyz
                  Content-Length: nnn


·           How to sent HTML form data?

            Let's consider html script

            <form method="POST" action="/home/form.php">
                  Field 1:
                  <input type="text" name="id1" /><br />
                  Field 2:
                  <input type="text" name="id2" /><br />
                  <input type="submit" value="Submit" />
            </form>

            1/ Parameter message could be formatted in this way:

                        message = 'id1=First+field&id2=Second+field'

            Function call is: UPLOAD('http://www.mysite.com/home/form.php',message)

            Generated HTTP request header is:

            POST target_object HTTP/1.1
            Host: hostname[:port]
            Content-Type: application/x-www-form-urlencoded
            Content-Length: nnn

            2/ Parameter message could be formatted in this way

                        message = 'FORM-DATA;TEXT=First field;NAME=id1;TEXT=Second field;NAME=id2'

            where
                        FORM-DATA    = fixed text, required.
                        TEXT                = form field value.
                        NAME              = form field name. Pair (TEXT, NAME) is required.

            Function call is: UPLOAD('http://www.mysite.com/home/form.php',message)

            Generated HTTP request header is:

            POST /home/upload.php HTTP/1.1
            Host: hostname[:port]
            Content-Type: multipart/form-data; boundary=xxxxxxxxxxxx
            Content-Length: nnn


·           How to upload a file?

            Parameter message must be formatted in this way:

                        message = 'FORM-DATA;FILE=absolute_path;NAME=fieldname'

            where is: absolute_path is name of file to be uploaded, fieldname is name of form field.

            If html script is

            <html>
            <head><title>File Upload</title></head>
            <body>
            <form method="POST" action="/home/upload.php">
                  File name:
                  <input type="file" name="fileid" /><br />
                  <input type="submit" value="Submit" />
              </form>
            </body>
            </html>
            
            message could be

                        message = 'FORM-DATA;FILE=c:\temp\myfile.zip;NAME=fileid'

            Function call is: UPLOAD('http://www.mysite.com/home/upload.php',message)

            Generated HTTP request header is:

            POST /home/upload.php HTTP/1.1
            Host: hostname[:port]
            Content-Type: multipart/form-data; boundary=xxxxxxxxxxxx
            Content-Length: nnn

            This function should be used to upload relatively small files.

Examples

  
      $clob_out = 'FORM-DATA;FILE=c:\temp\myfile.zip;NAME=fileid'
      $clob_res = Upload('http://mysite.com/home/upload.php',$clob_out)
      $clob_res = Upload('http://mysite.com/test.php',$clob_out,,'user','mypassw')
