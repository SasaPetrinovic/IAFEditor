ROUND
   

This function rounds a numeric value.

Format

            ROUND(numeric_exp, numdec)

Return Value Type

            Numeric.

Parameters


·     numeric_exp - numeric value to be rounded.
·     numdec - number of resulting decimals, numeric.
·     If numdec is greater than zero, then it represents number of decimals.
·     If numdec is 0, input number will be rounded to integer.
·     if numdec is less than zero, then the input number is rounded to the left of the decimal point.


Note

            Numdec is defined as follows:

  
Numeric value
1
1
8
2,
5
6
Numdec
-3
-2
-1
0
1
2


            Algorithm:

            If the digit at round position has value less than 5, it will be rounded to 0. All digits
            right from this position will be set to 0. All digits left from this position stay
            unchanged.

            If the digit at round position has value greater or equal 5, it will be rounded to 0,
            but first digit left from this position will be incremented by 1.

Examples

  
      acc_31 = 157,567
      $C = ROUND(acc_31,2) !! result is 157,57
      $C = ROUND(acc_31,1) !! result is 157,6
      $C = ROUND(acc_31,0) !! result is 158
      $C = ROUND(acc_31,-2) !! result is 200

      acc_31 = 157,127
      $C = ROUND(acc_31,2) !! result is 157,13
      $C = ROUND(acc_31,1) !! result is 157,1
      $C = ROUND(acc_31,0) !! result is 157
      $C = ROUND(acc_31,-2) !! result is 200

      acc_31 = -157,567
      $C = ROUND(acc_31,2) !! result is -157,57
      $C = ROUND(acc_31,1) !! result is -157,6
      $C = ROUND(acc_31,0) !! result is -158
      $C = ROUND(acc_31,-2) !! result is -200

      acc_31 = 9,999
      $C = ROUND(acc_31,2) !! result is 10,00
      $C = ROUND(acc_31,1) !! result is 10,0
      $C = ROUND(acc_31,0) !! result is 10
