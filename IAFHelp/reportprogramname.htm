Program name
   

This is the first dialog of program generator. In this dialog, we choose program name and an operation which will be performed (modify, delete, rename, ...).

Application

This combobox shows all applications in Data Dictionary. Current application is displayed in the edit part of the combobox. If DD has one application only, combobox is disabled, else current application can be changed.

Program name

If we click list of programs, this edit box displays selected program. After that, we can click any of enabled buttons (modify, delete, generate, ...).

If we enter program name (with possible wildcards) and press Enter, then this string becomes the filter for the list. Allowed wildcards characters are: * and ?.

Examples:
  
*
 All programs
A*
 Program names starting with letter A
*A
 Program names ending with letter A
*PRG*
 Program names containing PRG
PRG?
 Program names exactly four characters long, starting with PRG


List

List on the right-side of the dialog shows all interactive programs in this application. Naturally, when we enter this generator for the first time, this list will be empty.

Program type options

            Defines type of programs to be displayed in the list.

            Reports
Displays reports only.
            Batch
Displays batch programs only.
            Report & batch
Display all non-interactive programs (reports and batch).

List options

            Names
Displays program name, last version number, user and modification date.
            Creation and modification dates
Displays program name, creation date and modification date.
            Descriptions
Displays program name and program description.
            Versions
Displays all versions of a program. After program modifying, new version of the program will be stored in DD. New program gets version number 1, next modification has version number 2, etc. When we execute compressing of the DD, all previous versions will be removed and only last version stays in the DD.
 
            Programs modified since
Displays all programs modified since requested date.

New

To create new program, click New. This action takes us to Program type dialog.

Modify

To modify an existing program, select program name in the list and click Modify

-or-

Manually enter program name in the edit box and click Modify.


Delete

To delete a program, select program name in the list and click Delete

-or-

Manually enter program name in the edit box and click Delete.

If we want to delete more than one program, manually enter program name with wildcards
and click Delete. All the programs (*) cannot be removed.

The programs, removed by this operation, can be undeleted - see Undelete programs.
This possibility exists until the next DD compressing. The compress permanently removes old program versions and deleted programs from DD.


Run

We can run any previously generated program.

Select program name in the list and click Run

-or-

Manually enter program name in the edit box and click Run.

If the file program_name.exe does not exist in the current directory,  File dialog appears on the screen and we can choose any executable program.


Generate

To generate executable program, select program name in the list and click Generate

-or-

Manually enter program name in the edit box and click Generate.

Iaf generates appropriate C++ (or C) code, activates C++ (or C) compiler and linker. At the end of this process, if successful, the file program_name.exe will be created, and we can execute it.

Copy

In real-life programming situations, often happens that two or more programs are alike.
We make one program, copy it to another program (clone) and then make changes to new program.

To do that, select program name in the list and click Copy

-or-

Manually enter program name in the edit box and click Copy.

The copy dialog appears. Enter new program name and click OK. The generator copies old program to the new one and directly activates the painter. This new program will be saved to the DD choosing menu command File/Exit and Save. If we choose the Quit command, new program will not be saved.

This is just one of possibilities to include a program in another one. The other ways will be seen later on.

Rename

To rename a program, select program name in the list and click Rename

-or-

Manually enter program name in the edit box and click Rename.


Unlock

When we open a program for modification, Iaf locks this program thus preventing the other programmers to access it. If PC or generator crashes before the normal exit, this program stays locked. To unlock the program, select it and click Unlock.


Interactive generator

Activates Interactive program generator.

Help

Displays this text.

Close

Closes the dialog and returns to the main program.
