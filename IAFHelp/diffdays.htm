DIFF_DAYS
   

This function returns number of days between two date values.

Format

            DIFF_DAYS(date1, date2)

Result Value Type

            Numeric.

Parameters


·     date1, date2 - date or datetime fields.

Note

            Result is a positive number.

Examples

  
      $n = DIFF_DAYS("1-mar-2009","1-mar-2009") !! result is 0
      $n = DIFF_DAYS("2-mar-2009”,”1-mar-2009") !! result is 1
      $n = DIFF_DAYS("1-mar-2009”,”2-mar-2009") !! result is 1
