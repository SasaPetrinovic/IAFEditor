DO_WHILE
   

This statement marks beginning of a cycle.

Format

            DO_WHILE cond_exp
                        ...
            statement(s)
                        ...
            END_DO


Where


·     cond_exp - conditional expression. While its value is TRUE, the cycle is performing. When program comes to DO_WHILE, if conditional expression is FALSE, whole block of statements up to END_DO will be skipped.

Note

            DO_WHILE cycle must end with END_DO statement.


Example

  
      $tot = 0
      counter_1 = 0
      do_while counter_1 < 10
            add 1 to counter_1
            add $arr(counter_1) to $tot
      end_do
