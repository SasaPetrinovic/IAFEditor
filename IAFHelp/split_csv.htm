SPLIT_CSV
   

This function splits an alpha field containing csv string into alpha array.

Format

            SPLIT_CSV(separator,text_qualifier,code_page,alpha_field)

Return Value Type

            Alpha array.

Parameters

·     separator - alpha field, length = 1. Character that separates csv fields.
·     text_qualifier - integer denoting the character that encloses csv fields. Allowed values: 0 - no qualifier, 1 - quotation mark, 2 - apostrophe. All of the text between qualifiers is treated as one field, even if the field contains separator character. Two consecutive text qualifiers in the field are treated as one character.
·     code_page - integer. If output array is an utf-16 array, this parameter will be used in conversion. Value 0 denotes current code page. UTF-8 code page is 65001. See code page.
·     alpha_field - input alpha field.


Note

            The function can be used in an assignment only. Syntax is:

                        array() = SPLIT_CSV(separator,tq,cp,alpha_field)

Examples
  

If the input field $INP contains

      100;200;"text1";"text2""ext";text3;text4

and $arrw is an alpha array, 10 utf-16 elements, the following statement

$arrw() = SPLIT_CSV(';',1,0,$inp)

gives the result

$arrw(1) = '100'
$arrw(2) = '200'
$arrw(3) = 'text1'
$arrw(4) = 'text2"ext'
$arrw(5) = 'text3'
$arrw(6) = 'text4'
$arrw(7) = ... = space
