SUM
   

This function returns sum of elements of a numeric array.

Format

            SUM (array[,start_index, end_index])

Return Value Type

            Numeric.

Parameters

·     array - numeric array.
·     start_index - integer, starting index (1, 2, ...).
·     end_index - integer, ending index >= starting index.

Examples
  
      acc_1 = sum($arr1) !! sum of all elements
      acc_1 = sum($arr1,1,12) !! sum of the first 12 elements
      acc_1 = sum($arr1,1,counter_1)
