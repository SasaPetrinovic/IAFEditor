Setparttextcolor
   


This method sets the color of the part of text that matches the given pattern.

Format 1
  
 Setparttextcolor (pattern, bkColor, txColor)
 Sets color of an object
Format 2
  
 Setparttextcolor (row, column, pattern, bkColor, txColor)
 Sets color of a grid cell

Parameters



·     pattern - text, alpha literal or alpha variable, against which a field is compared. The comparison is case insensitive. If a database field, displaying in a sourcefield object, is an ANSI field, the pattern should be ANSI one, too. If the database field is Unicode utf-16, the pattern should be in utf-16 format.

·     bkcolor - background color of the pattern, a numeric variable with valid RGB color value or one of the predefined colors or RGB function. Optional.

·     txcolor - text color of the pattern, a numeric variable with valid RGB color value or one of the predefined colors or RGB function. Optional.

·     row - grid row, numeric value. Rows are counted from 1.To specify current row, we can use the internal variable CURRENT_LINE or star (*). Also, the constant ENTIRE_COLUMN can be used to denote all rows in a column.

·     column - grid column, numeric value. The columns are counted from 1.

Applied To

            Source field, single line.

Note

            The most efficient way to use this method is in 'Table starting actions' of a grid:

                        grid_x.Setparttextcolor(ENTIRE_COLUMN, col_n,pattern,bkColor,txColor)
or
                        ME.Setparttextcolor(ENTIRE_COLUMN, col_n,pattern,bkColor,txColor)


Examples

  
      src_1.SetPartTextColor('abc',,'blue')
! If the field value is 'Text abcdef', the result will be: Text abcdef.

      src_1.SetPartTextColor('abc','yellow','blue')
      src_1.SetPartTextColor('abc',rgb(64,64,64),'blue')

      grid_1.SetPartTextColor(current_line,1,'abc',rgb(64,64,64),'blue')

! In 'Table starting actions'
      grid_1.SetPartTextColor(entire_column,1,'abc',,'blue')
