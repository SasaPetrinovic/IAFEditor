Runmacro
   

This method runs a MS Word macro and, optionally, stores VBA (Visual Basic for Applications) code to Word document.


Format

            Runmacro(macro_name [,VBA_filename][,arg1])

Parameters


·     macro_name - name of a macro, an alpha variable or alpha literal. The macro must be stored in file IAFMACRO.DOT. If macro_name is prefixed by DOCUMENT:, a macro from current document will be run. Prefixed form is "DOCUMENT:macro_name".
·     VBA_filename - name of text file containing VBA procedure(s), alpha variable or alpha literal.
·     arg1 - value of the first argument, alpha variable or alpha literal.


Return Value

            Status, numeric variable.

            Possible values:

            0          Ok
            -1         Document not opened
            -2         Macro does not exist

Applied To

            Word object.

Examples

  
      $stat = Word.Runmacro('macro1')           !! macro from IAFDIR:IAFMACRO.DOT
      $stat = Word.Runmacro('DOCUMENT:macro1')  !! macro from current document

      $macro = 'my_macro'
      $stat = Word.Runmacro($macro)

! Run VBA procedure declared as 'Sub Mysub(arg1 as String)'
      $stat = Word.Runmacro('Mysub',,'abcd')

! Store VBA code from c:\temp\vbacode.txt into open document
! and, after that, execute myproc1.
      $stat = Word.Runmacro('myproc1','c:\temp\vbacode.txt')
