Define global procedures / functions
   

This dialog enables us to view all global procedures/functions and to select a procedure/function for modification or deleting.

Show all

Show all global procedures and functions.

Show procedures only

Show all global procedures.

Show functions only

Show all global functions.

Show versions

Show all versions of procedures/functions. As we know, each Save command creates new version of a procedure/function. After DD compressing, all the previous versions will be purged.

Modify

Modify selected procedure/function.

New procedure

Create new global procedure.

New function

Create new global function.

Delete

Delete selected global procedure/function. After deletion, all the versions are removed from the DD. Just one particular previous version of procedure/function cannot be deleted.

Search

Searches global procedures/functions according to search criteria, look Search global procedures/functions.
