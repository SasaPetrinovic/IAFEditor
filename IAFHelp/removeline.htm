Removeline
   


This method removes one line from a grid.

Format

            Removeline(line )

Parameters


·     line - line number, integer

Applied To

            Grid.

Note

            If you work with data-entry grid, and you enter data into a table or file, you have to delete appropriate record explicitly. Removeline method just visually removes the line from the grid and adjusts the internal key table.


            As you know, there is a built-in action 'line deleting'. To interactively remove line from a data-entry grid, the following procedure must be performed: click side heading to select whole line (only current line can be selected). Press 'Delete' key. Message box appears. Click 'Yes'.
            This built-in action deletes current record, throws out record key from the internal key table, removes current line from the grid and calls 'Line deleting' procedure.

Examples

  
      Table_1.removeline(counter_1)
      Table_1.removeline(2)
      Table_1.removeline(current_line)
