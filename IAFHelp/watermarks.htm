Watermarks printing
   
A watermark is text placed behind report objects.

To add a watermark to the report

            1. Choose menu command Define/Watermark printing from main menu.

            2. In the Enable watermark printing dialog, select a watermark from the list.

            3. Check 'Enable printing watermark in [Report N]'.

            4. Choose 'Print on' option.

            5. Eventually check 'Show watermark in the painter'.

            6. Click OK.

To disable enabled watermark

            1. Choose menu command Define/Watermark printing from main menu.

            2. In the Enable watermark printing dialog, uncheck 'Enable printing watermark in [Report N]'.

            3. Click OK.

To create new watermark

            In the Enable watermark printing dialog, click 'New'. Watermarks are stored in the Data Dictionary.


The watermark, if enabled, implicitly creates object named WATERMARK with two methods:

            WATERMARK.Setvisible(0) - sets watermark to the invisible state.
            WATERMARK.Display(some_text) - changes watermark text at run-time.

If report type is Report, each report can have different watermark. Also, if report type is Form then each page can have different watermark. Object names are WATERMARK_1 for page 1, WATERMARK_2 for page 2, etc.

Commands to suppress watermark on page 2 (or page 10) are:

            WATERMARK_2.Setvisible(0)
            WATERMARK_10.Setvisible(0)

If there is only one report (page) in a program, object WATERMARK should be used. This object is synonym for WATERMARK_1.

These methods should be invoked before program cycle (usually in PROGRAM TITLE).
