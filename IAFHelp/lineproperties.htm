Line properties
   

Object name

When the new line is created, generator automatically assigns the name Line_nnn, where nnn = 1, 2, …
Object name can be maximum 30 characters long. Allowed characters are A-Z, 0-9, _ and $.
First character must be a letter. Object name is case-insensitive. Object name can be used in Methods.

Member of group (information only, cannot be changed)

If the object is a member of a group, the name of this group is displayed. An object can be the member of one group only.

Visible

Determines whether the object is visible when the subform is first displayed.

Line style

  
Transparent
Line is invisible
Solid
Line is standard, solid, line
Dots
Dotted line
Dashes
Dashed line

Line color

Line color.

Line width

Possible values: 0, 1, 2, ... pixels.

Special effects

If line width >= 2 and y_size = 0, or x_size = 0, ie. line is horizontal or vertical one, you can choose one of the following:

  
Flat
Standard line
Raised
Line with raised effects
Sunken
Line with sunken effects

X position, Y position

Change line position on the screen.

X size, Y size

Change line size.

If x_size = 0, the line is vertical; if y_size = 0, the line is horizontal one.

Note: this job, the position or size changing, can be done by using movement / resizing commands, but this is the easiest way to make a line horizontal or vertical.

Z-Order

Possible values: 0, -1, -2, -3, ... See Z-Order.
