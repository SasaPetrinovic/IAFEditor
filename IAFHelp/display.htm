Display
   

This method displays value expression.

Format 1

            Display (val_exp)

Format 2

            Display (line, column, val_exp)


Parameters


·    val_exp - a value expression to be displayed
·    line - cell row, integer (format 2). The first line is 1. Top heading has line = -1.
·    column - cell column, integer (format 2). The first column is 1.

Applied To

            Format 1 applies to an ordinary object.
            Format 2 applies to a grid cell.

Notes

            1/
            The object must have 'Background style = Normal'.

            2/
            Numeric, date and time fields will be edited before displaying.

            3/
            Combo box, check box, radiobutton : val_exp denotes value, not a text. To change checkbox or radiobutton label, use Displaylabel method.

            4/
            Radio button: this method should be applied to the first radiobutton in the group (the button with the smallest sequence in the group of the buttons). A group of radio buttons is defined implicitly by the screen painter: all radio buttons within one frame belong to one group. This frame can be invisible at run-time, but must exist in the painter because it is the only way to determine group of radio buttons.

            5/
            FORM: the method changes program caption (title).

            6/
            Grid, graph with sequence = 0: to display such objects, use format Objectname.Display() or Objectname.Display("").

            7/
            To display a field from a report to the program title, use PROGRAM_TITLE object: Program_title.Display("xyz").

Examples

  
      sourcefield_1.display("some text")
      sourcefield_1.display("text = " | $msg)
      sourcefield_1.display(Compress("text = ",$msg))
! Run-time engine uses edit-mask of sourcefield_2 to edit acc_1
      Sourcefield_2.Display(acc_1)

            table_1.display(current_line,1,"test")
      table_1.display(*,1,"test")   !!the same as above
      table_1.display(-1,1,"top hdg text")      !! top heading
      checkbox_1.display("1")

      program_title.display("a text")
      status_bar.display("xxx")

      form.Display("New caption")
