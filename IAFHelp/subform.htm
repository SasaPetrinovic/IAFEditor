Subform
   

A subform is complex object containing many related objects.These objects make an unit within main form, usually displaying/manipulating related set of data. Any program can have up to 20 subforms.

First of all, define all the objects which belong to the subform. After that, click button Subform in toolbox or choose menu command Objects/Subform. Move the cursor to the desired position, then click and select (rubber-band) rectangular area to surround all previously defined objects. Definition dialog appears on the screen.
