Statements
   

ADD adds numeric_exp to a numeric field.
BREAK exits immediately innermost loop.
BREAK_FIELD defines terms for group actions in batch programs.
CALL calls function  written in C programming language.
CALL_PROGRAM calls an external executable program (EXE).
CALL_STOREDPROCEDURE calls a stored procedure (ODBC).
CALL_WINDOW calls a window (subform).
CHAIN calls another program or procedure.
CHECK controls entered value.
CHECK_RECORD/CHECK_NORECORD controls (non)existing record.
CLOSE closes file.
COMMIT confirms transaction.
DEFINE_LOCAL defines local field in a procedure.
DEFINE_RECAP_RECORD defines recapitulation record in reports.
DELETE_RECORD removes an existing record from an indexed file or RDB table.
DESTINATION defines data entry field.
DETAIL_CALC specifies part  of a batch program which is executed.
DISPLAY_WINDOW_FIELDS displays fields on the screen.
DO_WHILE marks beginning of a cycle.
END_ ends block of instructions.
ERRMSG displays error message if the CHECK expression value is FALSE.
EXECSQL executes a SQL statement.
EXECUTE executes operating system commands.
FINISH_WINDOW
FUNCTION  is a procedure which has arguments and returns a value.
IF - conditional execution of block of the statements.
INIT initializes field values (ZERO, SPACE, ZERO_DATE, ZERO_TIME).
INIT_RECORD fills all fields in record wih initial values.
MAIL sends a mail.
OPEN opens a file.
PAUSE pauses a program.
PERFORM executes local or global procedure.
PROCEDURE represents a beginning of a procedure.
READ reads one record.
READ_LE reads a record from an indexed file (table).
READ_NEXT reads next record from a file or RDB table in a sequence.
READ_ALL reads record groups from a file or RDB table in a cycle.
READ_EXTERNAL reads global symbol.
RESET closes, deletes, re-creates and opens empty ISAM file.
REWIND positions on a beginning of a file.
ROLLBACK cancels operations in transaction.
SELECT
SELECT_ALL
SELECT_START
SKIP skips n lines in a report.
SOURCE defines fields for display on the screen or  on report.
START positions on a record which is determined by a key.
STOP_RUN stops program.
TABLE defines the table with which the program is currently working.
TOTAL_CALC - part of a batch program for processing group records.
UPDATE updates existing record of a file (table).
WAIT suspends the program execution for a specified interval.
WARMSG displays warning.
WHENEVER database fatal error processing
WINDOW transfers control to n-th window (subform).
WRITE writes record into a file (table).
WRITE_AUDIT writes record in journal file.
WRITE_EXTERNAL defines symbols or logical names.
