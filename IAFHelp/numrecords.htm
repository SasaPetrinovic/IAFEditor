NUMRECORDS
   

This function returns number of records in a file or in an RDB table.

Format

            NUMRECORDS(table_name)

Return Value Type

            Numeric.

Parameters

·    table_name - alpha literal containing table name from ‘Datasource’ dialog.

Notes

            If table_name denotes an RDB table, the function calculates result using SQL statement

                        SELECT count(*) from table_name

            If table_name denotes an indexed Btrieve file, program calculates result using Btrieve function BSTAT.

            If table_name denotes a sequential file with fixed record length, program calculates number of records as

                        numrecords = file_size / record_length

Example
  
      counter_1 = numrecords('tbl1')
