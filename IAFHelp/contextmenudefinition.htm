Context menu definition
   

Context menu is a popup menu displayed as the result of a right-click within the form.

There are two phases in context menu definition process. The first, context menu layout has to be defined. The second, the event handlers for each menu item should be programmed. If, for particular menu item, event handler does not exist, this item will be disabled at run-time.

Add new menu item text

Enter the text in "Menu item text" field and click "Add text". This text appears on the context menu panel, along with menu item identifier. This identifier is automatically assigned by generator. Default identifiers are cmitem_1, cmitem_2, etc. Menu item identifier is a sequence of English letters, digits, underscore character(_)  and dollar sign ($). The first character must be a letter (A to Z). Maximum number of characters in an identifier is 30. Identifiers are case-insensitive, so CMITEM_1 and cmitem_1 denotes the same menu item.

Add separator

Click "Add separator". Menu separator appears on the context menu panel.

Delete menu item (text or separator)

Right-click the menu item on context menu panel. Select "Delete menu item" from the context menu.

Modify menu item text

Right-click the menu item on the context menu panel. Select "Modify text" from the context menu.

Modify menu item identifier

Right-click the menu item on the context menu panel. Select "Modify identifier" from the context menu.

Change menu item position

Click the menu item on the context menu panel and drag it, down or up, to the appropriate position.


Event handlers for menu items

Each event handler has the form

ON_EVENT_PROCEDURE OnClickMenuItem(menu_item_identifier)
            ...
            event handler code
            ...
END_PROCEDURE

On_event_procedure and OnClickMenuItem are keywords denoting the start of the handler.
Menu_item_identifier defines menu item, case-insensitive.
End_procedure denotes end of the particular handler.

When you select menu command Define/Context menu/Context menu code for the first time, the skeleton for all the handlers will be stored automatically into the code buffer.

Enabling/disabling menu items at run-time

To enable/disable a menu item, Setenable method can be used in the form

menu_item_identifier.SetEnable(flag)
