Subform properties
   

Object name

When the new subform is created, generator automatically assigns the name Subform_n, where n = 1, 2, …20. Object name can be used in Methods. Subform name cannot be changed.

Subform type
  
Type
Description
 
 
Standard
Normal subform.
Tabbed subform
All tabbed subforms occupy the same space on main frame, but only one could be visible at the moment. To change active subform, user should click appropriate tab.
Slave (invoked by CALL_WINDOW)
This type of subform could be invoked via CALL_WINDOW statement only.
Overlay
Many subforms overlays the same area in the parent subform. This means the following: parent subform and one of overlay subforms are active at the same moment.
To display an overlay, use the DISPLAY method,
for example, subform_5.display("").

Tooltip
 


Next subform

Applies to standard and tabbed subform. Defines the subform where program control goes to the end of the current subform processing.

Caption

Subform title, applies to tabbed subforms.

Conditional

Applies to standard and tabbed subform. If the subform is conditional, run-time engine checks the condition before activating the subform. If condition is satisfied, subform displays and sets the focus to the first active field in sequence.

Condition

Subform condition, a standard conditional expression.

Suppress DEFAULT READ in this subform

If this option is set, code for default read will not be generated. Normally, when all segments of the primary key of a table are filled (if table has access mode = update), code for implicit reading will be generated, see Implicit record reading (Default reading).

Ignore ERRMSG during display phase

When the program is in display phase (after DEFAULT READ or DISPLAY_WINDOW_FIELDS), all the statements will be executed, including CHECK and ERRMSG. In some cases, this could result in many message boxes, sound signals, etc. If this option is set, any error message will not be displayed in display phase.

Font name

Choose one from offered font names, applies to tabbed subforms.

Font size

Specifies font size of the text.

Bold

Specify that font weight is 700. Normal weight is 400.

Italic

Specify whether text is italic.

Underline

Specify whether text is underlined.

Border style

Values:

Transparent      border is not seen,
Solid                 the border is standard line,
Dots                 the border is dotted line,
Dashes             the border is dashed line.

Border color

Is applied only if border style is not transparent.

Border width

Is applied only if border style is not transparent. Possibly values are 0, 1, 2,... pixels.

Text color

Color of text, applies to tabbed subforms.
