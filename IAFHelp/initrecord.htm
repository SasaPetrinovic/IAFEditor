INIT_RECORD
   

This statement fills all fields in record with initial values (ZERO, SPACE, ZERO_DATE, ZERO_TIME).

Format

            INIT_RECORD file[.record]

Where


·     file - a name of the file (table) from datasource dialog.
·     record - a name of the record, if the file has many record definitions.

Note

            For RDB tables, INIT_RECORD moves default values (if exist) into record buffer if access mode is 'update' or 'write'.

            Example: if table TBL1 was created using SQL command
            CREATE TABLE TBL1 (col1 varchar(5) default 'abcd')
            after INIT_RECORD field 'COL1' will contain 'abcd'.

Example

  
      INIT_RECORD tbl1
      INIT_RECORD file1.redef2
