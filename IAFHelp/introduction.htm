Introduction to IAF
   

Activation

To activate generator, just enter IAF in Run dialog and click OK. Main program form appears on the screen. The folder containing iaf.exe have to be defined by PATH environment variable.


·     If you use Iaf for the first time, select "New Data Dictionary" menu command to create and open Data Dictionary and application.

·     Otherwise, open DD/Application by "Open Data Dictionary" menu command or select a DD/Application from MRU list.


Command line activation

Command line has the format

IAF [dd_name[ application]]

DD_name, application - optional parameters.

If dd_name contains blanks, use question marks to surround the parameter. If the application is omitted, the first created application will be opened.

Examples:

Iaf
Iaf c:\my_dddir\mydd
Iaf c:\my_dddir\mydd.dd1
Iaf c:\my_dddir\mydd myappl
Iaf c:\my_dddir\mydd.dd1myappl

Iaf "c:\my dddir\mydd.dd1"
Iaf "c:\my dddir\mydd.dd1" myappl
