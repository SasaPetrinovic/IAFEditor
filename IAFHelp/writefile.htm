Write
   

This method writes one record to a text file.

Format

            Write(field)

Parameters


·     field - buffer containing text, an alpha variable. If text file is utf-16 type, this variable must be a UTF-16 one.


Return Value

            Status, numeric variable.

            Possible values:

            0          Ok.
            -2         Write error
            -3         File not opened
            
Applied To

            File object.


Example

  
      $stat = File.write($rec)
