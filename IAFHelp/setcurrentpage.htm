Setcurrentpage
   

This method sets current page in tab control.

Format

            Setcurrentpage()


Parameters

            None.


Applied To

            Tab control.

Note

            This method just switches between two tab pages, displaying previously displayed fields. If you want to execute program code in new selected page, Init method should be used.

Examples

  
      Tab_1.tabPage1.Setcurrentpage()
      $loc = 'tab_1.tabpage_2'
      $loc.Setcurrentpage()

! Change page, init fields and execute code
            Tab_1.tabPage_2.Setcurrentpage()
            Tab_1.tabPage_2.Init()
            Tab_1.Setfocus()
