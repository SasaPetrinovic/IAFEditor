DeleteDirectory
   

This method deletes directory from the connected FTP server.

Format

            Deletedirectory(dirname)

Parameters


·     dirname - an alpha variable or alpha literal. Specifies directory name.


Return Value

            Alpha variable. If connecting was successful, value is blank. Otherwise, the variable holds an error message.


Applied To

            Ftp object.

Note


Examples

  
      $msg = Ftp.Deletedirectory('/dir1/dir2')
      if $msg <> SPACE
            Pause 'Error: ' | $msg
      end_if
