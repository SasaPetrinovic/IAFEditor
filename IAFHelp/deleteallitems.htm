Deleteallitems
   

This method removes all items from an object.

Format 1
 
            Deleteallitems()

Format 2

  
Deleteallitems (line)
This format is applied to grid combobox only.


Parameters


·     line - grid line, numeric value. The lines are counted from 1.


Applied To

            Combobox, data selector, tree.

Notes

            Format 2 is applied to grid combobox if every line contains comboboxes with different content.

Examples

  
! Empty standard combo-box
            Combobox_1.Deleteallitems()

! Empty grid combobox in current line and fill it again
            Combobox_gr.Deleteallitems(current_line)
      Combobox_gr.InsertItemInLineCombo(current_line,'01','Item 1')
      Combobox_gr.InsertItemInLineCombo(current_line,'02','Item 2')
      Combobox_gr.InsertItemInLineCombo(current_line,'03','Item 3')
