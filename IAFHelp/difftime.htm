DIFF_TIME
   

This function returns time difference between two time values.

Format

            DIFF_TIME(time1, time2)

Return Value Type

            Time, 6 digits, abs(time1 - time2).

Parameters


·    time1, time2 - standard time fields.

Examples

  
      $time = DIFF_TIME($time1, systime)
      $time = DIFF_TIME(time(datetime1),time(datetime2))
      $time = DIFF_TIME("12:01","12:00") !! result=00:01:00
      $time = DIFF_TIME("12:00","12:01") !! result=00:01:00
