CONVERT_YMD_MDY
   

This function converts a date value to an alphanumeric one.

Format

            CONVERT_YMD_MDY(date_exp)

Result Value Type

            Alpha.
            
Parameters

·    date_exp -
Note

            Result is a 8 or 6-character alphanumeric value. The first two characters denote
            the month of date_exp, the second two characters denote the day of date_exp,
            and last four (two) digits denote year of date_exp.

            This functions is usually used to export data to external applications.

Example
  
      $C = CONVERT_YMD_MDY(“1-feb-2001”) !! result is "02012001"
