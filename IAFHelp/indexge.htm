INDEXGE
   

This function compares a field to array elements, beginning from the first element. If an array element is greater or equal to the field, function returns index of that element.

Format

            INDEXGE(array,field)

Return Value Type

            Numeric.

Parameters


·     array - an array of elements, alpha, numeric, date, time, datetime. Array specification () can be used with this parameter.
·     field - any field. Field type must be the same as type of array elements.

Notes

            The first element has index 1. If the field was not found in the array, result is -1.

            The array elements must be sorted in ascending order.

Examples

  
! Let $array has values 100, 200, 300, 400, 500
      counter_1 = indexge($array,400)     !! result is 4
      counter_1 = indexge($array,350)     !! result is 4
      counter_1 = indexge($array,50)      !! result is 1
      counter_1 = indexge($array,600)     !! result is -1
