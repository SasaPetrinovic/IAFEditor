Setnumbercolumns
   

This method sets number of columns in a read-only grid.

Format

            Setnumbercolumns (columns)

Parameters

·     columns - number of columns, integer

Applied To

            Read-only grid.

Note

            New columns take on attributes (font, colors, alignment) from last created column.

Example

  
      table_1.Setnumbercolumns(counter_1)
