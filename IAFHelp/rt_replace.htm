RT_REPLACE
   

This function is used to compose a SQL statement at run-time.

Format

            RT_REPLACE(alpha_variable)

Return Value Type

            None.

Parameters


·     alpha_variable - value of this variable will be inserted into a SQL statement at run-time. The value replaces RT_REPLACE(alpha_variable).

Examples

  

1/
      Execsql Insert into tbl values ('abc', rt_replace($a80))

If $a80 contains text 'value xyz', the above statement at run-time will be

      Insert into tbl values ('abc', 'value xyz')

The same result can be reached by using parameter:

      Execsql Insert into tbl values ('abc', :$a80)

but there are situations where parameters cannot be used.

Notes:

If $a80 contains text value xyz, the run-time statement will be

      Insert into tbl values ('abc', value xyz) !!invalid

If $a80 contains text "value xyz", the run-time statement will be

      Insert into tbl values ('abc', "value xyz") !!invalid

2/
      Execsql Insert into tbl (col1, col2)
            Select rt_replace($a80), fld2 from tbl2

            In above example, we can not use parameter, because SQL parser does not allow parameter in select_list. At run-time, this statement will be

      Insert into tbl (col1, col2)
            Select 'value xyz', fld2 from tbl2
