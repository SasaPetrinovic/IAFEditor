Open
   

This method opens printer port.

Format

            Open(printer_port)

Parameters


·     printer_port - an alpha variable or alpha literal. Specifies name of the printer port, such as LPT1:, LPT2:, COM1:, ...

Return Value

            Status, numeric variable.

            Possible values:

            0          Ok
            -1         Printer port already open
            > 0       An error
            
Applied To

            Lineprinter object.

Note

Examples

  
      $lpstat = Lineprinter.Open('LPT1:')
      $lpstat = Lineprinter.Open('COM2:')

      $portname = 'LPT2:'
      $lpstat = Lineprinter.Open($portname)
      if $lpstat = 0
            ...
      end_if
