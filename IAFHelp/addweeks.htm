ADD_WEEKS
   

This function adds number of weeks to a date.

Format

            ADD_WEEKS(date, numweeks)

Return Value Type

            Date.

Parameters


·     date - date_exp
·     numweeks - number of weeks, numeric_exp

Note

            1 week = 7 days.

            ADD_WEEKS(date,numweeks) = ADD_DAYS(date,7*numweeks)

Examples

  
      $date = ADD_WEEKS(today,2)
 


See Also   ADD_DAYS, ADD_MONTHS, ADD_YEARS, Date Functions
