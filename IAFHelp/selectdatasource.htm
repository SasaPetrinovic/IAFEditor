Select datasource
   


List


List on the left-side of the dialog shows all ODBC datasources defined on your PC, including User DSN and System DSN.

Database

Database shows default database type (MS SQL Server, Oracle, DB2, ...) used in your application.

DB2 schema

If we select DB2 database, a schema for tables loading can be entered. If this parameter is empty, the default schema defined by login username will be used.

Include all users

Ignore login username or DB2 schema and store all the tables from the database into the list. In this case, the system tables will be included too.

Include synonyms

Show synonyms in the list.

Include views

Show views in the list.

Create trace file

If this check-box is checked, a trace file will be created during loading phase. This file can be used for debugging purposes only. File name is DDLOADTRAC.LOG and resides on default directory. This check-box is visible in "Load table definitions" dialog in Data Dictionary utility.
