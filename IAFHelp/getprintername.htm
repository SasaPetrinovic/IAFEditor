GETPRINTERNAME
   

This function returns printer name from list of available printers.

Format

            GETPRINTERNAME(index)

Return Value Type

            Alpha.

Parameters


·     index - index in the list of printers, integer. Index starts with 1. This parameter must be in the range from 1 to number of available printers.



Note

            Result is blank string if index <= 0, or index > number_of_available_printers.


Examples

  
!! Read names of all available printers
      counter_1 = 0
      do_while 1=1
            add 1 to counter_1
            $printer = getprintername(counter_1)
            if $printer = SPACE     !! End of list
                  break
            end_if
            ...
            ... !! do something with $printer
      end_do
