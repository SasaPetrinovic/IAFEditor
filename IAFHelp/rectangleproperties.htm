Rectangle properties
   

Object name

When the new rectangle is created, generator automatically assigns the name Rectangle_nnn, where nnn = 1, 2, …
Object name can be maximum 30 characters long. Allowed characters are A-Z, 0-9, _ and $.
First character must be a letter. Object name is case-insensitive. Object name can be used in Methods.

Member of group (information only, cannot be changed)

If the object is a member of a group, the name of this group is displayed. An object can be the member of one group only.

Visible

Determines whether the object is visible when the subform is first displayed.

Rectangle style

  
Standard
Rectangle has standard corners
Rounded corners
Rectangle has rounded corners

Background style

  
Normal
Background is filled with the background color
Transparent
Background remains unchanged

Background color

This attribute is applied only if the background style is normal.

Special effects

  
Flat
Border is standard line
Raised
Border with raised edge
Sunken
Border with sunken edge
Etched
Border is etched line
Bump
Border is bumped line
Shadow
Shadow below and to the right of the rectangle

Border style

  
Transparent
Border is not seen
Solid
Border is standard, solid, line
Dots
Border is dotted line
Dashes
Border is dashed line

Border color

Is applied only if border style is not transparent.

Border width

Is applied only if border style is not transparent. Possibly values are 0, 1, 2,... pixels.

Shading style

Specifies that Color_from fades gradually to Color_to. Valid if background style is Normal.

  
None
No shading
Horizontal
Horizontal pattern
Vertical
Vertical pattern

Shading variant

Possible values are 1 to 4, valid if shading style is Horizontal or Vertical.

Color from

Starting color for shading, valid if shading style is Horizontal or Vertical.

Color to

Ending color for shading, valid if shading style is Horizontal or Vertical.

Z-Order

Possible values: 0, -1, -2, -3, ... See Z-Order.

Extend to bottom of lineset (reports only)

If you want to surround all the lines of a lineset by the rectangle, select this check box. It implies that the rectangle will have variable height.
