Init
   

This method initializes object on the screen.

Format

            Init()

Parameters

            None.

Applied To

            All objects.

Note

            The initialization is just visual. For combobox, Init() will remove any text from the screen, but the refilling is not necessary because the internal structures (data and text) keep previous values. For tab control, all the fields in current page will be initialized.

            All data-entry objects (check box, radio button, destination field, combo box, data-entry grid, data selector) have 'field_processed' flag. This flag is set when the data is entered into the field. The Init() method resets this flag as well, but memory buffer remains unchanged.

Examples

  
      combobox_1.Init()
      form.Init()
