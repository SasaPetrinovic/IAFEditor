WRITE_EXTERNAL
   

This statement writes or deletes a symbol.

Format 1


            WRITE_EXTERNAL /NAME: symbol_name   field_1[,…, field_n]

Format 2

            WRITE_EXTERNAL /VARNAME: alpha_variable  field_1[,…, field_n]

Format 3

            WRITE_EXTERNAL /DELETE /NAME: symbol_name

Format 4

            WRITE_EXTERNAL /DELETE /VARNAME: alpha_variable


Where


·     symbol_name - the name of a symbol. The first character of name must not be underline (_). Blanks in name not allowed. If the first character is a dollar sign ($), the symbol will be written into the system registry under the key HKEY_CURRENT_USER\software\sing\iafw\rt. Otherwise, the symbol will be written into SYMNTT00.IAF file residing on the default directory. Also, the full name of registry key can be used in the form: HKEY_CURRENT_USER\subk1\subk2\sym or HKEY_LOCAL_MACHINE\subk1\subk2\sym.
·     alpha_variable - variable containing symbol name.
·     field_1,..., field_n - list of fields, making symbol value.


            Formats 1 and 2 write a symbol. If the symbol already exists, symbol value will be modified.

            Formats 3 and 4 remove a symbol. If we want to delete many symbols, wildcard character can be used:
            write_external/delete/name: $*   -> delete all symbols from HKEY_CURRENT_USER\software\sing\iafw\rt.
            write_external/delete/name: $namestart*
            write_external/delete/name: $*nameend
            

Qualifiers

            /NAME defines symbol name.
            /VARNAME defines an alpha variable containing symbol name.
            /DELETE defines symbol removing operation.


Note

            WRITE_EXTERNAL statement is used mainly to move parameters from one program to another one. The mechanism is the following: one program writes a symbols and chains (calls) next program. Called program reads symbol via READ_EXTERNAL statement.

            If we use many fields in the statement, their values will be concatenated using TAB character as a field separator. Symbol value length is maximum 250 bytes.


Examples

  
      WRITE_EXTERNAL/NAME: sym1 counter_1 !! write into symntt00.iaf
      WRITE_EXTERNAL/NAME: sym2 counter_1, counter_2
      WRITE_EXTERNAL/NAME: $sym1 counter_1 !! write into registry
      WRITE_EXTERNAL/NAME: $HKEY_CURRENT_USER\ABC\SYM1 "1"
      WRITE_EXTERNAL/NAME: $HKEY_LOCAL_MACHINE\SOFTWARE\SYM1 "1"

      $sname = "$sym1"
      WRITE_EXTERNAL/VARNAME: $sname counter_1

!! Delete SYM1 from HKEY_CURRENT_USER\software\sing\iafw\rt
      WRITE_EXTERNAL/DELETE/NAME: $sym1
      WRITE_EXTERNAL/DELETE/NAME: $menu_cmd*
      WRITE_EXTERNAL/DELETE/NAME: $*menu
      WRITE_EXTERNAL/DELETE/NAME: $*
      WRITE_EXTERNAL/DELETE/NAME: $HKEY_CURRENT_USER\ABC\SYM1
