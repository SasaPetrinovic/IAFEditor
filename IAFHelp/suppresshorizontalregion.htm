Suppress horizontal region
   

Horizontal region is a region within one lineset.

One horizontal region cannot be extended across multiple linesets.

Maximum number of regions in one report is 10.

For each region you define region position and a condition which specifies when this region is to be suppressed.

In Region column you have to enter start and the end of the region in pixels, in the form

starting_pixel-ending_pixel

Example:

101-120            Horizontal region between 101. (including) and 120. (including) pixels.
                                    The minus character is required.

In Condition column, you have to enter the suppress condition. The maximal number of characters is 70.

Example:

acc_1 = 0 and acc_2 = 0

The above condition need to be read like this: when acc_1 has value 0 and acc_2 has value 0, the region from 101. up to 120. pixels will not be printed.

What if the condition has more than 70 characters?
In that case, a local variable can be defined, let's say $suppreg, who can have two values: 1 or 0. In a field, who belongs to that lineset, the value of the variable need to be calculated.

Then, in Condition column you can write

$suppreg = 1    or         
$suppreg = 0    depending on how the value is calculated.


When the region does not print, whole report moves up, so that white emptiness cannot be seen on the paper.

Using menu command 'Show/ Suppress horizontal regions' you can see graphically regions on the report.

Example

  
Region
Condition
101-120
acc_1 = 0
121-140
acc_1 <> 0
 
 
 
 


How to remove a condition?

Blank either Region or Condition column, then click OK.

Important

If a field partly lies in the region, for example region is between 101. and 120. and the field is between 115. and 130. pixels, unclear (confusing) report will be generated at run-time.
