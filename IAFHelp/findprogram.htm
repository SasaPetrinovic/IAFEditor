Find Program
   

Find a program in Data Dictionary. This option is especially useful if a Data Dictionary contains many applications.

Program name

Enter the program name with wildcard characters. Wildcard characters are * (replaces many characters), and ? (replaces exactly one character).

Examples:

            *
            prog*
            *prog
            *prog*
            prog???
            ???

Program type

Select program type (interactive, report, batch).


Note

To activate appropriate generator, double click the list.
