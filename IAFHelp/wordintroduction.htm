Using Word - Introduction
   

Programs can transfer data to a MS Word document.
For this purpose we use object named Word and its methods.
There is only one Word object in a program. This means that two documents cannot be opened at the same time.

Program transfers data to MS Word in two ways: via bookmarks or using mailmerge method.

All methods return a value.

A program uses COM (OLE) interface to manipulate objects belonging to MS Word.
MS Word must be installed on local PC.

Object name:

            Word

Methods:

            Append - appends a file (Word or Excel) to the current document

            ExportPDF - Export to PDF file

            Close - discards any changes and stops Word
            Open - starts MS Word and opens document
            Run - saves or prints document
            Runmacro - runs a macro
            Setbookmark - stores text into bookmark
            Setcodepage - sets default code page for subsequent calls

            Opentable - creates new table or opens the existing one
            Insertrow - insert row into table
            Setcell - stores value into table cell
            Repeatheader- repeats table heading
            Closetable - closes table opened by Opentable

            Mergeopen - creates and opens data file for mailmerge
            Mergedata - stores one field into the mailmerge record
            Mergewrite - writes one record into data file
            Mergeclose - closes mailmerge data file
