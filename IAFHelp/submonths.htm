SUB_MONTHS
   

This function subtracts a number of months from a date.

Format

            SUB_MONTHS(date,nummonths)

Return Value Type

            Date.

Parameters

·     date - date_exp
·     nummonths - number of months, numeric_exp
Note

            Out-of-range result date will be corrected automatically.

Examples
  
      SUB_MONTHS (today,5)
      SUB_MONTHS (BEGINCY,-3)
      SUB_MONTHS (today,counter_1)

      SUB_MONTHS ("31-may-2001",1) !! result = 30-apr-2001
      SUB_MONTHS ("31-mar-2001",1) !! result = 28-feb-2001
