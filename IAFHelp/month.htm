MONTH
   

This function returns month of a date.

Format

            MONTH(date)

Result Value Type

            Numeric.

Parameters


·    date - date or datetime field.


Note

            If date = zero_date or date = NULL, result is 0.
            Otherwise, result is a number in the range from 1 to 12.

Examples

  
      $n2 = month(today)
