Rewind
   

This method repositions file pointer.

Format

            Rewind([seek_flag])


Parameter


·     seek_flag - new value of file pointer, alpha literal or alpha variable. Possible values: "begin", "end". Default value is "begin".

            Begin - positions file pointer to the beginning of the file.
            End - positions file pointer to the end of the file.


Example

  
      $stat = File.Rewind()
      $stat = File.Rewind('begin')
      $stat = File.Rewind('end')
