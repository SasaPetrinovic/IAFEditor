MESSAGEBOX
   

This function displays a message in a dialog box, waits for the user to click a button, and returns an integer indicating which button the user clicked.

Format


      MESSAGEBOX(prompt[, buttons] [, title])

Return Value Type

            Numeric.

            Possible values:
                        1          OK clicked
                        2          Cancel
                        3          Abort
                        4          Retry
                        5          Ignore
                        6          Yes
                        7          No


Parameters


·     prompt - Text to be displayed in the dialog box. Maximum length of the text is approximately 1024 characters, depending on the width of the character used.
If the text consists of more than one line, you can separate the lines using a carriage return/linefeed character combination (CHR(13) | CHR(10)) between each line.

·     button - numeric_exp
Sum of values specifying the number and type of buttons to display, the icon style to use, the identity of the default button, and the modality of the message box.
If ommited, the default value for button is 0.

Values:
0  Display OK button only.
1  Display OK and Cancel buttons.
2  Display Abort, Retry, and Ignore buttons.
3  Display Yes, No, and Cancel buttons.
4  Display Yes and No buttons.
5  Display Retry and Cancel buttons.

16 Display Critical Message icon.
32 Display Warning Query icon.
48 Display Warning Message icon.
64 Display Information Message icon.

0      First button is default.
256  Second button is default.
512  Third button is default.

0           Application modal; the user must respond to the message box before continuing
4096     System modal; all applications are suspended until the user responds to the
65536   Specifies the message box window as the foreground window
524288  Text is right aligned

                        1048576           Show suppress message check box.
                        2097152           Reset suppress condition for one message.
                        4194304           Reset all suppress conditions.

            When adding numbers to create a final value for the buttons argument, only one number from each group can be used.

·     title - Text to be displayed in the title bar of the dialog box.
If ommited, the program name is placed in the title bar.

Note

            If the dialog box displays a Cancel button, pressing the ESC key has the same effect             as clicking Cancel.

Examples

  
      if Messagebox("Remove record?",4+32+256,"Removing") = 6
            !! buttons Yes and NO
            !! icon = questionmark
            !! default button = No.
            !! result = 6 if user clicked Yes
            ...
      end_if

      Messagebox("Msg text",1+1048576,$title)   !! Show check box
      Messagebox("Msg text",2097152,$title)     !! Reset previous
      Messagebox("",4194304)  !! Reset all suppressed messages
