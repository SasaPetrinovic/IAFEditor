Timer
   

The timer is an object enabling a code (procedure) activation when a time period elapses. The object picture, visible in the form painter, will not be seen at run-time.

To insert new timer on the form, click button Timer in toolbox or choose menu command Objects/Timer. Move the cursor to the desired position,  then click. New timer is placed on the form.

To change timer properties double click on the timer

-or-

select the timer, right click to display context menu and choose Properties.
