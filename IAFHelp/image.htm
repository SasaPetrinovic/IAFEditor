Image
   

To insert new image on the form, click button Image in toolbox or choose menu command Objects/Image. Move the cursor to the desired position, then click. File open dialog appears on the screen, current folder is iafdir\iafres. All the images which we intend to use in our applications, should be copied to this folder. Iaf can display ICO, BMP, JPG and GIF images, including GIF animations. Select an image, clik Open. New image is placed on the form.

To change image properties double click on the image

-or-

select the image, right click to display context menu and choose Properties.

How run-time engine processes the image?

If Picture type is embeded, the image is linked to the executable program.

If Picture type is linked, the image file must be copied together with executable program to each PC running the program.

Programs with embeded images are simpler to deployment, but their size is bigger.
