Group of objects
   

A group is not object itself. The group is, simply,  a set of objects.

Create new group

Select objects, then choose menu command Objects/Groups/Create group. Enter group name in the dialog which appears on the screen. The group name is maximum 30 characters long. Allowed characters are A-Z, 0-9, _ and $. First character must be a letter.

One object can be member of one group only.

Remove an existing group

Choose menu command Objects/Groups/Ungroup and select a group from the list.

The following methods can be used on groups:

Init
Setbackcolor
Settextcolor
Setfont
Setvisible
Setenable
Setreadonly
