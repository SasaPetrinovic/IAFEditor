Close
   

This method discards any changes and stops Excel. After this action, another document can be opened, etc.

Format

            Close()

Return Value

            Status, numeric variable.

            Possible values:

            0          Ok
            -1         Document not opened
            
Applied To

            Excel object.

Note

            This method is usually used after the reading input document.

Examples
  
      $stat = Excel.Close()
