INIT_WINDOW_FIELDS
   

This statement visually initializes fields in the subform from the current field up to the end of subform.

Format


            INIT_WINDOW_FIELDS

Note

            INIT_WINDOW_FIELDS and DISPLAY_WINDOW_FIELDS statements replace default reading algorithm.

            Fields with sequence zero will not be initialized.

Example

  
      Destination = $seg1
      Select * from tbl1 Where kseg = :$seg1
      If record tbl1 found
            display_window_fields
      else
            init_window_fields
      End_if
