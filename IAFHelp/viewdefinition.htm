View definition
   

The view determines the way of the records reading in the program cycle (reports and batch programs).

The maximal number of the lines in view definition is 200.



Special case: the view is empty.

If the main table represents a RDB table, the generator generates

            SELECT * FROM main_table ORDER BY primary_key_segments


If the main table represents a sequential file, program reads whole file.

If the main file represents an ISAM files (like Btrieve), program reads whole file sorted by primary key.



Standard view

If the main table represents a RDB table, we use a standard SELECT statement. Although in the program cycle we want to read many records (rows), we use SELECT, not SELECT_ALL.

In the view we can write down just one SELECT statement.

Examples:

SELECT select_list FROM tbl1,tbl2 WHERE ... ORDER BY ...
SELECT * from tbl1 LEFT OUTER JOIN tbl2 ON fld1=fld2 WHERE ...
SELECT * from tbl1 UNION tbl2 WHERE ... ORDER BY ...

View with sequential file

If the main table represents a sequential file, we use an old syntax

            FILE_NAME [Where ...] [[NATIONAL_]SORTED_BY sort_field1, ..., sort_fieldN]

FILE_NAME is main file from Datasource dialog.
SORT_FIELDn has the format

            [ASCENDING | DESCENDING] field_name

Examples:
            SEQFILE SORTED_BY fld1, fld2
            SEQFILE SORTED_BY fld1, DESCENDING fld2
            SEQFILE NATIONAL_SORTED_BY fld1
            SEQFILE WHERE fldx>100 SORTED_BY fld1, fld2

View in recapitulation report

Recapitulation file is an internal, sequential, file used to print recapitulation report. We can use SQL-like syntax to define selection criteria and sort fields for this file:

            Select * from RECAP_FILE [Where ...] [Order By sort_field1, ..., sort_fieldN]


SORT_FIELDn has the format:

            recap_field [COLLATE collation_name] [ASC | DESC]

COLLATE clause should be applied to alpha fields only.


Examples:

Select * from RECAP_FILE ORDER BY recap_1, recap_5
Select * from RECAP_FILE ORDER BY recap_1, recap_5 DESC
Select * from RECAP_FILE ORDER BY recap_1, recap_5[2:3] Collate English
Select * from RECAP_FILE ORDER BY recap_1, recap_5[2:3] Collate English DESC

Select * from RECAP_FILE WHERE recap_1 = 'ABC'
Select * from RECAP_FILE WHERE recap_1 > 'ABC' ORDER BY recap_1
