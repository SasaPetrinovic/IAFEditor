Source Field
   

A source field displays value expression (a variable, formula, function, ...).

To insert new source field on the form, click button Source Field in toolbox or choose menu command Objects/Source Field.

Move the cursor to the desired position, then click. A dialog "Sourcefield definition" appears on the screen.


Enter the variable name (field to be displayed) or click Browse to choose a variable from a list. Click OK. If the variable exists, object SOURCEFIELD will be created: default edit mask is displayed on the screen, and one source line is placed in code buffer. The line is

Source = variable_name

See Source statement.

The source field object has the two components: visual appearance (edit mask on the screen) and code buffer (contains source statements defining which value will be seen on the screen at run-time).

The edit mask can be changed by context menu command "Change edit mask". To change the code, choose context menu command "Modify code".

Display multiline field

Enter the variable name, check "Multiline" check box and click OK. Resize the object.

Display an image

If an alpha variable contains path to image file, or if a blob variable contains image itself, enter the variable name, check "Image" check box and click OK. Resize the object. At run-time, the image will be centered if its dimensions are smaller than the rectangle. Otherwise, the image will be stretched.

Display a formula

Enter a numeric variable, for example ACC_1, and click OK. Edit mask appears on the screen. Choose "Modify code" command and change the statement like this:

Source = var_1 + var_2 / var_3 * 100,00

We can use any statement, function, internal variable in code buffer:

if acc_1 > 0
      source = acc_1
else_if acc_1 = 0
      source = var_1 - var_2
else
      source = abs(acc_2)
end_if

Maximal number of lines in the code buffer (for one object) is 1000, and at least one SOURCE statement must be encountered in the buffer.

All the variables in the "source" statements must have the same type (alpha, numeric, date or time).

If we want to display variables of the different types at the same place, then we must use an intermediate alpha variable:

$a30 = "ABCDE"
if acc_1 > 0
      $a30 = edit(acc_1,"ZZZ.ZZZ.ZZ9,99")
else_if acc_1 = 0
      $a30 = edit(today,"DD/MM/YYYY")
end_if
source = $a30

An alternative way to insert new source field

Choose Objects/Field Explorer menu command. Expand the tree in the dialog.

Click button Source Field in toolbox or choose menu command Objects/Source Field.

Drag the variable name from the tree to the form.


How run-time engine processes the source field?

If the object has sequence <> 0, when program flow reaches this object, the statements from code buffer will be executed. The value of last encountered source statement will be displayed.

If the object sequence is equal to zero, all the statements from the code buffer are ignored, and the object must be displayed using Display method. This is valid in interactive programs or  program title. In a report, the source field must have sequence <> 0.
