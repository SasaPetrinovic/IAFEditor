Setlineheight
   

This method sets grid line height.

Format

            Setlineheight (line, height)


Parameters


·     line - grid line, numeric value, integer. The first line is 1. Special value is CURRENT_LINE. Column heading has line = -1.

·     height - line height in pixels, integer. If height = -1, autofit operation will be applied.


Applied To

            Grid.

Note

            The grid attribute ' Variable row height ' must be checked.

Examples

  
      Table_1.Setlineheight(5,20)
      Table_1.Setlineheight(current_line,25)
      Table_1.Setlineheight($line,$h)
      Table_1.Setlineheight(-1,-1) !! Autofit column heading
