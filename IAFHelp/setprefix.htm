Setprefix
   

Defines qualifier of nodes or elements if they need to be qualified.


Format

            Setprefix (qualifier)

Parameters


·     qualifier - an alpha variable or alpha literal. Specifies the qualifier.

Return Value

            Status, numeric variable.

            Possible values:

            0          Ok
            <> 0     Error
            
Applied To

            Xml object.

Note

            To disable current qualifier, call the method with empty string.


Example

  
      $stat = Xml.Setprefix("def")
