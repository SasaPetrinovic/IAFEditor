Numeric Functions
   


ABS returns absolute value.
CEIL rounds a numeric value.
DIGIT_OF returns one digit from a numeric value.
IDIVIDE returns result from the integer division.
INDEX compares a field to array elements.
INDEXGE compares a field to array elements.
INDEXLE compares a field to array elements.
MOD returns remainder from the integer division.
NUM converts alpha field to numeric field.
NUMRECORDS returns number of records in a file or in an RDB table.
ROUND rounds numeric field.
SQRT returns square root of a numeric value.
SUM returns sum of elements of a numeric array.
SYMBOL returns value of a symbol.
TRUNC truncates numeric field.
VALUE converts an alpha literal to the numeric value.
