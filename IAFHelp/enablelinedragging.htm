Enablelinedragging
   

This method enables or disables line dragging.

Format

            Enablelinedragging (value)

Parameters


·    value - a numeric variable or numeric literal.
                        value = 0, line dragging will be disabled,
                        value <> 0, line dragging will be enabled.

Applied To

            Display-only grid.

Notes


Example

  
      Table_1.Enablelinedragging(0) !! Disables dragging
      Table_1.Enablelinedragging(1) !! Enables dragging
