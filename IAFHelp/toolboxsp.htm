Toolbox
   

The toolbox contains button palette for object creating. To create an object, click a toolbox button, move the cursor at desired position on work area and click.


The buttons on the toolbox work as toggle switch: one click puts the button in active state enabling object creating, the next click returns the button to inactive state. If an object is successfully created, the button automatically returns to inactive state. To create next object, we must click the button again, etc.

If we want to create more objects of the same type, one after another, Ctrl/Click has to be used. In that case, the button remains active until next Click.
