Mailing label format
   

Top margin (mm)

            The distance from the top edge of the paper to the first label in millimeters.

Side margin (mm)

            The distance from the left edge of the paper to the first label in millimeters.

Width (mm)

            The width of the label in millimeters. If there is horizontal gap between labels, the gap size should be added to the width.

Height (mm)

            The height of the label in millimeters. If there is vertical gap between labels, the gap size should be added to the height.

Number across

            Number of labels in a row.

Number down

            Number of rows.

Page size

            Type of the paper (A4, A5, Letter, ...). If 'Custom' is selected, form name should be defined too (see Form name bellow).

Draw in physical units (mm) at run-time

            If pre-printed forms are used, this option enables printing text or graphics to exact position, for example, 5 mm from the left edge of the label.

Draw cross lines to separate labels

            If ordinary, clear paper is used, lines between labels can be drawn for easier paper cutting. 

Number of copies

            Numeric literal or numeric variable. If blank, 1 is assumed. This parameters is valid only if we choose direct printing in Print dialog at run-time.

Form name

            This parameter is mainly used for nonstandard paper size. Run-time engine try to sets paper size to "form name" if the default printer supports such paper. If this parameter is blank, default paper will be determined according to report dimensions (page width, page height).

Print-file name, title, ... : see Report parameters .
