Introduction
   

To avoid windows spooler and directly print plain text to a printer on a PC, LINEPRINTER object can be used. Printer can be locally connected to the PC's port, or can be shared printer on local area network. If the printer is shared resource, redirection must be performed via NET USE command, for example: NET USE LPT2: \\server\printer1.

All methods return an integer value.


Object name:

            Lineprinter

Methods:

            Open - opens a printer port
            Write - writes a text to the printer
            Close - closes the printer port



            Example

            $lpstat = Lineprinter.Open('LPT1:')
            if $lpstat = 0
                        $lpstat = Lineprinter.Write('Some text')
                        $lpstat = Lineprinter.Write(crlf)

                        $lpstat = Lineprinter.Write(compress_trim('Some text',crlf))

                        $lpstat = Lineprinter.Write(compress_trim($escseq1,'Some text',$escseq2,crlf))

                        $lpstat = Lineprinter.Close()
            end_if
