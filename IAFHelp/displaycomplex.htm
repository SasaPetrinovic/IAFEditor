Displaycomplex
   

This method displays a text. Some segments of the text could have various font attributes (size, weight, italic, underline).

Format

            Displaycomplex (field, fontsize, fontweight, fontitalic, fontunderline, ...)


Parameters


·     field - a field to be displayed, one segment of the text.
·     fontsize - font size, integer.
·     fontweight - font weight (intensity), an integer from 0 to 1000.
·     fontitalic - italic flag, integer. Value 0, normal; <> 0, italic.
·     fontunderline - underline flag, integer. Value 0, normal; <> 0, underline.


Applied To

            Source field.

Notes


·           The object must have 'Background style = Normal'.

·           Numeric, date and time fields will be edited before displaying.

·           All the fields will be all-trimmed before displaying. The distance between various segments is the average width of a character.

·           Font name is defined in Properties dialog.

·           The height of the object has to be appropriate to accept all the segments heights.

·           Cannot be used in a grid.

Examples

  
      sourcefield_1.displaycomplex(f1,14,400,0,0,today,24,700,1,0)
      sourcefield_1.displaycomplex( \
                        f1,14,400,0,0, \
                        acc_1,16,400,1,0, \
                        today,14,700,0,0, \
                        systime,14,700,0,1)
