ADD_SECONDS
   

This function adds number of seconds to a time or datetime field.

Format

            ADD_SECONDS(time, numseconds)

Return Value Type

            If 'time' parameter is a time field, return value is time.
            If 'time' parameter is a datetime field, return value is datetime.

Parameters


·     time - time_exp, or datetime field
·     numseconds - number of seconds, numeric_exp

Note 1

            If 'time' parameter is a time, out-of-range result time will be corrected automatically.

Note 2

            If 'time' parameter is a datetime, result is datetime field and all the field's components will be adjusted.


Examples

  
      $res_time = ADD_SECONDS (systime,5)
      $res_time = ADD_SECONDS (systime,counter_1)
      $res_time = ADD_SECONDS ("23:59:50",20) !! result is 00:00:10

      $dt = ADD_SECONDS("2009-12-31 23:59:59",2) !!result is 2010-01-01 00:00:02
