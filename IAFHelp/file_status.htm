FILE_STATUS
   

This function returns file status.

Format

            FILE_STATUS(file_name)

Return Value Type

            Alphanumeric.

            Possible values:
  
Result description
Result output format
 
 
Logical name does not exist
-1
Other errors
numeric_error;textual description, for example

2;The system cannot find the file specified.
 
Successful
0;file_size;date_created;date_modified, for example

0;14560;20090120101533;20090504125123;



Parameters


·     file_name - name of file to be analyzed, logical names allowed.


Note

            How to check folder existence?
            Add a file name to folder string, invoke FILE_STATUS and check return value. If the first character of return value is '3',  folder does not exist.

            $newstring = compress_trim($folder_name,"\x.xxx")
            $fs        = file_status($newstring)
            if $fs[1:1] = "3"
                        Pause "Folder does not exist"
            end_if


Examples

  
      $FS = FILE_STATUS("c:\temp\a.tmp")
      $FS = FILE_STATUS("dat_dir:a.tmp")
where $FS is defined as alpha, 128 characters long
