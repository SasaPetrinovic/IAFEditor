Setindent
   

This method sets new indentation, i.e. the margin between the border and the text.


Format 1
            
Setindent(indent)
Set indentation of an object

Format 2
  
Setindent(line, col, indent)
Set indentation of a grid cell

Parameters


·     indent - new indentation, in pixels, integer literal or integer variable.

·     line - grid line, numeric value. The lines are counted from 1.To specify current line, we can use the internal variable CURRENT_LINE or star (*).

·     col - grid column, numeric value. The columns are counted from 1.


Applied To

            Text and source field.

Note

            If an object is left aligned (Horizontal alignment = left), this method changes left margin. If object is right aligned (Horizontal alignment = right),  the method moves right margin to the left.


Examples

  
      label_1.Setindent(5) !! moves left margin 5 pixels to the right
      sourcefield_1.Setindent(10)
      table_1.Setindent(counter_1,2,5)
      table_1.Setindent(current_line,2,5)
      table_1.Setindent(*,2,5) !! same as above
