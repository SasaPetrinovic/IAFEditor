Pagesetup
   

This method changes some page layout parameters.

Format

            Pagesetup (parameters)


Parameters


·     parameters - parameters to be changed, alpha literal or alpha variable. A parameter is text in the form: keyword = value. If more keywords are used in the same function call, they should be separated by semicolon (;). The keywords, implemented so far are: ORIENTATION (changes page orientation), BOTTOMMARGIN (changes bottom margin), LEFTMARGIN (changes left margin), RIGHTMARGIN (changes right margin), TOPMARGIN (changes top margin), ZOOM (changes zoom),  GRIDLINES (print with gridlines).

            ORIENTATION can have two values: xlLandscape or xlPortrait.
            BOTTOMMARGIN, LEFTMARGIN, RIGHTMARGIN, TOPMARGIN are decimal numbers representing centimeters.
            ZOOM is an integer.
            GRIDLINES values: 1 or Y (set gridlines); 0 or N (no gridlines).

Return Value

            Status, numeric variable.

            Possible values:

            0          Ok
            -1         Document not opened
            -2         Invalid parameter
            
Applied To

            Excel object.

Examples

  
      $stat = Excel.Pagesetup('Orientation=xlLandscape')
      $stat = Excel.Pagesetup('Orientation=xlPortrait')
      $stat = Excel.Pagesetup('Leftmargin=2.0') !! margin=2cm
      $stat = Excel.Pagesetup('Leftmargin=1')   !! margin=1cm
      $stat = Excel.Pagesetup('Orientation=xlLandscape; Leftmargin=1')
      $stat = Excel.Pagesetup('Leftmargin=1; Rightmargin=2.5; Topmargin=1')
      $stat = Excel.Pagesetup('Gridlines=Y')    !! use gridlines
