Hyperlink
   

Hyperlink in reports performs the same function as a hyperlink in a html page. We can create hyperlink to a web page, an email address or a file.

Hyperlink options

  
Email address
Hyperlink contains email address
Web page
Hyperlink contains web page name
File
Hyperlink contains file name

Tooltip text

Tooltip is a text that will be displayed when the cursor hovers above the hyperlink.

Tooltip can be an alpha literal, like "This is tooltip" or an alpha variable. If we use variable, its value must be set before object printing.

Example:

$alphatt = "This is tooltip for " | afield


Hyperlink address (email address, web page name, file name)

This field contains actual hyperlink: email address, web page name or file name.
We can specify alpha literal or alpha variable. Of course, value of the variable must be set before object printing.

File name can be absolute or relative.

Examples:

"emaddress@provider.com"
"http://www.acompany.com"

"c:\data\myfile.xls"    !! absolute path
"myfile.xls"            !! same directory as print file's dir
".\myfile.xls"          !! same as above
"subdir\myfile.xls"     !! relative: printfile_dir\subdir\...
"..\subdir\myfile.xls"  !! relative: the same level as printfile_dir
