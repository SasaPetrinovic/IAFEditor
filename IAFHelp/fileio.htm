FILE_IO
   

This function handles files.

Format


·           FILE_IO(“dir”, src_file, dest_file[, modifier])

            Stores list of files or directiories defined by src_file into the dest_file.
            Wildcard characters (* and ?) in src_file allowed, for example: *.dat.

            Allowed modifier values are:
            "D": list directories.
            "T": show modification date/time. Field separator is TAB (ASCII code 9).
            "TC": show creation date/time. Field separator is TAB (ASCII code 9).
            "S": time component contains seconds and milliseconds. Otherwise, only hours and minutes will be returned.

            If the function succeeds, the result is zero. If the function fails, result contains error status.


·           FILE_IO(“copy”, src_file, dest_file[, modifier])

            Copies an existing src_file into a new dest_file.

            Allowed modifier value is "O": overwrite existing output file.

            If the function succeeds, the result is zero. If the function fails, result contains error status.


·           FILE_IO(“append”, src_file, dest_file)

            Appends src_file to dest_file. If dest_file does not exist, it will be created. Multiple files in src_file (wildcards) not allowed.

            If the function succeeds, the result is zero. If the function fails, result contains error status.


·           FILE_IO(“delete”, src_file)

            Deletes (removes) src_file.

            If the function succeeds, the result is zero. If the function fails, result contains error status.


·           FILE_IO(“rename”, src_file, dest_file[, modifier])

            Renames src_file to dest_file.

            Alowed modifier value is:
            "O": overwrite existing output file.

            If the function succeeds, the result is zero. If the function fails, result contains error status.


·           FILE_IO(“move”, src_file, dest_file[, modifier])

            Moves src_file to dest_file.

            Allowed modifier value is:
            "O": overwrite existing output file.

            If the function succeeds, the result is zero. If the function fails, result contains error status.


·           FILE_IO("size",src_file)

            Returns file size in bytes.

            If the function succeeds, the result is >= 0 and contains file size.
            Result = -1, logical name not found
            Result = -2, file not found
            Result =-13, access denied.


·           FILE_IO("isunicode",src_file)

            Checks if the file contains Unicode text.

            Result = 0, file is not a Unicode
            Result = 1, file contains Unicode text
            Result = -1, logical name not found
            Result = -2, file not found
            Result =-13, access denied.


·           FILE_IO("create_dir",dir_name)

            Creates new directory. Dir_name is alpha literal or alpha variable and contains absolute path, for example 'c:\tstdir\level1\level2\level3'.

            Result = 0, ok
            Result = 1, directory already exists,
            Result < 0, error (negative value of errno).


Result Value Type

            Numeric.

Parameters

·     src_file - alpha literal, alpha variable
·     dest_file - alpha literal, alpha variable
·     modifier - alpha literal, alpha variable

Note
            
            Logical names in src_file and dest_file are allowed.

Examples

  
      $Status = FILE_IO("DIR", "*.DAT", "OUT.TXT")
      $Status = FILE_IO("DIR", "*.DAT", "OUT.TXT","T")
      $Status = FILE_IO("DIR", "*.DAT", "OUT.TXT","TC")
      $Status = FILE_IO("DIR", "*.DAT", "OUT.TXT","TS")
      $Status = FILE_IO("DIR", "*.DAT", "OUT.TXT","TCS")
      $Status = FILE_IO("DIR", "*.DAT", "OUT.TXT","DT")
      $Status = FILE_IO("COPY","DAT_DIR:F1.DAT","OUT_DIR:F2.DAT")
      $Status = FILE_IO("DELETE", "DAT_DIR:FILE1.DAT")
      $Delvar = "c:\dat\file1.idx"
      $Status = FILE_IO("delete",$Delvar)
      $Status = FILE_IO('delete','c:\temp\*.*')
      $Status = FILE_IO('delete','c:\temp\*.tmp')

      $Status = FILE_IO("COPY", $SRC, $DST, "O")
      $Status = FILE_IO('COPY', 'c:\dat\f1.cpp', 'd:\outdat\f1.cpp')
      $Status = FILE_IO('COPY', 'c:\dat\f1.cpp', 'd:\outdat\fout.cpp')
      $Status = FILE_IO('COPY', 'c:\dat\*.cpp', 'd:\outdat')
      $Status = FILE_IO('COPY', 'c:\dat\*.cpp', 'd:\outdat\')     !! same as above
      $Status = FILE_IO('COPY', 'c:\dat\*.cpp', 'd:\outdat\*.*')  !! same as above

      $size   = FILE_IO("size","mydir:file.txt")
      $status = FILE_IO("CREATE_DIR","c:\tstdir\level1")
