GETSCRIPT
   

This function tries to determine script used in an Unicode text.

Format

            GETSCRIPT(Unicode_text)

Result Value Type

            Numeric.

            Values:

            0          - Basic Latin
            1          - Latin 1 supplement
            2          - Latin Extended A
            3          - Latin Extended B
            4          - Greek
            5          - Cyrillic
            6          - Armenian
            7          - Georgian
            8          - Hebrew
            9          - Arabic

            -1         - Unknown

Parameters


·     Unicode_text  - utf-16 alpha field.


Note

Examples

  
      $res = GETSCRIPT($utf16)
