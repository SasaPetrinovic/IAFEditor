COMPARE_COLLATE
   

This function compares two alpha fields according to a collation.

Format

            COMPARE_COLLATE(str1, str2, collation_name)

Return Value Type

            Numeric.

            Result value:

            < 0 if str1 < str2
            > 0 if str1 > str2
            = 0 if str1 = str2

Parameters


·     str1 - the first alpha field.
·     str2 - the second alpha field.
·     collation_name - alpha literal denoting collation.

Note

Examples

  
      $res = COMPARE_COLLATE(str1, str2, 'English')
      $res = COMPARE_COLLATE(str1, str2, 'English_CI')

      $res = COMPARE_COLLATE('A', 'B', 'English') !! $res =-1
      $res = COMPARE_COLLATE('C', 'B', 'English') !! $res = 1
      $res = COMPARE_COLLATE('A', 'a', 'English') !! $res =-1
      $res = COMPARE_COLLATE('A', 'a', 'English_CI') !! $res = 0
