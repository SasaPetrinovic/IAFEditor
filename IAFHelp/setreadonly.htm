Setreadonly
   

This method disables/enables keyboard and mouse input for an object.


Format

            Setreadonly(value)


Parameters


·     value - a numeric variable, integer. Value <> 0, object is read-only and cannot be edited or clicked,  value = 0, object is not read-only.

Applied To

            Active objects: Destination field, Combobox, Checkbox, Radiobutton, Pushbutton, Bitmapbutton, Grid, Form.

Notes

1/
            Read-only state remains active until object.Setreadonly(0), object.Setenable(1) or FINISH_WINDOW.

2/
            
            In FORM.SETREADONLY(1), all the fields on the current subform become read-only, ie. can not be edited. This situation continues until the next subform initialization (finish_window). In this case, the good practice is to create a cancel button (sequence = 0) on the subform and do the following:

            if condition
                        form.setreadonly(1)
                        cancel_button.setreadonly(0)
                        cancel_button.setfocus()
            end_if
