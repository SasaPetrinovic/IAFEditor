Sumcolif
   

This method returns conditional sum of values in a column of a grid.

Format

            Sumcolif (column_number, criteria)

Parameters


·     column_number - numeric literal or numeric variable representing sum column. If this column contains non-numeric fields, result will be zero. The first column is 1, second one is 2, etc.

·     criteria - conditional expression that defines which cells will be added. Conditional expression has the form

      RUNNINGCELL(comp_column)   comparison_operator   value

      This means the following: if cell in column COMP_COLUMN has the value 'VALUE', then the corresponding value in column COLUMN_NUMBER will be added to the sum.


Applied To

            Grid.

Examples

  
      acc_1 = table_1.sumcolif(1, runningcell(2)='D') !! sum values in column 1
      acc_1 = table_1.sumcolif(9, runningcell(8)<0) !! sum values in column 9
