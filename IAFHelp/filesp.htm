File Menu
   

Save to recovery file

Saves current work to journal file program_name.JOU.
If generator abnormally ends, next time when we try to modify this program,  the message "Recovery file exists. Use it?" appears. If we click YES, our saved work will be restored.

See also Default/Options dialog for autosave interval definition.

Run

Activate File dialog to choose any executable program.

Exit & generate

Writes current work to the Data Dictionary and generates executable program.

Exit & save

Writes current work to the Data Dictionary.

Quit

Abandons all changes and closes generator.
