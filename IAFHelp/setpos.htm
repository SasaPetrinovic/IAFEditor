Setpos
   

This method sets the current position of the progress bar.

Format

            Setpos(pos)

Parameters


·     pos - numeric variable specifying current position of the progress bar.


Applied To

            Progress bar.

Example

  
      progress_1.setrange(1,100)
      counter_1 = 0
      do_while counter_1 < 100
            add 1 to counter_1
            progress_1.Setpos(counter_1)
      end_do
