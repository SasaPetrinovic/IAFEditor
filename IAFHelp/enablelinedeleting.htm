Enablelinedeleting
   

This method enables or disables line deleting.

Format

            Enablelinedeleting(value)

Parameters


·     value - a numeric variable, integer. Value=0 disables line deleting. Value <> 0 enables line deleting.

Applied To

            Data-entry grid.

Note

            By default, the lines can be deleted.

            This method is used usually in 'Table starting actions'.

Example

  
            Table_1.enablelinedeleting(0) !! disable line deleting
      Table_1.enablelinedeleting(1) !! enable line deleting

!! Lets look at the following piece of code

!! Table starting actions
      select_all/load_key * from tbl1 where fld1 = :$Par1
      end_select
      Table_1.Enablelinedeleting(0)
!!
Loop SELECT_ALL ... END_SELECT reads record set from TBL1, simultaneously building internal table of primary keys (qualifier /LOAD_KEY). Run-time engine uses this internal table to properly display grid lines.
Method ENABLELINEDELETING(0) disables deleting of lines displayed in the grid.

At the end of 'Table starting actions', run-time goes through the internal table and displays grid lines, one by one. After the last line is displayed, 'Tables ending actions' will be activated, if exist, and finally, the cursor will be positioned in the first destination field of the first empty line.

In this case, new lines can be entered, the existing lines can be modified, but we cannot delete any grid line (the new or existing ones).
