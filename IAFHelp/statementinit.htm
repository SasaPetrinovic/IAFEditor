INIT
   

This statement initializes field values (ZERO, SPACE, ZERO_DATE, ZERO_TIME).

Format 1


            INIT field_1[(index)], ..., field_n[(index)]

Format 2

            INIT field()


Where


·     field_1,..., field_n - fields used in the program.
·     field() - an array. In this case, all the array elements will be initialized.


Examples

  
      INIT acc_1, counter_1, $loc1, $loc2($ndx)
      INIT $loc_array()
