Using Excel - Introduction
   

Programs can transfer data to and from a MS Excel document.
For this purpose we use object named Excel and its methods.
There is only one Excel object in a program. This means that two documents cannot be opened at the same time.

All methods return a value.

A program uses COM (OLE) interface to manipulate objects belonging to MS Excel.
MS Excel must be installed on local PC.


Object name:

            Excel

Methods:

            Close - discards any changes and stops Excel
            ExportXML - exports Excel workbook to XML file
            Getcell - reads a value from MS Excel cell
            Getnumbercols - returns number of columns in the current sheet
            Getnumberrows - returns number of rows in the current sheet
            Getnumbersheets - returns number of sheets in the current workbook
            Getsheetname - returns name of the current sheet
            Joincells - combines a range of cells into one large cell.
            Loadexternal - loads external datasource.
            Open - starts MS Excel and opens (creates) document
            Pagesetup - changes page layout parameters
            Run - saves the document, breaks connection, and gives control to the user
            Runmacro - runs a procedure or macro
            Setalignment - aligns text in a cell.
            Setbackcolor - sets the background color of a range of cells.
            Setbordercolor - sets the border color of a range of cells.
            Setcell - stores value to an Excel cell
            Setcellfont - changes font attributes of an Excel cell
            Setcolumnwidth - sets column width.
            Setsheet - changes current sheet
            Settextcolor -  sets the text color of a range of cells.
            Setsheetname - changes name of current sheet / creates new sheet
