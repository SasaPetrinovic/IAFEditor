Strings definition
   


In this dialog (menu command Define/Strings) we can define strings which will be used in our applications. Strings are global resources stored in DD7.

The list shows all strings already defined in the Data Dictionary.

New string

Enter new string.

Edit string

Modify an existing string.

Delete string

Remove existing string from the DD7.

Refresh

Load strings into the list.

Export

Store strings to an external file. The file has UTF-8 format. The first three bytes in the export file, byte order mask, are EF BB BF hexadecimal. Each line consists of string identifier, one blank character and string text.

Import

Load strings definition into DD7 from an external UTF-8 file. Such file can be created by any text editor with UTF-8 option ('Save as' command, UTF-8 encoding). Line format is:

string_identifier<blank>text

Example:

ids_1 This is message 1
ids_2 Message 2
ident_nosuchrecord Record with this key does not exist
