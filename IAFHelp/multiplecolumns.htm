Multiple columns
   

This option applies to 'Detail' lineset only.

Normally, a detail line prints below the previous one, down the paper.
But, if the detail line contains few fields, we can define columns to save the paper.

Number of columns

Enter number of columns. Iaf automatically calculates largest number of columns regarding to the current width of the detail line.

To disable columnar report, enter 0 or 1.

Print direction

Across then down - print detail line in a column, then move to the next (right-hand) column and so on, up to the most right-hand column. After that, move down the page, print detail line in the first column, etc.

Down then across - print detail line in a column, move down the page, and so on up to the overflow position. After that, move to the right-hand column, etc.
