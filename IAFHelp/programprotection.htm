Program protection
   

Each Iaf program can be protected from unauthorized access. There are two kinds of protection: by password and by username.

Protection by password

To access a program, user must enter a password. We define password prompt, ie. text which will be displayed on the screen, and password itself, ie. string that has to be entered.
If the user enters wrong password, program stops its execution.
Program protection dialog appears at program initialization phase before database connection.

Protection by username

We can allow up to 10 users to access the program. Only users logged under these usernames can activate this program.
