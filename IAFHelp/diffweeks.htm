DIFF_WEEKS
   

This function returns number of weeks between two date values.

Format

            DIFF_WEEKS(date1, date2)

Result Value Type

            Numeric.

Parameters

·    date1 -
·    date2 -
Note

            Result is a positive integer.

            Algorithm:

            Diff_weeks = diff_days / 7

Examples
  
      $D = DIFF_WEEKS("7-jan-2001","1-jan-2001") !! result is 0
      $D = DIFF_WEEKS("8-jan-2001","1-jan-2001") !! result is 1
      $D = DIFF_WEEKS("10-jan-2001","1-jan-2001") !! result is 1
      $D = DIFF_WEEKS("1-jan-2001","10-jan-2001") !! result is 1
