ADD_DAYS
   

This function adds number of days to a date.

Format

            ADD_DAYS(date, numdays)

Return Value Type

            Date.

Parameters


·     date - date expression
·     numdays - number of days, numeric expression

Examples

  
      $Date = ADD_DAYS(today,5)
      $Date = ADD_DAYS(BEGINCY,-12)
      $Date = ADD_DAYS(today,counter_1)
      $Date = ADD_DAYS(beginy(today),15)
