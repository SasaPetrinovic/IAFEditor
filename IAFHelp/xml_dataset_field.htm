XML_DATASET_FIELD
   

This function reads one field from a XML dataset.


Format

            XML_DATASET_FIELD(xml_variable, node_name)

Result Value Type

            Any field.

Parameters


·     xml_variable - alpha variable containing XML dataset. Variable type is CLOB, longchar (type = S).

·     node_name - node name, alpha literal or alpha variable.

Note

            XML dataset must be established by READ_ALL command, format 3.


Examples

  
      read_all xml_dataset($s,'books.book')
            $fld = xml_dataset_field($s,'title')
            $fld = xml_dataset_field($s,'group_1.gf_1')
      end_read
