Mergeopen
   

This method creates new mailmerge data file.

Format

            Mergeopen(filename)

Parameters

·     filename - an alpha variable or alpha literal. Specifies name of the mailmerge data file. Logical names in filename are allowed. We suppose that there exists a Word document which use this file for mailmerge.

Return Value

            Status, numeric variable.

            Possible values:

            0          Ok
            -1         Document not opened
            
Applied To

            Word object.

Examples

  
      $stat = Word.Mergeopen("c:\tmp\mm.txt")
 

Using Word
