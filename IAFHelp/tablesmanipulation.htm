Tables Manipulation
   

The list shows already loaded tables.

Modify

To modify a table definition, select the table in the list and click Modify

-or-

Double click the table name in the list.

This option should be used very carefully because the modifications apply to Data Dictionary definition only. If some field attributes not loaded properly, such as primary key information, we can change them in this dialog. Also, some fields can be removed from the definition if we do not want to use them in our programs. Table definition in the datasource stays unchanged.

If we load table definition again from the datasource, all these changes will be lost.

Delete


To delete a table definition from the Data Dictionary, select the table in the list and click Delete.

Delete unused tables

To delete all unused tables (the tables not used in programs), click Delete unused tables. The list of unused tables appears on the screen. We can delete all these table definitions or just some of them.

Documentation

In this dialog we can enter description of a table.

Unlock

If Data Dictionary Maintenance program crashes during table modification, the table definition remains locked. To unlock the table, select it and click Unlock.
