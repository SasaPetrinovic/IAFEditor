YEAR_MONTH
   

This function returns 6-digit year and month of a date.

Format

            YEAR_MONTH(date_exp)

Return Value Type

            Numeric.

Parameters

·     date_exp -
Notes

            If date = zero_date or date = NULL, result is 0.

            If date_exp is a 6-digit date, the first two digits of the result will be 19 or 20,
            depending on IAF_CENTURY_START symbol.

Examples
  
      YEAR_MONTH("1-jan-2001") !! result is 200101
      YEAR_MONTH("10-feb-2001") !! result is 200102
