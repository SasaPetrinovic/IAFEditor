Column heading properties
   

In this dialog we can set some attributes for column heading text. These attributes can be different from appropriate field attributes.


Background color

This attribute displays selected color or text Default color if the color is the same as grid default color. Double click brings up a color definition dialog.

Text color

This attribute displays selected color or text Default color if the color is the same as grid default color. Double click brings up a color definition dialog.

Font name

Choose one from offered font names.

Font size

Specifies font size of the text.

Bold

Specify that font weight is 700. Normal weight is 400.

Italic

Specify whether text is italic.

Underline

Specify whether text is underlined.


Horizontal text alignment

Values:

Left
Center
Right

Default is Center.
