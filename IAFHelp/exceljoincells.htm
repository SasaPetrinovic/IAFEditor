Joincells
   

This method combines a range of cells into one large cell (Excel).

Format 1

            Joincells (range)

Format 2

            Joincells (line1, col1, line2, col2)

Parameters


·     range - range of cells in the form STARTING_CELL:ENDING_CELL, alpha variable.
·     line1, col1 - starting cell, numeric.
·     line2, col2 - ending cell, numeric.

Return Value

            Status, numeric variable.

            Possible values:

            0          Ok
            -1         Document not opened
            -2         Invalid range

Applied To

            Excel.

Note

            To display a text into large cell, use STARTING_CELL as a parameter.


Example

  
      $status = Excel.Joincells('A1:F1')
      $lin = 1
      $status = Excel.Joincells(compress_trim('A',$lin,':','F',$lin))
      $status = Excel.Setcell(1,'A','some text')

      $status = Excel.Joincells(1,1,1,6)
