CALL_STOREDPROCEDURE
   

This statement calls a stored database procedure via ODBC.

Format

            CALL_STOREDPROCEDURE procname([arg1][,arg2][,…argN]) [RETURN_VALUE retval]

Where


·     procname - name of the stored procedure. Procedure name is an identifier or a local alpha variable.
·     arg1,..., argN - arguments.
·     retval - return value, integer, optional parameter. Stored procedure can return (via RETURN statement) an integer value to the calling program. This value is usually used to indicate success or failure of the stored procedure.

Notes

            If procedure name contain blanks, use [ and ] separators.

            Unused arguments can be omitted.
            
            An argument can be:

            NULL - null value
            literal,
            variable


Examples

  
      CALL_STOREDPROCEDURE proc1(NULL, 'ABC',, $FLD1)
      CALL_STOREDPROCEDURE owner.proc1(NULL, 'ABC',, $FLD1)
      CALL_STOREDPROCEDURE [procname with blanks]('ABC')
      CALL_STOREDPROCEDURE p1('ABC') RETURN_VALUE $retval
! where $retval is numeric local field

! Local variable can be used as procedure name
      $procname = 'owner.proc1'
      CALL_STOREDPROCEDURE $procname(NULL, 'ABC',, $FLD1)
