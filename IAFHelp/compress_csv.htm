COMPRESS_CSV
   

This function creates a concatenation of many trimmed (left and right) values adding a separator at the end of each input value. Resulting string can be written to a CSV file. Length of separator is one character. If input variables are numeric or date/time ones, default editing will be applied.

Format

            COMPRESS_CSV(separator, val_exp1, val_exp2, …, val_expN)

Return Value Type

            Alpha.

Parameters


·     separator - one character.
·     val_exp1,...,  val_expN - input variables. Maximum number of input values is 512.

Notes


·     All alpha fields are surrounded by quotation marks.
·     One quotation mark in input field will be replaced by two quotation marks.
·     Special function LEADZERO (alpha_field) can be used as an input parameter. This function preserves leading zeros (blanks) when MS Excel loads CSV file.
·     Special function BLANKZERO(num_field) can be used as an input parameter. This function writes out empty (null) value if input field is zero.
·     Special function DATEONLY(datetime_field) can be used as an input parameter. This function writes out date part of a datetime field.
·     Special function MAX255(alpha_field) can be used as an input parameter. This function writes out only 255 characters of a long alpha field.

            LEADZERO, BLANKZERO, DATEONLY and MAX255 are used only in COMPRESS_CSV.


Examples

  
$A = COMPRESS_CSV(';', 'AB C   ', '   AB', 'X') !! result is "AB C";"AB";"X"
$A = COMPRESS_CSV(';', 'abcd','abc"def') !! result is "abcd";"abc""def"
counter_1 = 10
$A = COMPRESS_CSV(';', 'abcd',LeadZero('001'),counter_1) !! result is "abcd";"=""001""";10
$A = COMPRESS_CSV(';', 'abcd','001',counter_1) !! result is "abcd";"001";10
counter_1 = 0
$A = COMPRESS_CSV(';', 'abcd',BlankZero(counter_1),'001') !! result is "abcd";;"001"
