IsLineModified
   

This method determines whether the grid current line has been modified.


Format

            IsLineModified()


Return Value Type

            Numeric.

Return Value

            If a new line is entered, result is 2.
            For existing lines, if any field is modified, result is 1, otherwise 0.

Applied To

            Data-entry grid.

Note

            This function is usually used in 'Line ending actions' of the grid.

            The function takes into account all the fields (columns) except 'Source fields'.

Examples

  
      $status = Table_1.IsLineModified()
      if $status = 2
            Pause 'New line'
      else_if $status = 1
            Pause 'Modified !'
      end_if
