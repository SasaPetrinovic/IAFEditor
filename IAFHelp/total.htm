TOTAL
   

This aggregate function returns sum of numeric_exp for all processed records between two breaks.

Format

            TOTAL(numeric_exp [,cond_exp])

Return Value Type

            Numeric.

Parameters


·     numeric_exp - numeric expression to be sumarrized.
·     cond_exp - conditional expression.


Note

            If cond_exp does not exist, the function unconditionally sumarizes numeric_exp.
            If cond_exp exists, the function sumarizes numeric_exp only from records which calculate cond_exp to TRUE.

            Numeric_exp and cond_exp are calculated in detail lineset or detail calculations.
            This function can be used in reports and batch programs only.

            The following set of instructions, used in a subtotal, is invalid
                        if cond_expr
                                    Source = total (field1)
                        end_if
            because TOTAL summarizes values from all records.

            The correct syntax is:
                        Source = TOTAL(field1, cond_expr)


Examples

  
      TOTAL(field1)
      TOTAL(field1 + field2 + field3)
      TOTAL(field1*field2)
      TOTAL(field1, field2 > 10)
