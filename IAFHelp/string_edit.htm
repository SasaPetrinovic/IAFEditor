STRING_EDIT
   


This function returns string text, simultaneously replacing placeholders by  parameters.

Format

            STRING_EDIT(alpha_var, par1, par2, ..., parN)

Return Value Type

            Alpha.

Parameters


·     alpha_var - an alpha variable, alpha literal or STRING function. The number of placeholders (%) in input string, and number of declared parameters should be the same.
·     par1, par2, ... parN - the variables whose values will replace placeholders. The first variable (par1) will replace the first placeholder, etc.


Notes

            There are two special characters: % (parameter placeholder) and \ (escape character). The parameter placeholder enables inserting program variables into a text using. The escape character allows including into a text some characters which cannot be typed in directly.

            Parameter placeholder

            The percent sign (%) is placeholder. The next string contains two placeholders:

            Invalid value, must be in the range % to %

            Special characters
  
Line feed
\n
Carriage return
\r
Tabulator
\t
Backslash
\\
Percent sign
\%

            The next string contains one placeholder and the escape character:

            The percent is % \%


Examples

  
      errmsg = STRING_EDIT(string(ids_1),acc_1,acc_2)

If the string IDS_1 contains text:
Invalid value, must be in the range % to %
and acc_1, acc_2 contain 100 and 200 respectively, run-time text will be
Invalid value, must be in the range 100,00 to 200,00

If you want to display a percent sign, the escape character have to be used in the string.
