Selectitem
   

This method selects one line in a display-only grid.

Format

            Selectitem (line)

Parameters

·     line - line number, integer. The first line has number 1, etc. There are two special values:

             0         - clears selection,
            -1         - selects all lines.


Applied To

            Grid.

Note


Examples

  
      Table_1.Selectitem(counter_1)
      Table_1.Selectitem(1)   !!Select line 1
      Table_1.Selectitem(0)   !!Clear current selection
      Table_1.Selectitem(-1)  !!Select all lines
