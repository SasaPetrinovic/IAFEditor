READ_NEXT
   

This statement reads next record from a sequential file.

Format

            READ_NEXT file

Where


·    file - the name of the file from datasource dialog. File must be sequential.


Note

            To read group of records, we use READ_ALL statement. READ_NEXT is used under very special circumstances.


Example

  
      REWIND file1
      READ_NEXT file1
      IF EOF = 0  !! if not end of file
            ...
            do_something
            ...
      END_IF
