Mergedata
   

This method stores one field into the mailmerge record buffer.

Format

            Mergedata(variable [,code_page])

Parameters

·     variable - value which will be stored into the record buffer.

·     codepage - if variable is an alpha_exp, this parameter specifies variable's code page. The parameter is very important because MS Office tools (Excel, Word, ...) use UNICODE to represent text data, while text data in programs use, in most cases, single-byte representation.

Return Value

            Status, numeric variable.

            Possible values:

            0          Ok
            -1         Data file not opened
            
Applied To

            Word object.

Examples
  
      $stat = Word.Mergedata($fld)
