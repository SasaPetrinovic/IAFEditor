Loadexternal
   

This method loads external datasource into Excel worksheet.

Format 1

            Loadexternal (textfile_source, destination_cell)

Format 2

            Loadexternal (database_source, destination_cell, sql_command)

Parameters


·     textfile_source - file name in the form: 'TEXT;path'. Keyword TEXT is required.
·     destination_cell - destination cell in the form 'Xn'. This is starting cell where the resulting set will be placed.
·     database_source - database connection string in the form 'ODBC;DSN=dsn;UID=uid;PWD=password'. Keyword ODBC is required.
·     sql_command - SQL command to be run on the ODBC datasource..


Return Value

            Status, numeric variable.

            Possible values:

            0          Ok
            -1         Document not opened
            -2         Invalid range

Applied To

            Excel.

Note


Examples

  
      $status = Excel.Loadexternal('TEXT;c:\temp\File1.csv','A1')
      $status = Excel.Loadexternal('ODBC;DSN=mydsn;UID=myuid;PWD=mypwd', \
                                    'A1', \
                                    'Select * from tbl')

! Example
      $status = Excel.Open()
      if $status = 0
            $status = Excel.Loadexternal('TEXT;c:\temp\File1.csv','A1')
            $status = Excel.Run()
      end_if

Using Excel
