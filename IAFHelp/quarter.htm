QUARTER
   

This function returns a quarter of a year.

Format

            QUARTER (date)

Result Value Type

            Numeric.


Parameters


·    date - date expression (date literal, date variable, date function).


Note

            If date = zero_date or date = NULL, result is 0.
            Otherwise, result is a number in the range 1 to 4.

            If date between 1-JAN and 31-MAR, result is 1.
            If date between 1-APR and 30-JUN, result is 2.
            If date between 1-JUL and 30-SEP, result is 3.
            If date between 1-OCT and 31-DEC, result is 4.


Examples

  
      counter_1 = quarter("1-jun-2007") !! result = 2
      counter_1 = quarter("2007-06-01") !! result = 2
