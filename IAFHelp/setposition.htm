Setposition
   

This method sets the new position and/or new size of an object.

Format

            Setposition(x,y,width,height)

Parameters


·     x - x position in pixels, integer
·     y - y position in pixels, integer
·     width - width in pixels, integer
·     height - height in pixels, integer

Applied To

            All objects.

Note

            Unused parameters can be omitted.

            If you use this method with an object X, Y denotes client coordinates (relative to the left upper corner of the form). But, if you use this method with FORM, X and Y denotes screen coordinates.

Examples

  
      obj.Setposition(100,10) !! changes x, y
      obj.Setposition(,10)    !! changes y
      obj.Setposition(100,10,200,50) !! changes x,y,width,height
      form.Setposition(200,100)      !! screen coordinates
