Insertiteminlinecombo
   

This method inserts an item into a grid combobox if comboboxes in every line have different content.

Format


            InsertItemInLineCombo(line, data, text_label)


Parameters

·    line - grid line containing combobox, integer. Lines are counted from 1.
·    data - data to be stored into this combobox. When user selects an item, the data will be moved to appropriate variable.
·    text_label - text to be displayed.

Applied To

            ComboBox in grid.

Note

            Every item has two parts: text and associated data. All texts are displayed by the combobox. When user selects the item, associated data will be moved to program variable.


Example

  

! Example 1: Fill-in a combo box in the first line
      combo_col1.InsertItemInLineCombo(1,'01','Item 1')
      combo_col1.InsertItemInLineCombo(1,'02','Item 2')
      combo_col1.InsertItemInLineCombo(1,'03','Item 3')
      combo_col1.InsertItemInLineCombo(1,'04','Item 4')
      combo_col1.InsertItemInLineCombo(1,'05','Item 5')

! At run-time, if user selects "Item 3", value "03" will be moved to
! the variable defined by "Destination = prog_var"
 
! Example 2: fill-in combo box from the table
      combo_col1.Deleteallitems(1)
      select_all * from tbl1 where fld1=100
            combo_col1.InsertItemInLineCombo(1,tbl1.text,tbl1.data)
      end_read
