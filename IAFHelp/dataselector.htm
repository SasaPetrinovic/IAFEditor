Data Selector
   

We can consider data selector as a complex object which enables multiple selection. In fact, the data selector is a dialog displaying two-columns list. The first column displays data, and the second one displays the text (data description). Using standard Windows actions (click, Ctrl/Click, Shift/Click) user selects some items and clicks Ok button. All the selected data will be moved into output array. Number of selected items will be stored in appropriate output variable. Later, in our program, we can use these variables in IF statement.

To insert new data selector on the form, click button Dataselector in toolbox or choose menu command Objects/Data selector. Move the cursor to the desired position,  then click. Definition dialog appears on the screen.
