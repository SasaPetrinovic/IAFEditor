CALL_PROGRAM
   

This statement calls an external executable program (EXE).

Format

            CALL_PROGRAM [/WAIT][/NOWAIT][/NONIAF] program_name

Where


·     program_name - name of the executable program (must be an alphanumeric literal or an alpha field)

Qualifiers

            /WAIT               Syncronous call, blocks calling program until execution of new program is complete.
            /NOWAIT          Asyncronous call, continues to execute calling program concurrently with new one.
            /NONIAF           New program was not generated by Iaf. Default is Iaf program.

Note

            If neither /WAIT nor /NOWAIT appears in the line, Iaf use an internal algorithm to
            choose the type of the call: in button click actions, or toolbar_button click actions,
            call will be asyncronous. Otherwise, it will be syncronous (procedures,
            calculations, etc...).

            With /NONIAF, command line parameters can be used, i.e.
            CALL_PROGRAM /NONIAF "noniaf_program arg1 arg2 arg3"

Examples

  
      CALL_PROGRAM "pgm1"
      CALL_PROGRAM /WAIT "c:\myappl\pgm"
      CALL_PROGRAM /WAIT/NONIAF "appl_exe:pgm"
      CALL_PROGRAM /WAIT/NONIAF "pgm arg1 arg2"
      $cmd = "vmslogin c:\appl"
      call_program $cmd
