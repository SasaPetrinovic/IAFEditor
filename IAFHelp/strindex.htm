STR_INDEX
   

This function returns the position of the first occurrence of a character in an alphanumeric string.

Format

            STR_INDEX(alpha_var, char)

Return Value Type

            Numeric.

Parameters


·     alpha_var - input alpha variable.
·     char - a character to search for, alpha literal or alpha field. If you specify an alpha variable, only the first character will be used in search process.

Note
            
            If character is not found, result is –1, otherwise, result is between 1 and length of
            alpha_var.

            The leftmost character has position 1.

Examples

  
      $res = STR_INDEX("ABCD","B") !! result is 2
      $res = STR_INDEX("ABCD","X") !! result is -1
      $res = STR_INDEX("ABCD","BX") !! result is 2

      $AVAR = "BXY"
      $res = STR_INDEX("ABCD",$AVAR)      !! result is 2
