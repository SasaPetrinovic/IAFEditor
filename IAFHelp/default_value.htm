DEFAULT_VALUE
   

This function returns default value used in CREATE TABLE statement.

Format

            DEFAULT_VALUE(field)

Return Value Type

            Alpha, numeric or date.

Parameters

·     field - a table field.

Notes

            If default value does not exist, result is initial value of the field (zero, space, zero_date, zero_time).

Example
  
! Consider the following SQL statement:
! CREATE TABLE tbl (
!     col1 int not null primary key,
!     col2 varchar(5)   default 'abcd',
!     col3 decimal(9,2) default 12.34,
!     col4 decimal(9,2) default -1.00,
!     col5 varchar(80)  default null)

      $defa = DEFAULT_VALUE(tbl.col2)     !!result='abcd'
      acc_1 = DEFAULT_VALUE(tbl.col3)     !!result=12.34
