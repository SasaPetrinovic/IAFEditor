Sumcol
   

This method returns sum of values in a column of a grid.

Format

            Sumcol(column_number)

Parameters


·     column_number - numeric literal or numeric variable representing column. If this column contains non-numeric fields, result will be zero. The first column is 1, second one is 2, etc.

Applied To

            Grid.

Examples

  
      acc_1 = table_1.sumcol(1) !! sum values in column 1
      acc_1 = table_1.sumcol(9) !! sum values in column 9
