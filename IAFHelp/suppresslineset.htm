Suppress lineset
   

A lineset is a part of a report: page header, subheader, detail line, subtotal.

For each lineset you can define a condition which specifies when this lineset is to be suppressed.

Lineset

In Lineset column you see all linesets in the report.

Condition

In Condition column, you have to enter the suppress condition. The maximal number of characters is 70.
If a lineset always prints, this cell must be empty.

Example:

acc_1 = 0 and acc_2 = 0

The above condition need to be read like this: when acc_1 has value 0 and acc_2 has value 0, entire lineset (detail line, for example) will not be printed.

What if the condition has more than 70 characters?
In that case, a local variable can be defined, let's say $suppls, who can have two values: 1 or 0. In a field, who belongs to that lineset, the value of the variable need to be calculated.

Then, in Condition column you can write

$suppls = 1       or         
$suppls = 0       depending on how the value is calculated.

Column B

If you want to just skip a lineset, set check-box in column B. In this case, suppress lineset condition will be checked at the very beginning of the lineset. If the condition is met, lineset will be skipped and calculations in lineset fields are not performed. This option accelerates program execution.

Column E

If you want to check suppress lineset condition at the end of the lineset, immediately before lineset printing, set check-box in column E. Lineset calculations are performed. Variables which appears in the condition need not reset in the same lineset.


Note

In the first line, the text 'Whole report' appears. If you enter condition in this line, all linesets will be suppressed. All calculations and actions are performed, but there is no print output.
