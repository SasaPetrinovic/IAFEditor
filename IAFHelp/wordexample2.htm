Using Word - Example 2
   

Open existing Word file (template), store data into three bookmarks "name", "date" and "amount", and save document under new name.

$wstat = Word.Open("docdir:template.doc")
if $wstat = 0
      $wstat = Word.Setbookmark("name",$name,1250)
      $wstat = Word.Setbookmark("date",today)
      $wstat = Word.Setbookmark("amount",edit(acc_1,"---.---.--9,99"))
      $wstat = Word.Run("save","outdir:newname.doc")
else
      Pause "Error openning input file"
end_if

After the Run instruction, file outdir:newname.doc is created and Word is active.


Important note

After Setbookmark method, using Run without some of 'save' options is not recommended.
An user could activate, accidentally,  'Save' command in Word and overwrite original template by active document. Active document does not contain bookmarks, and next time the program will fail because it will not find any bookmark. Therefore, the following combination is not recommendable:

...
$wstat = Word.Setbookmark("somebm",variable)
$wstat = Word.Run()
...
