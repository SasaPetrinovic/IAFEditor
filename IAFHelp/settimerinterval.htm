Settimerinterval
   

This method sets new timer interval value, in milliseconds.

Format

            Settimerinterval(time_interval)

Parameters


·     time_interval - numeric integer variable or numeric literal.

Applied To

            Timer.

Note

            If the timer is active, this method stops the timer, sets new value and starts it again. If timer was stopped, this method just sets new value.

Example

  
      Timer_1.Settimerinterval(10000)     !! 10 sec
