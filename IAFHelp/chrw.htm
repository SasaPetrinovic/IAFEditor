CHRW
   

This function returns the wide character (Unicode) that has the code value specified by numeric_exp.

Format

            CHRW(numeric_exp)

Return Value Type

            Alpha.

Parameters


·     numeric_exp - numeric literal or numeric field in the range 0 to 65535.

Examples

  
      $W = CHRW(9) !! result is a TAB haracter
      $W = CHRW(27) !! result is an ESC character
      $W = CHRW(65) !! result is "A"
      $W = CHRW(1040) !! result is cyrillic "A"
