Setcell
   

This method stores value to an Excel cell.

Format 1

            Setcell(range, variable [,codepage])

Format 2

            Setcell(line, column, variable [,codepage])

Parameters


·     range - cell in the form COLUMNline, alpha variable. Examples:  A1, A2, B1, X1, AA1, AA2, AA100, ...

·     line - row, numeric value, integer. The first line is 1.

·     column - column, numeric value, integer. The first column is 1. Instead of column number, an alpha literal containing column header can be used: 'A', 'B', ..., 'AA', 'AB', ...

·     variable - value which will be written to the cell. If the value starts by '=' (equal sign) a formula will be stored to the cell.

·     codepage - if variable is an alpha one, this parameter specifies variable's code page. The parameter is very important because MS Office tools (Excel, Word, ...) use UNICODE to represent text data, while text data in programs use, in most cases, single-byte representation. Default is 1252 (Western).


Return Value

            Status, numeric variable.

            Possible values:

            0          Ok
            -1         Document not opened

Applied To

            Excel object.

Note

            If a formula has more than one argument, arguments are separated by comma:

            Excel.Setcell(1,1,'=SUMIF(E1:E3,"A",F1:F3)')
            Excel.Setcell(1,1,'=SUMIF(Sheet1!E1:Sheet1!E3,"A",Sheet1!F1:Sheet1!F3)')
            

Examples

  
      ! Format 2
      $stat = Excel.Setcell(1,1,acc_1)
      $stat = Excel.Setcell(1,2,today)
      $stat = Excel.Setcell(1,"B",today) !! same as above
      $stat = Excel.Setcell(1,3,textdata,1250)
      $stat = Excel.Setcell(100,'A',"=SUM(A1:A10)") !! formula
      $stat = Excel.Setcell(100,'A','=SUMIF(A1:A10,"A",B1:B10)') !! formula

      $stat = Excel.Setcell('A1',acc_1) !! Format 1
      $stat = Excel.Setcell('B1',today)
      $stat = Excel.Setcell('C1',textdata,1250)
      $stat = Excel.Setcell('AA1',acc_2)
 

Using Excel
