Copy program
   

Create a copy of a program with, possible, renaming.


From

            Program to be copied (input program).

To

            Program to be created (output program). Is we copy programs within the same application, this name must be different from input program name.

Output application

            If our Data Dictionary contains many applications, we can copy (move) a program from one application to another one.

Copy

            Make new copy of a program.

Move

            Move (rename) program from one application to another one. If the both applications are the same, this is an ordinary rename operation.

Note

            Form painter will be activated immediately, if we copy a program within the same application.

Error messages

            If output program name is blank or invalid, error message is: "Invalid program name".

            If the output program with the same type already exists, error message appears on the screen: "Program already exists".

            If the output program with different type already exists (input is interactive one, output is report, and vice versa), error message is "Non-unique program name".
