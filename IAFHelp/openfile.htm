Open
   

This method opens/creates a text file.

Format

            Open (filename, openmode [,texttype])

Parameters


·     filename - an alpha variable or alpha literal. Specifies name of the text file. Logical names in filename are allowed.
·     openmode - mode, alpha variable or alpha literal. Possible values are: "new", "append", "read_only".

            New     - Creates a new, empty file.
            Append - Opens an existing file and positions file pointer to the end. If file does not exist, it will be created.
            Read_only - Opens file for reading. The file must exist.


·     texttype - type of text, alpha variable or alpha literal. Possible values: "ansi", "utf-8", "utf-16", "utf-16big".  Default is "ansi". Valid only if openmode="new". After file creation, byte-order mark will be inserted at the beginning of the file, if texttype is "utf-8", "utf-16" or "utf-16big". If openmode="append" or "read_only", this parameter is ignored.

            Ansi      - Ordinary ANSI text, one character is one byte long.
            UTF-8   - Unicode text, a character is represented by multi-byte sequence.
            UTF-16 - Unicode text, each character is two bytes long.
            UTF-16BIG - Unicode text, each character is two bytes long, big endian.

            Byte-order marks are:
  
for UTF-8
0xEF 0xBB 0xBF  (3 bytes)
for UTF-16
0xFF 0xFE  (2 bytes)
for UTF-16BIG
0xEF 0xFF  (2 bytes)


Return Value

            Status, numeric variable.

            Possible values:
  
0
Ok
-1
File already open
-10
Logical name not found
-11
Invalid argument 'openmode'
> 0
File input/output error (see errno constant)

Applied To

            File object.

Note


Examples

  
      $stat = File.Open("c:\data\test.txt","new","utf-16")
      $stat = File.Open("datadir:test.txt","append")
      $stat = File.Open($file_name,"read_only")
      if $stat = 0
            ...
      end_if
