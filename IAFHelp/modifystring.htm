Modify string
   


An identifier is a sequence of English letters, digits, underscore character(_)  and dollar sign ($). The first character must be a letter (A to Z). Maximum number of characters in an identifier is 30. Identifiers are case-insensitive, so IDS_1 and ids_1 denotes the same string.

Examples:

IDS_1
IDS_2
ids_3
ids_nosuchrecord
ident_invrange

Text is any text stored in Unicode UTF-8 format.

Examples:

This is message 1
Invalid date
Invalid range


These strings can be displayed using STRING function.

There are two special characters: % (parameter placeholder) and \ (escape character).
The parameter placeholder enables inserting program variables into a text using STRING_EDIT function. The escape character allows including into a text some characters which cannot be typed in directly.

Parameter placeholder

The percent sign (%) is placeholder

Invalid value, must be in the range % to %

Such strings can be displayed using STRING_EDIT function.

Example: if we want to display two-lines message, we can enter

Two-lines message\r\nThis is the second line.

At run-time, the ERRMSG statement will display above text in the form

Two-line message
This is the second line.

Example: string with placeholders

Invalid value, must be in the range % to %

To display above text, we should use STRING_EDIT function with two parameters, like
STRING_EDIT(IDS_1, acc_1, acc_2).

Example: string with placeholders and fixed percent sign

Invalid value, must be less than %\%

At run-time, if value of par1 is 60, this text will be

Invalid value, must be less than 60,00%

Let's remember: \% is not a placeholder. This is a special character denoting percent sign.
