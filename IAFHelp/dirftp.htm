Dir
   

This method stores list of the files on the server into a text file on local machine.

Format

            Dir(pattern[, filename])

Parameters


·     pattern - an alpha variable or alpha literal. Specifies file search pattern with wildcards.

·     filename - an alpha variable or alpha literal, optional. Specifies text file name. Logical names in file name allowed. If omitted, the first file entry will be stored into the output variable.

            Record has four fields:

            - file name
            - file type : "D" - directory, "F" - file
            - date modification in the form YYYY-MM-DD HH:MM:SS
            - file size in bytes

            Fields are separated by tabulator (TAB = CHR(9)).

Return Value

            Alpha variable. If connecting was successful, value is blank. Otherwise, the variable holds an error message.


Applied To

            Ftp object.

Note


Examples

  
      $msg = Ftp.Dir('*.*','c:\temp\list.txt')
      if $msg <> SPACE
            Pause 'Error: ' | $msg
      end_if
      $msg = Ftp.Dir('*.exe','c:\temp\list.txt')
      $msg = Ftp.Dir('*.exe','tmpdir:list.txt') !!logical name

      $msg = Ftp.Dir('*.exe') !!$msg contains the first file entry
 

Using FTP
