CHECK
   

This statement controls entered value.

Format

            CHECK cond_exp

Where


·     cond_exp - a conditional expression (which determines control under entered field)


Note

            CHECK can be used after a DESTINATION statement.
            CHECK must be followed by a ERRMSG or WARMSG.


Example

  
      DESTINATION = field1
      CHECK field1 < 100
      ERRMSG = "Entered value must be less than 100"
