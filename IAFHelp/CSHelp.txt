.topic 200

The Z-Order defines the position (depth, layer) of the object when many passive objects overlaps each other. The object with Z-Order = 0 is on the top and overlaps all other objects.
Value of -1 denotes one layer bellow top objects, -2 denotes next lower layer, etc.

The passive objects are:

	Text
	Line
	Rectangle
	3D Panel
	Ellipse
	Image
	Frame
