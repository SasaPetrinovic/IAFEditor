GZIP_DECOMPRESS
   

This function decompresses gzip (.gz) file.

Format

            GZIP_DECOMPRESS(input_file, output_file)


Return Value Type

            Numeric. If the function fails result is -1, otherwise, result is 0.


Parameters


·     input_file - input GZIP file (.gz), alpha literal or alpha variable.
·     output_file - output (decompressed) file, alpha literal or alpha variable.

Note

            GZIP_DECOMPRESS uses ZLIB library written by Jean-Loup Gailly and Mark Adler.


Examples

  
      $numstat = GZIP_DECOMPRESS("a.gz","a.txt")
