REPLACE
   

This function replaces all occurrences of one string by another string.

Format

            REPLACE(alpha_exp, from_str, to_str [,collation])

Return Value Type

            Alpha.

Parameters


·     alpha_exp - input field, alpha.
·     from_str - string to be replaced.
·     to_str - string that replaces from_str.
·     collation - alpha literal denoting collation. This parameter is mainly used to specify case-insensitive option.

Note

            Output string could be truncated if the output buffer is too small.

Examples

  
      $outstr = REPLACE($INPSTR,'xx','xyz') !! replace xx by xyz
If the $INPSTR contains 'abcxxyy', the result will be 'abcxyzyy'

      $outstr = replace($inpstr,chr(9),' ') !! replace TABs by spaces

      $outstr = replace($inpstr,alltrim($from),alltrim($to))

       ! FROM_STR is case-insensitive: abc, ABC, Abc, etc.
      $outstr = replace($inpstr,'abc','123','english_CI')
