STR_FILL
   

This function fills alpha variable by 'fill string'.

Format

            STR_FILL(fill_string[, repeat_count])

Return Value Type

            Alpha.

Parameters


·     fill_string - alpha literal or alpha variable.
·     repeat_count - integer, optional. Number of times the fill_string is repeated. If repeat_count is omitted, whole output variable will be filled by fill_string.

Notes

·           If fill_string is an alpha variable, function first removes trailing blanks and then uses remaining text as a fill_string.

·           If fill_string is an alpha literal, string is used without changes.

·           If output variable is a CLOB (field type = S), repeat_count is required.

·           Commands
            $A = str_fill('')       (empty fill_string)
            $A = str_fill(' ')      (fill_string = space)
            $A = str_fill(' ',0)    (repeat_count = 0)
            have the same effect as
            INIT $A
            but INIT is slightly faster.
            

Examples

  
      $res = STR_FILL('-')
      $res = STR_FILL('-',10)
      $res = STR_FILL('-=*')
      $res = STR_FILL('-=*',5)

      $fill= 'abc'
      $res = STR_FILL($fill)  !! fill_string is 'abc'

      $fill= 'ab'
      $res = STR_FILL($fill)  !! fill_string is 'ab'
