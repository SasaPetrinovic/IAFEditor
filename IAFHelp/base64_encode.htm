BASE64_ENCODE
   

This function transforms input data to a base64 format.


Format

            BASE64_ENCODE(input_data, data_length)

Return Value Type

            Alpha in base64 format.

Parameters

·     input_data - input data, alpha or binary.
·     data_length - number of input bytes.


Note

            Base64 data requires 33% more space than original data.

            Formula for output length calculation is:

                        num_of_base64_chars = ((input_length+2)/3)*4

Example
  
            $out64 = BASE64_ENCODE($inp,str_length($inp))
