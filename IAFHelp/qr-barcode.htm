QR barcode
   


QR (Quick Response) is a two-dimensional bar code. A QR code consists of black modules (square dots) arranged in a square grid on a white background. Minimum barcode size is 21 x 21 modules, maximum one is 177 x 177 modules.

The symbol can store up to 2953 bytes, or 4296 alphanumeric characters, or 7089 digits. To encode letters from any code page, data should be in UTF-8 format.


Error correction level

            Error correction algorithm allows the barcode to avoid various damages of the symbol.

            Low      - 7% of codewords can be restored
            Medium- 15% of codewords can be restored
            Quartile- 25% of codewords can be restored
            High     - 30% of codewords can be restored

            Default value: Medium.

Fixed dimensions

·           On a label or pre-printed form barcode dimensions should be fixed.

·           If data field length does not change considerably, barcode dimensions should be fixed as well.
      Example: data field contains 60 digits, error correction level = High. QR algorithm discovers that number of modules is 33. If module width is 1mm, total width of barcode in painter should be 33mm. Such barcode will be perfectly scanned.

·           If data field length changes significantly, barcode dimensions should be variable. In this case, the attribute 'Horizontal alignment' is important while 'Vertical alignment' is ignored.

      Horizontal alignment = left: top left corner of barcode is locked. Barcode increases to the right and down.
      Horizontal alignment = right: top right corner of barcode is locked. Barcode increases to the left and down.
      Horizontal alignment = center: top center point of barcode is locked. Barcode increases to the left, right and down.
      

Units of measurements

            Millimeters or inches.

Minimum module width

            A module is the smallest unit of the barcode symbol. If modules are too small, barcode is not scannable. The minimum value is 0,4 mm.

Minimum barcode width

            Display only parameter. Minimum size should be 30 mm.

Maximum barcode width

            If barcode has variable dimensions, enter maximum width.

Micro QR

            To decrease bar code size select this option.

Example

            Display characters from code page 1250.

            $x = compress_trim(fld1, fld2, ...)
            $x = convert($x,1250,65001)      !! convert to UTF-8
