Subforms
   

Add

Adds new subform.

Delete

Deletes existing subform(s).

Layout

Shows positions of all subforms if program has more than one subform.

Arrange

Adjusts positions on the screen of subforms.

Swap

Changes mutual position of two subforms.

Example: if we swap subform 7 and subform 2, subform 2 becomes subform 7 and subform 7 becomes subform 2.

Move

Move one subform from one position to the another one.
For example, move subform 7 before subform 1. In this case, subform 7 becomes subform 1, subform 1 becomes subform 2, etc.

Insert

Inserts empty subform before selected subform.

Include

Inserts subform from another program.

Quit subform

Abandons all changes on current subform.
