Open
   

This method starts MS Word and opens existing document.

Format

            Open(filename [,readonly])

Parameters

·     filename - an alpha variable or alpha literal. Specifies name of the existing Word document. Logical names in filename are allowed.

·     readonly - second parameter, if used, is alpha literal containing text "read_only". In that case, document cannot be changed.

Return Value

            Status, numeric variable.

            Possible values:

            0          Ok
            -1         Cannot activate Word
            -2         Document already opened
            -3         File not found
            -4         Logical name not found
            -5         File locked by another user

Applied To

            Word object.

Note

            To release current Word document use Close or Run method.

Examples

  
            Variants

            Open(filename)
            Opens existing file.
            Open(filename,"read_only")
            Opens existing file, changes not allowed

      $stat = Word.Open("c:\doc\test.doc")
      $stat = Word.Open("docdir:test.doc")
      $stat = Word.Open($file_name)
      $stat = Word.Open("c:\doc\test.doc","read_only")
      if $stat = 0
            ...
      end_if
