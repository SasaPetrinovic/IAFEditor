SOURCE
   

This statement defines fields for display on the screen or  on report.

Format

            SOURCE = val_exp
                        [/FILL: char]
                        [/FIRST]
                        [/MOVE]
                        [/RESET_ON_BREAK]

Where

·     val_exp - the expression which is displaying
·     char - ASCII character

Qualifiers

            /FILL replaces left blanks by char.

            /FIRST is used in reports only. Field with this qualifier is showed in the report only if value changes or if there is a jump to new page.

            /MOVE determines field for transfer from a subprogram to main program.

            /RESET_ON_BREAK is used in reports only, for page counter initialization. Can be used only with internal field PAGE_COUNTER. When page header break occurs, page_counter will be set to 1.

Note

The statement defines the source of the information only (field1, counter_1, acc_1, today, ...), but the form of the presentation (text, image, barcode) should be specified in corresponding object (property name = Source field style).


Examples

  
      Source = page_counter
      Source = page_counter /reset_on_break
      Source = acc_1 /fill: *
