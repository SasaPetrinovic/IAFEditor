Graph
   

We use a graph to display data graphically.

To insert new graph on the form, click button Graph in toolbox or choose menu command Objects/Graph. Move the cursor to the desired position,  then click. This is the top left corner of the graph. Drag to the right and bottom and release the button. Graph definition dialog appears on the screen.

Graph type

We can plot three types of graphs: histogram (bar chart), pie chart or line chart.

Histogram plots data as vertical columns with two axes: X and Y.
X axis is called Category axis.
Y axis is called Values axis.

Chart source data

In this dialog we specify data to be displayed.

Data series is a group of related data. In our case, all data belonging to one data series are stored in one array. Each data series has two components: data series name and data series values.

Data series name is an alpha literal or alpha variable.
Data series values represent a numeric array containing values.

To define data series, specify name, the array containing values and click Add.
The series name appears in the list.

To change a defined data series, select appropriate series in the list. Change name or/and values and click Update.

To remove a data series, select appropriate series in the list and click Remove.

Note: pie chart has one data series only.

Category (X) axes labels

Alpha array, number of elements = number of categories.

Number of categories

Integer, numeric literal or variable. This value determines number of elements in data series values array.


Examples

1/
If we want to display number of sold books by months for current year (number of series = 1), we should enter the following:

Series name = "Current year"
Values = $nbooks, where is $nbooks numeric array, number of elements=12.
Category (X) axes labels = $lab, where is $lab alpha array, number of elements=12, element values are: "Jan", "Feb", "Mar", "Apr", "May", "Jun", "Jul", "Aug", "Sep", "Oct" ,"Nov", "Dec".
 
Number of categories = 12

2/
If we want to display number of sold books by months for current and previous year (number of series = 2), we should enter the following:

Series 1
Series name = "Current year"
Values = $nbooks_cy, where is $nbooks_cy numeric array, number of elements=12.

Series 2
Series name = "Previous  year"
Values = $nbooks_py, where is $nbooks_py numeric array, number of elements=12.

Category (X) axes labels = $lab, where is $lab alpha array, number of elements=12, element values are: "Jan", "Feb", "Mar", "Apr", "May", "Jun", "Jul", "Aug", "Sep", "Oct" ,"Nov", "Dec".
 
Number of categories = 12
