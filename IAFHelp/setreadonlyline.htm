Setreadonlyline
   

This method disables fields editing in one line of a data-entry grid.


Format

            Setreadonlyline(line, flag)


Parameters


·     line - grid line, integer.
·     flag - integer. If flag <> 0, line is read-only and the fields can not be changed. If flag = 0,  this is standard line. Initially, all displayed lines are standard ones.

Applied To

            Applies to a data-entry grid.


Note


See also Setreadonly
