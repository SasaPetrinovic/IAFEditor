Isenabled
   

This method returns the value of enable state of an object.


Format

            Isenabled()

Return Value Type

            Numeric.

Return Value

            If the object is enabled, result is 1, otherwise result is 0.

Parameters

            None.

Applied To

            All objects.

Examples

  
      $status = button_1.Isenabled()
      if button_3.Isenabled() = 1
            Pause "Enabled !"
      end_if

      $objname = 'image_1'
      $status = $objname.Isenabled()
