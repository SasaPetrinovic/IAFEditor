Autocomplete
   

In this dialog parameters for autocomplete operation can be entered .

Strings source


Array or a lookup dialog.


Array holding autocomplete strings

Enter name of array containing strings needing to be displayed in autocomplete list. The array should be filled before the field gets the focus.

Number of strings

Numeric literal or variable denoting real number of strings in the array.

Lookup

Name of lookup dialog used in autocomplete operation.

SELECT statement in the dialog should be in the form

SELECT text_field FROM mytable WHERE text_field LIKE :LOOKUP_FIELD ORDER BY text_field

SELECT TOP n text_field FROM mytable WHERE text_field LIKE :LOOKUP_FIELD

SELECT id,name FROM mytable WHERE id LIKE :LOOKUP_FIELD ORDER BY id

Minimum number of characters

Minimum number of characters a user has to type before the autocomplete operation activates.

Delay

Number of milliseconds the autocomplete waits after a keystroke to activate itself.
