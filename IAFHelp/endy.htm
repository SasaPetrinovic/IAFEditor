ENDY
   

This function returns a date representing end of the year of date_exp.

Format

            ENDY(date_exp)

Return Value Type

            Date.

Parameters


·    date_exp - input date or datetime.

Examples

  
      $E = ENDY("12-mar-2001") !! result is 31-dec-2001
