Lookup dialog
   

/LOOKUP qualifier displays lookup dialog at run-time.

Format:

Destination = field/LOOKUP: SQL_statement [HEADING heading_text] [TITLE title_text]
Destination = field/LOOKUPA: SQL_statement [HEADING heading_text] [TITLE title_text]

or

Destination = field/LOOKUP: lookup_name
Destination = field/LOOKUPA: lookup_name

Lookup name is name of lookup dialog defined by Define lookup dialog painter command. This dialog enables us to define complex lookup dialog.

If you use /LOOKUP, the dialog will be displayed if you click appropriate magnifier or press CTRL+A when the field has the input focus.
If you use /LOOKUPA, the dialog will be displayed automatically when the field gets input focus.

SQL_statement is a standard SELECT statement, specifying fields to be displayed in the dialog.

SELECT fld1, fld2 FROM tbl ORDER BY fld1
SELECT DISTINCT fld1, fld2 FROM tbl ORDER BY fld1
SELECT fld1, fld2 FROM tbl WHERE fldx=1 ORDER BY fld1
SELECT fld1, fld2 FROM tbl WHERE fldx=:$parameter ORDER BY fld1
SELECT fld1, fld2 FROM tbl WHERE fldx=:$parameter group by fld1,fld2 ORDER BY fld1,fld2
SELECT fld1, fld2,count(*) into :$cnt FROM tbl group by fld1,fld2 ORDER BY fld1,fld2

Table specified in FROM clause need not be selected in Datasource dialog.

HEADING denotes column headers, alpha literal or alpha field. Column headers are delimited by SPACE. If you want to define another delimiter, enter the header text in the form "SEP=, Col head1, Col head2". In previous example, delimiter is comma.

SELECT fld1, fld2 FROM tbl ORDER BY fld1 HEADING "Col1 Col2"
SELECT fld1, fld2 FROM tbl ORDER BY fld1 HEADING $hdr
SELECT fld1, fld2 FROM tbl ORDER BY fld1 HEADING "Col1_longtext Col2"
SELECT fld1, fld2 FROM tbl ORDER BY fld1 HEADING "SEP=, Col hd1, Col hd2"

TITLE defines dialog caption, alpha literal or alpha field.

SELECT fld1, fld2 FROM tbl ORDER BY fld1 HEADING "Col1 Col2" TITLE "Table TBL"
SELECT fld1, fld2 FROM tbl ORDER BY fld1 HEADING "Col1 Col2" TITLE $title
SELECT fld1, fld2 FROM tbl ORDER BY fld1 HEADING $hdr TITLE $title

Note

To move the first column to the destination field, select a line then click Move button or press Enter key.
