Addline
   


This method adds one line to a grid. The line will be added after last displayed one.

Format

            Addline(field1, field2, ..., fieldN )


Parameters


·     field1 - field to be displayed in the first column.
·     field2 - field to be displayed in the second column.
·     fieldN - field to be displayed in the last column.


Applied To

            Grid.

Note

            Number of parameters must be equal to number of the columns.

Example

  
      Table_1.Addline(counter_1, acc_1, today, systime)
