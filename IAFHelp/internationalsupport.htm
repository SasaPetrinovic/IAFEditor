International Support
   


International support (multilinguality) enabled

If you plan to create a multilingual application, this option should be checked.


Default language

This is the language you use to develop the applications. If you do not want multilingual applications, leave this field empty or enter the text just for your information. But, if you intend to develop multilingual applications, this field is required. In that case, you can use English name for your language (English, France, German, ...) or enter your local name, for example 'Deutsch' (for German). This parameter can be changed at any stage of application development.
