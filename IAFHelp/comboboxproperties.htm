Combobox properties
   

Object name

When the new combobox is created, generator automatically assigns the name Combobox_nnn, where nnn = 1, 2, …
Object name can be maximum 30 characters long. Allowed characters are A-Z, 0-9, _ and $.
First character must be a letter. Object name is case-insensitive. Object name can be used in Methods.

Member of group (information only, cannot be changed)

If the object is a member of a group, the name of this group is displayed. An object can be the member of one group only.

Field name (information only, cannot be changed)

Name of the destination field. In IAF, data entered via keyboard must be recorded somewhere: into a table field, a local field, a counter, an accumulator. This place we call destination field.

Show combo as Listbox

If there is enough space on the form, a list box can be created instead of a combobox. Just increase object height and select this check box. By default, vertical scroll bar is hidden. If the list box contains number of items that can not fit the visible space, vertical scroll bar will automatically appear. See examples in InsertItemEx.

Visible

Determines whether the object is visible when the subform is first displayed.

Enabled

Determines whether the object is disabled when the subform is first displayed or initialized. When the object is disabled, mouse and keyboard input are ignored.


Display data and text

Normally, the combobox shows text only. If you want to see corresponding data which will be stored into the field, set this checkbox.

Read only

When this attribute is set, combobox looks like standard one, mouse click can be accepted, but selection cannot be changed.

Sequence number

If the combobox has sequence number 0, click is allowed if the combobox is enabled.

If the combobox has nonzero sequence, click will be accepted only if all objects with the smaller sequence number are processed.


Font name

Choose one from offered font names.

Font size

Specifies font size of the text.

Bold

Specify that font weight is 700. Normal weight is 400.

Italic

Specify whether text is italic.

Underline

Specify whether text is underlined.


Background color

Text color

Color of text.

Tooltip text

Tooltip text is a text that will be displayed when the cursor hovers above the combo box. See Tooltip text definition.

Invalid data error message

This text will be displayed if data field contains invalid value.
