Table (grid)
   

A table (grid) is most complex object in the painter because the table contains another objects: source field, destination field, check box, combo box or button.



Display-only table is a table containing source fields, read-only check boxes or buttons. In other words, such table (grid) just displays data.

Data-entry table contains at least one destination field, check box or combo box.

Maximum number of columns in 100.

Insert new table (grid)

To insert new table on the form, click button Table(grid) in toolbox or choose menu command Objects/Table(grid). Move the cursor to the desired position,  then click. A new table appears on the screen having just one column.

Insert new column

Right click on the top (column) heading of an existing column and select 'Insert left' or 'Insert right' option in the context menu. New column has the same width as current column.


Insert new field

Click appropriate button in toolbox (Source field, Destination field, check box, combo box or button) or choose corresponding menu command. Place the cursor to the first line (the line just below top heading) and click.


Modify column heading

Double click on the top (column) heading and enter the text in the dialog which appears on the screen.
