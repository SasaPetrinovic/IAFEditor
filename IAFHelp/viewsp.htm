Status bar

These options define visibility status of some parts of the generator.

Grid

If this option is checked, grid is displayed on the working area. The grid allows easier objects positioning.

Snap to grid

If the grid is displayed and 'Snap to grid' is checked, newly created object will be aligned to the grid.

Refresh

Redraws all objects.

Window edge hit

If this option is checked, a subform can be selected if we click on its edge only.
On the contrary, subform is selected if we click at any point within the subform.

Show tooltips

If this option is checked, tooltip appears on the screen when cursor hovers over an object.

Insert at last point clicked

If this option is checked, the paste command creates new objects at position defined by last click. If the option is not checked, new object will be created at (0,0).

See Paste command in Edit section.

Editor options

Activates dialog to define some editor options: keyword colors and tab size.
