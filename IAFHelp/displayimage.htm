Displayimage
   

This method displays an image.

Format 1


            Displayimage(image_path)

Format 2


            Displayimage(image_field, image_type)

Parameters


·     image_path - alpha literal or alpha variable containing image path. The path can be absolute, relative, with or without logical names. File extension defines image type: ICO, BMP, GIF, JPG (JPEG).

·     image_field - table field containing image. Field type must be LONGVARBINARY. In MS SQL Server appropriate datatype is 'image', in Oracle 'long raw' or 'blob'.

·     image_type - specifies image type, integer. Possible values:

            0 - Auto. Runtime engine will try to determine image type.
            1 - ICO
            2 - BMP
            3 - GIF
            4 - JPG (JPEG)


 
Applied To

            Sourcefield, Bitmapbutton, Image, DestinationField.

Notes

            Bitmapbutton can display ICO, BMP or GIF images only. To remove picture from a bitmapbutton, use an empty string as an argument.

            To remove picture from a source field, image or destination field, use Init() method.


Examples

  
      sourcefield_1.Displayimage('c:\images\img1.bmp')
      sourcefield_1.Displayimage('myimages:img1.bmp')
      sourcefield_1.Displayimage(mytbl.pct,0)
      sourcefield_1.Init()    !!remove picture

! BITMAPBUTTON images

! Embedded images (image name is a literal)
! If any of embedded images does not exist at generation-time, executable program will not be generated.

! Image 'mybmp.bmp' from working folder is linked to the executable program
      bitmapbutton_1.Displayimage('mybmp.bmp')
! Image 'mybmp.bmp' from c:\iafw\iafres is linked to the executable program
      bitmapbutton_1.Displayimage('#mybmp.bmp')
! Image 'mybmp.bmp' from c:\temp is linked to the executable program
      bitmapbutton_1.Displayimage('c:\temp\mybmp.bmp')
! The animation 'myani.gif' from c:\temp is linked to the executable program
      bitmapbutton_1.Displayimage('c:\temp\myani.gif')

! Linked images (image name is a variable)
!     File c:\temp\myfile.ico will be opened at run-time
      $a80 = 'c:\temp\myfile.ico'
      bitmapbutton_1.Displayimage($a80)

! Image from a BLOB field
      Select fld1, fld2, pct FROM mytbl WHERE ...
      bitmapbutton_1.Displayimage(mytbl.pct,0)

      bitmapbutton_1.Displayimage("")     !!remove picture
