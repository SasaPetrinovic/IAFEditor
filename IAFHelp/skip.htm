SKIP
   

This statement skips n lines in a report.

Format

·          SKIP_AFTER [line | PAGE]

·          SKIP_BEFORE [line | PAGE | COLUMN]

Where

·    line - a number of the lines which are skipped. Number has to be greater than zero and less than page size.
·    PAGE - determines jump to a new page.
·    COLUMN - valid for multi-column reports only. Current print position is moved to the top of the next column.


Note

            Depending on a use of _AFTER or _BEFORE, lines are being skipped after or before the lineset where instruction was used.

            If line or PAGE is omitted, default skip is 1 line.

            In GUI reports one line is 16 pixels high.

Examples
  
      SKIP_AFTER 2      !!Print this lineset and skip 32 pixels
      SKIP_BEFORE 1 !! skip 16 pixels before printing the lineset
      SKIP_BEFORE       !!Same as above. Default skip is 16
      SKIP_BEFORE PAGE  !!Jump to the next page
      SKIP_BEFORE COLUMN      !!Jump to the top of the next column
