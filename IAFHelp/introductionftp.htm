Introduction
   

Object FTP allows client access to a FTP server. By using this object, we can download files from the server, upload files to the server, etc.

All methods return a value. Return value is an alpha field. If return value is blank, method has successfully ended, otherwise, return value holds an error message.


Object name:

            Ftp

Methods:
  
Close
Closes connection to the server
Createdirectory
Creates remote directory
Delete
Deletes remote file
Deletedirectory
Deletes remote directory
Dir
Gets list of files
Downloadfile
Gets a file from the server (download)
Getcurrentdirectory
Gets current directory
Open
Establishes connection to a FTP server
Rename
Renames remote file
Setcurrentdirectory
Changes the directory on the server
Uploadfile
Stores a file to the server (upload)
