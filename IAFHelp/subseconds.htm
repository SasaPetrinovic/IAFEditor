SUB_SECONDS
   

This function subtracts a number of seconds from a time or datetime field.

Format

            SUB_SECONDS(time, numseconds)

Return Value Type

            If 'time' parameter is a time field, return value is time.
            If 'time' parameter is a datetime field, return value is datetime.

Parameters


·     time - time_expression or datetime field.
·     numseconds - number of seconds, numeric_exp


Note 1

            If 'time' parameter is a time, out-of-range result time will be corrected automatically.

Note 2

            If 'time' parameter is a datetime, result is datetime field and all the field's components will be adjusted.

Note 3

            SUB_SECONDS(time,-10) gives the same result as ADD_SECONDS(time,10).

Examples

  
      $res_time = SUB_SECONDS (systime,5)
      $res_time = SUB_SECONDS (systime,counter_1)
      $res_time = SUB_SECONDS ("00:00:05",10) !! result = 23:59:55

      $res_dt = SUB_SECONDS("2009-12-01 15:00:00",5)
