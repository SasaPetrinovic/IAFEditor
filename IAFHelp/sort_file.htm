SORT_FILE
   

This statement sorts a sequential file.

Format

            SORT_FILE seqfile, out_file, sort_key1, ..., sort_keyN

            Sort_keyN has the format

            field_name [COLLATE collation_name] [ASC | DESC]

Where


·     seqfile   - an input sequential file from Datasource dialog.
·     out_file - physical name of output file, optional parameter. If omitted, input file will be overwritten. Output file has the same record format as input file.

Notes

            COLLATE is applicable only for alpha fields. Default collation_name is English.
            ASC is ascending sort order (default).
            DESC is descending sort order.

            Status of sort operation is in SORT_STATUS internal variable. If sort successful, SORT_STATUS contains number of sorted records. Negative value denotes sort error.


Examples

  
      sort_file rseq,,fld1          !!ascending
      sort_file rseq,,fld1[1:3]
      sort_file rseq,,fld1,fld2,fld3      !!ascending
      sort_file rseq,,fld1 DESC     !!descending
      sort_file rseq,,fld1 DESC, fld2 ASC
      sort_file rseq,,fld1 COLLATE English DESC, fld2

      sort_file rseq,,fld1 COLLATE English_CI DESC
      if SORT_STATUS < 0
            Pause 'Sort error='|sort_status
      else
            Pause 'Number of sorted records='|sort_status
      end_if

      sort_file rseq,'c:\temp\outfile.seq',fld1
      sort_file rseq,'tmpdir:outfile.seq',fld1
