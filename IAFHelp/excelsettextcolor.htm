Settextcolor
   

This method sets the text color of a range of Excel cells.

Format 1

            Settextcolor (range, color)

Format 2


            Settextcolor (line, column, color)



Parameters


·     range - range of cells in the form STARTING_CELL:ENDING_CELL, alpha variable. ENDING_CELL can be omitted.

·     color - a numeric variable with valid RGB color value or one of the predefined colors or RGB function.

·     line - row, numeric value, integer. The first line is 1, etc.

·     column - column, numeric value, integer. The first column is 1.

Return Value

            Status, numeric variable.

            Possible values:

            0          Ok
            -1         Document not opened
            -2         Invalid range
            -3         Error

Applied To

            Excel object.

Examples

  
      $status = Excel.Settextcolor('a1','black')
      $status = Excel.Settextcolor('a1:f1','red')
      $lin = 1
      $status = Excel.Settextcolor(compress_trim('A',$lin,':','F',$lin),'red')
      $status = Excel.settextcolor('a1:a1',RGB(255,0,0))
      $status = Excel.Settextcolor('a1:a10',255)
