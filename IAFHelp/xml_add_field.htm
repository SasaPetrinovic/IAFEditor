XML_ADD_FIELD
   

This function adds one field to a XML stream.


Format

            XML_ADD_FIELD(xml_variable, node_name, field_value[,code_page])

Result Value Type

            Xml variable (type = S).

Parameters


·     xml_variable - alpha variable containing XML stream. Variable type is CLOB, longchar (type = S).

·     node_name - name of XML node, alpha literal or alpha variable, required.

·     field_value - field to be inserted into XML stream, required.

·     code_page - if field_value is an ANSI alpha, this parameter specifies variable's code page. The parameter is optional, default value is ANSI code page.


Note

            The function inserts starting node, field value and ending node in the form <node_name>field_value</node_name>.

            Alpha fields will be converted internally into UTF-8.
            Default edit mask for a date field is 'YYYY-MM-DDT00:00:00'.
            Default edit mask for a datetime field is 'YYYY-MM-DDTHH:MM:SS'.
            Default edit mask for a numeric field is -------------9. Decimal point is POINT.


Examples

  
! Local field $xml has type S
      init $xml
      $xml = xml_add_field($xml,'fld1',today)
      $xml = xml_add_field($xml,'fld1',timestamp)
      $xml = xml_add_field($xml,'fld1',counter_1) !! num without decimals
      $xml = xml_add_field($xml,'fld1',acc_1) !! num with two decimals
      $xml = xml_add_field($xml,'fld1',$alpha,1252)
